#總經分析範例
**沛然量化分析觀點 Risksoft Capital Outlook**
分析標的：Global Assets Model 總經分析
更新日期：{數據日期}

**宏觀經濟移向牛尾**
　　1. 宏觀經濟為牛市或熊市將根據 a.全球資產與美元相對關係{ code = AllAssets Index:DXY Curncy }及 b.全球資產{ code = AllAssets Index }的「#35. 長期動能(共5個)」判斷牛熊市場狀況，寫出目前宏觀經濟牛熊判斷、長期趨勢解讀、中短期趨勢方向。
　　(1) 宏觀經濟牛熊判斷：
　　請根據補充資料判斷目前宏觀經濟為牛市、熊市或震盪市？並簡短說明
　　a. 牛市：當全球資產相對美元（Global Assets rel USD）且全球資產（Global Assets）趨勢持續 > 0，且 EVDS 長 > 0。
　　b. 熊市：當全球資產相對美元（Global Assets rel USD）且全球資產（Global Assets）趨勢持續 < 0，且 EVDS 長 < 0。
　　c. 震盪市：非牛市或熊市即為震盪市。
　　(2) 長期趨勢解讀：
　　a. 趨勢擴張/收斂：Global Assets 的 EVD 長 > 1 表示長期趨勢擴張，EVD 長 < 1 表示趨勢收斂。
　　b. 增速變化：Global Assets 的 EVS 長 > 0 表示長期增速持續，EVS 長 < 0 表示增速下降。
　　(3) 中短期趨勢方向：
　　a. 中短趨勢方向：Global Assets 的 總評分2 > 50 表示中短期趨勢向上，總評分2 < 50 表示向下。
　　b. 建議風險資產倉位 = 總評分2 （總評分2 = 倉位百分比）
　　2. 根據目前趨勢判斷，空間及時間上是否達成系統性風險發生的標準
　　(1)在空間上，若「#35. 長期動能(共5個)」當中有3個指標小於零，則警示注意系統性風險，若5個指標小於零，則發生系統性風險
　　(2)在時間上，全球宏觀經濟循環的向上週期為2年6個月，向下週期9~12個月，上次牛市啟動時間為2022年11月，請依照時間長度計算是否滿足系統性風險條件。

**2025 Q1/Q2由盛而弱**
　　(1) 預測2025年Q1/Q2宏觀全球資產走弱，美元將轉強。
　　(2) 請根據Global Assets Model中的10個商品，用中短期趨勢的"總評分2"由強至弱排序。
　　(3) 若目前為牛市，檢查Global Assets Model中的10個商品的長、中、短期指標，將有風險的商品列出，並說明警示的風險。
　　a. 「18. 短期動能(共8個)」當中有5個指標小於零，則警示注意短期市場風險。
　　b. 「28. 中期動能(共5個)」當中有3個指標小於零，則警示注意中期市場風險。
　　c. 「35. 長期動能(共5個)」當中有3個指標小於零，則警示注意系統性風險。

**風險資產總倉位**
　　(1)根據"宏觀經濟牛熊判斷"判斷整體風險資產倉位建議
    a. 牛市：倉位100%，無系統性風險。
    b. 熊市：倉位0%，建議持有美元。
    c. 震盪市：倉位40%，需留意市場風險。
    (2)根據“股債比”(TECH Index rel Bond Index)總評分2為目前建議股市配比，剩餘的百分比為建議債市配比，計算並顯示股債比。

**2025 Q1/Q2全球資產走弱是因，美元轉強是果**
　　(1) 根據宏觀經濟牛熊判斷，若目前在牛市或震盪市的階段，檢查Global Assets Model中的10個商品，將目前發生的風險列出
    a. Currency/USD 趨勢持續 < 0，且EVDS長 < 0，顯示注意超額匯率風險，全球資產可能在4至6個月內下跌
    b. Tech Index 趨勢持續 < 0，且EVDS長 < 0，顯示注意科技股風險，全球資產可能在2至3個月內下跌
    c. 美元指數 趨勢持續 > 0，且EVDS長 > 0，（美元上升對全球資產不利），顯示美元升、亞幣貶，注意全球股市風險，注意亞股風險
    d. Global Assets 趨勢持續 < 0，且EVDS長 < 0，顯示注意系統性風險，全球資產下跌，美元將升值
    e. 若未發生上述四種風險，顯示目前無發生系統性風險
    (2) 若未來上述前四點有三點發生，則表示未來美元將升值，建議出口商在未來匯率觸及預算匯率前，可以小部分慢慢換匯。
    (3) 若未來上述前四點有三點發生，則表示未來美元將升值，建議進口商在未來匯率觸及預算匯率前，積極均量分批購匯，以降低風險。

註1. 全球資產Global Assets分成七大類風險資產及美元，七大類資產分別為(1)比特幣 (2)能源類 (3)農產品類 (4)金屬類 (5)股指類 (6)債券類 (7)超額匯率，七大類都與美元呈負相關。

註2. 「超額匯率」是將原匯率漲跌幅，額外再除以美元指數漲跌幅，數學意義是了解匯率是否相對美元指數超漲(貶值)或超跌(升值)，物理意義類似台積電對台股指數的超額利潤(Alpha)。

原匯率的對風險資產的相關性(統計6年)約為-20%，表示因風險資產如股市下跌，則資金回流美元，造成當地匯率貶值，所以匯率與股市為負相關，但是在除以美元指數漲跌幅後，「超額匯率」對股市的相關性卻轉為+20%到+40%正相關，並且對風險資產的上漲或下跌有領先2到3個月的預測性。
