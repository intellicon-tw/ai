#GAM模型第1層級, 量化總經判定原則
1. 牛市、熊市、震盪市：
  - 牛市：下列四個條件都要滿足
      (1)全球資產相對美元(Global Assets rel USD)的"趨勢持續" > 0
      (2)全球資產相對美元(Global Assets rel USD)的"EVDS長" > 0
      (3)全球資產(Global Assets)的"趨勢持續" > 0
      (4)全球資產(Global Assets)的"EVDS長" > 0
  - 熊市：下列四個條件都要滿足
      (1)全球資產相對美元(Global Assets rel USD)的"趨勢持續" < 0
      (2)全球資產相對美元(Global Assets rel USD)的"EVDS長" < 0
      (3)全球資產(Global Assets)的"趨勢持續" < 0
      (4)全球資產(Global Assets)的"EVDS長" < 0
  - 震盪市：非牛市或熊市即為震盪市。
2. 商品類長短期趨勢判定：
  - 長期趨勢擴張：EVD 長 > 1
  - 長期增速持續：EVS 長 > 0
  - 中短期趨勢向上：總評分2 > 50
  - 系統性風險發生：EVD 長 < 0
3. 牛市與熊市週期：
  - 牛市上升階段：約1.5年內
  - 牛尾階段：約1.5至2.5年
  - 熊市下跌階段：約9個月內
  - 熊尾階段：約9至12個月內
4. 市場變動預測：
    預測市場轉變點，如牛市開始等於熊市結束。

##補充說明：
1. 全球資產漲跌影響美元走勢。
2. 宏觀經濟循環約3至3.5年一組周期，上行期間約2至2.5年，下行期間約6至9個月。2023 Q1開始的牛市預計至2025 Q1達高點，目前處在牛尾端。
3. 半導體領先指標可作為市場基本面的擇時指標。
4. 科技股指數由多個半導體和科技股指數組合而成，以降低波動率，使趨勢更明顯。
5. 超額匯率(Currency/USD)用於衡量貨幣對美元的超漲或超跌情況，有領先股市3至4個月的預測性。
6. 創業板領先指標觀察創業板對上證指數是否有超額利潤，可視為A股、港股市場的領先指標。
7. 進出口商操作：美元將升值時，出口商可小部分換匯；進口商積極購匯以降低風險。
8. 全球資產與美元關係：
  - 全球資產與美元呈負相關；市場走弱則美元走強。
  - 超額匯率(Currency/USD)與全球資產正相關，與美元負相關。


##量化總經分析報告的生成邏輯
沛然量化分析觀點 Risksoft Technology
更新日期：mm/dd/yyyy

1. 量化總經
(1) 按Global Assets、Global Assets rel USD判斷牛市、熊市、震盪市，並顯示進入牛市的日期。Global Assets預測漲跌幅，美元指數預測漲跌幅。
(2) 全球資產夏天、長期動能、中期動能、短期動能，動能與上週(T-5)比較增加或減少，全球資產預測漲跌幅。
(3) 美元產夏天、長期動能、中期動能、短期動能，動能與上週(T-5)比較增加或減少，美元預測漲跌幅。

2. 大類資產
(1) 按短期動能比較全球資產與美元強弱，參考Global Assets rel USD的總評分2當做倉位百分比，百分比十進位。按短期動能排序八大資產，顯示前三名，配重按照總評分2配置三等份。
(2) 匯率指數預測漲跌幅，按短期動能顯示貶值的前三名，升值的前三名，如遇缺不顯示。
(3) 按短期動能顯示強於全球的風險資產，顯示弱於全球的風險資產。顯示最強的風險資產其長期動能。顯示最弱的風險資產其長期動能。
(4) 股市(科技股)四季、天氣、長期動能、中期動能、短期動能，動能與上週(T-5)比較增加或減少，最近一次的買賣訊號，預測漲跌幅。
(5) 超額匯率四季、天氣、長期動能、中期動能、短期動能，動能與上週(T-5)比較增加或減少，最近一次的買賣訊號，預測漲跌幅。
