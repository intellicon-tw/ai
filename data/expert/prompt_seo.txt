#SEO prompt總覽
### 一、**標題與描述類**

1. **SEO友好的標題生成**：根據主題和關鍵字，自動生成具有吸引力且符合SEO的標題。
    - Prompt 示例：*「為[主題]生成五個吸引人且具SEO效果的標題，包含[關鍵字]。」*
2. **Meta Description 生成**：撰寫簡潔、包含關鍵字的元描述以提升點擊率。
    - Prompt 示例：*「根據這篇文章的內容，生成一個150字內的元描述，並包含主要關鍵字[關鍵字]。」*

### 二、**文章結構與內容類**

1. **清單式內容生成**：生成有條理的清單型文章（如「10個步驟」或「5個最佳做法」）。
    - Prompt 示例：*「根據[主題]生成一篇包含5個步驟的清單型文章，每個步驟都有具體說明，並自然融入關鍵字[關鍵字]。」*
2. **深度長文撰寫**：撰寫詳盡且具有SEO優化的長篇文章，針對特定主題進行深入探討。
    - Prompt 示例：*「撰寫一篇2000字的文章，詳述[主題]，文章應包括關鍵字[關鍵字]，並按章節劃分內容，適當使用H1、H2標題。」*
3. **操作指南與教學文章**：根據主題撰寫清晰的操作步驟或教學指南。
    - Prompt 示例：*「針對[主題]撰寫詳細的教學文章，分步驟描述，並包含實際操作的建議，融入SEO關鍵字[關鍵字]。」*

### 三、**特定用途內容類**

1. **商品與服務描述撰寫**：撰寫精確的產品描述，突出產品特性並符合SEO。
    - Prompt 示例：*「為[商品名稱]撰寫一段具吸引力且包含關鍵字的產品描述，突出其主要特性與優點。」*
2. **常見問題 (FAQ) 撰寫**：針對特定領域或產品生成FAQ內容，幫助提升搜尋結果的可見度。
    - Prompt 示例：*「生成關於[主題]的五個常見問題與解答，問題應包含關鍵字[關鍵字]並結構化呈現。」*
3. **案例研究與用戶見證撰寫**：撰寫展示實際應用場景的案例研究或用戶見證。
    - Prompt 示例：*「根據[案例主題]撰寫一篇案例研究文章，描述該案例的背景、挑戰、解決方案與結果，並適當運用SEO關鍵字。」*

### 四、**關鍵字優化與更新類**

1. **舊文章關鍵字優化**：重新優化現有文章，添加最新關鍵字和SEO策略。
    - Prompt 示例：*「檢查並重新優化這篇[文章主題]，加入當前熱門關鍵字[關鍵字]，並調整段落結構提升SEO效果。」*
2. **內容重寫與擴充**：針對既有內容進行重寫或擴充，以增加SEO價值。
    - Prompt 示例：*「將[現有文章]的內容擴充至兩倍長度，並整合新關鍵字[關鍵字]，提升文章的深度與權威性。」*

### 五、**結構與格式類**

1. **目錄生成與內部連結設計**：根據文章內容自動生成內部目錄與連結建議。
    - Prompt 示例：*「為這篇[文章主題]設計內部連結結構，包含相關章節的目錄與內鏈策略，優化SEO。」*
2. **SEO友好的標籤與標題設計**：根據文章內容設計合適的H1、H2、H3標題結構。
    - Prompt 示例：*「根據這篇文章的結構，生成一組合適的H1、H2標題，提升內容的SEO效果。」*

### 六、**多媒體內容類**

1. **圖片Alt文字生成**：針對文章內的圖片生成具關鍵字的Alt描述。
    - Prompt 示例：*「為這些圖片生成Alt文字描述，每個描述應包含SEO關鍵字[關鍵字]並簡明扼要。」*
2. **影片描述與標籤撰寫**：撰寫適合影片的描述與標籤，幫助提升影片的SEO排名。
    - Prompt 示例：*「撰寫[影片主題]的SEO優化描述，包含相關關鍵字，並生成適合的標籤。」*
