# 工作室2_研發技術

本工作室包含 4 個 Agent，共 1 個使用表格知識庫，1 個使用文件知識庫。

## Agent 清單

| 編號 | Agent 名稱 | 知識庫類型 | POC 方式 | 兩階段 |
|---|---|---|---|---|
| 2.1 | 認證法規查詢助理 | 全文查找知識庫（法規認證文件庫） | RAG + 工具 | 否 |
| 2.2 | BOM料號查詢助理 | 表格查詢知識庫（BOM 主檔） | csv 資料表 | 否 |
| 2.3 | 研發週報撰寫助理 | -（純提示工程） | 純提示工程 | 否 |
| 2.4 | 專利查詢助理 | -（Google Search 即時查詢） | RAG + 工具 | 否 |

## 各 Agent 子目錄

- `2.1_認證法規查詢助理/` — 1_prompt + 2_知識庫範例資料 + 3_驗測情境 + 4_正式上線前的準備指南
- `2.2_BOM料號查詢助理/` — 1_prompt + 2_知識庫範例資料 + 3_驗測情境 + 4_正式上線前的準備指南
- `2.3_研發週報撰寫助理/` — 1_prompt + 2_知識庫範例資料 + 3_驗測情境 + 4_正式上線前的準備指南
- `2.4_專利查詢助理/` — 1_prompt + 2_知識庫範例資料 + 3_驗測情境 + 4_正式上線前的準備指南
