# 工作室1_業務

本工作室包含 3 個 Agent，共 1 個使用表格知識庫，1 個使用文件知識庫。

## Agent 清單

| 編號 | Agent 名稱 | 知識庫類型 | POC 方式 | 兩階段 |
|---|---|---|---|---|
| 1.1 | 產品選型推薦助理 | 表格查詢知識庫（桓達產品規格主檔） | csv 資料表 | 否 |
| 1.2 | 業務話術與FAQ助理 | Agent 向量知識庫（業務 FAQ 與話術手冊） | RAG 文件 | 否 |
| 1.3 | 客戶需求整理助理 | -（純提示工程） | 純提示工程 | 否 |

## 各 Agent 子目錄

- `1.1_產品選型推薦助理/` — 1_prompt + 2_知識庫範例資料 + 3_驗測情境 + 4_正式上線前的準備指南
- `1.2_業務話術與FAQ助理/` — 1_prompt + 2_知識庫範例資料 + 3_驗測情境 + 4_正式上線前的準備指南
- `1.3_客戶需求整理助理/` — 1_prompt + 2_知識庫範例資料 + 3_驗測情境 + 4_正式上線前的準備指南
