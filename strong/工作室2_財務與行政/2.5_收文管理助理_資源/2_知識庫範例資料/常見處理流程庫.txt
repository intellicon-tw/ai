# 斯創教育公文常見處理流程庫

本知識庫整理斯創過去 3 年處理各類公部門公文的標準流程與經驗教訓。每類公文均含建議步驟、時程節點、產出物、注意事項。

## 一、林業及自然保育署委辦案陳報函

### 標準處理流程
1. **收文當日**：行政總監收件，判定屬期初／期中／期末／結案。
2. **D+1**：執行專案負責人（教育總監或專案經理）審閱主旨與限辦日。
3. **D+3**：召開內部準備會議，確認對應的工作項目與成果證明資料。
4. **D+5-7**：依分署規範格式撰寫陳報報告（含執行情形、財務、附件）。
5. **D+10**：執行長最終審閱。
6. **D+12**：用印寄出（紙本掛號 + Email 副本）。

### 注意事項（經驗教訓）
- 林業署各分署的格式略有差異，請套用既有「該分署最近一份函」的範本
- 期末報告若延遲超過限辦日 3 天，分署將啟動催辦並影響後續案件評分
- 附件須含財務核銷對照表，與會計部協同準備

### 產出物
- 陳報函（紙本 + PDF）
- 附件清單與成果證明
- 內部結案備忘錄

## 二、各部會補助辦法或標案公告

### 標準處理流程
1. **收文當日**：執行顧問與執行長共同判定是否申請。
2. **D+3**：若決定申請，由相對應主管擔任 PM，組成 3 人申請小組。
3. **D+5**：撰寫申請書骨架（含背景、目標、方法、團隊、預算）。
4. **D+10**：完成初稿，內部審閱。
5. **D+15**：修訂後送外部專家審閱（如有）。
6. **D+20**：定稿用印寄出。

### 注意事項
- 預算編列需保留 5-10% 安全餘裕，避免決標後追加困難
- 若涉及合作夥伴（永泰、永在），需提前 7 天確認合作意向書

### 產出物
- 申請書（紙本與電子檔）
- 預算明細表
- 合作備忘錄（如需要）

## 三、會議邀請函（部會主管會議、研討會、論壇）

### 標準處理流程
1. **收文當日**：由執行長秘書安排行事曆，評估是否出席。
2. **D+1**：確認出席者（執行長／教育總監／其他經理）。
3. **D+2**：回覆主辦單位（接受／婉拒）。
4. **D+3-5**：若需上台簡報，依「3.3 活動辦理分工」或「4.3 簡報製作」協助。
5. **會議當日**：依議程準備資料、攜帶名片。
6. **會議結束後 D+1**：撰寫會議重點摘要供內部分享。

### 注意事項
- 林業署、教育部年度大會優先出席
- 婉拒時建議推薦其他斯創夥伴出席，維持關係

## 四、人事相關公文（兵役、勞動部、稅務）

### 標準處理流程
1. **收文當日**：行政總監收件，判定涉及之員工。
2. **D+1-2**：通知該員工並確認資料齊備。
3. **D+3-5**：依公文要求填寫回覆或繳交資料。
4. **D+7**：用印寄出。
5. **建檔**：將公文與回覆副本存入該員工人事檔案。

### 注意事項
- 涉及個資的回覆，須去識別化非必要欄位
- 兵役相關公文有法定回覆期限，不可拖延

## 五、合作意向書、MOU 邀請

### 標準處理流程
1. **收文當日**：執行長與執行顧問評估合作意向。
2. **D+3**：內部討論合作框架（範圍、財務、IP 歸屬）。
3. **D+7**：法律審閱（如涉及智財或長期承諾）。
4. **D+10-14**：簽署。

### 注意事項
- MOU 雖無法律約束力，但會影響商譽，務必審慎
- 跨機構合作須事先告知所有相關專案 PM

## 六、稽核與審計相關公文

### 標準處理流程
1. **收文當日**：行政總監收件，判定稽核範圍。
2. **D+1**：通知會計事務所與相關專案 PM。
3. **D+3-7**：準備稽核資料（合約、發票、支出明細、成果報告）。
4. **D+10**：稽核會議。
5. **D+15-30**：依稽核發現改善並回函。

### 注意事項
- 公部門委辦案的稽核要求最嚴，建議每月做一次自我核帳
- 稽核改善若超過 30 天未回覆，將影響未來投標資格

## 七、未在以上類別的公文

若公文性質不在上述六類，請依以下基本流程處理：
1. 行政總監收件並判定派工對象
2. 派工對象 D+3 內回覆是否需要進一步行動
3. 若需正式回函，依「2.4 公文擬定助理」協助
4. 將此公文摘要 + 處理紀錄補入本流程庫，作為未來參考
