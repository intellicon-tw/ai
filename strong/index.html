★ 全體概覽 — 5 個工作室、25 隻 Agent

本 POC 套件涵蓋 5 個工作室、25 隻 Agent，從業務開發、財務行政、客戶與專案管理、知識整理，到滿月圓現場營運，為斯創教育提供企業級 AI 導入的完整藍圖。

關於示範資料的重要說明：所有 Agent 的知識庫均含智慧方案顧問團隊預先生成的示範資料。這些示範資料是為了讓學員在工作坊上立即看到 Agent 運作效果而設計，必須在實際上線前由斯創對應部門替換為真實資料，AI Agent 才能真正用於業務決策。每隻 Agent 的「上線指南」中都有明確指引替換哪些檔案、如何替換。

使用方式
  1. 從左側 sidebar 或上方工作室卡片，點擊任一 Agent 進入開發說明
  2. 每個 Agent 頁有 5 個 Tab：概覽、完整 Prompt、知識庫設定、驗測情境、上線指南
  3. 依 Tab 順序操作，預計每隻 Agent 60-90 分鐘可完成部署
