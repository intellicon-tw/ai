# 工作室1_GM業務智慧助理

本工作室包含 4 個 Agent，共 3 個使用表格知識庫，1 個使用文件知識庫。

## Agent 清單

| 編號 | Agent 名稱 | 知識庫類型 | POC 方式 | 兩階段 |
|---|---|---|---|---|
| 1.1 | 客戶研究助理 | 全文查找知識庫（客戶研究素材）+ 向量知識庫（東元研究範本，選配） | RAG 文件 | 條件式（上傳 PDF 時兩階段） |
| 1.2 | 商機分析師 | 表格查詢知識庫（東元GM能力對照表 + 歷史商機案例） | xlsx 資料表 | 否 |
| 1.3 | 詢價解析助理 | 表格查詢知識庫（產品型號標準化表） | xlsx 資料表 | 條件式（上傳 PDF 時兩階段） |
| 1.4 | 報價策略分析師 | 表格查詢知識庫（歷史報價成交 + BOM 標準成本） | xlsx 資料表 | 否 |

## 各 Agent 子目錄

- `1.1_客戶研究助理/` — prompt.txt、測試資料集.txt、驗測情境.txt
- `1.2_商機分析師/` — prompt.txt、測試資料集.txt、驗測情境.txt、知識庫 CSV
- `1.3_詢價解析助理/` — prompt.txt、測試資料集.txt、驗測情境.txt、知識庫 CSV
- `1.4_報價策略分析師/` — prompt.txt、測試資料集.txt、驗測情境.txt、知識庫 CSV
