# 工作室1_PE業務智慧助理

本工作室包含 4 個 Agent，共 3 個使用表格知識庫，2 個使用文件知識庫。

## Agent 清單

| 編號 | Agent 名稱 | 知識庫類型 | POC 方式 | 兩階段 |
|---|---|---|---|---|
| 1.1 | 估算作業查詢助手 | 表格查詢知識庫（設備標準單價 + 歷史案件 + 人月配比） | xlsx 資料表 | 否 |
| 1.2 | 銷售流程引導助手 | 表格查詢知識庫（估算項目對照 + 歷史案件 + 設備單價 + 人月配比） | xlsx 資料表 | 否 |
| 1.3 | 合約解析助手 | 全文查找知識庫（業務上傳的合約 PDF） | RAG 文件 | 條件式（上傳 PDF 時兩階段） |
| 1.4 | 合約問答助手 | 全文查找（合約）+ 表格查詢（標準條款 + 風險案例） | xlsx 資料表 | 條件式（上傳 PDF 時兩階段） |

## 各 Agent 子目錄

- `1.1_估算作業查詢助手/` — prompt.txt、測試資料集.txt、驗測情境.txt、知識庫 CSV
- `1.2_銷售流程引導助手/` — prompt.txt、測試資料集.txt、驗測情境.txt、知識庫 CSV
- `1.3_合約解析助手/` — prompt.txt、測試資料集.txt、驗測情境.txt
- `1.4_合約問答助手/` — prompt.txt、測試資料集.txt、驗測情境.txt、知識庫 CSV
