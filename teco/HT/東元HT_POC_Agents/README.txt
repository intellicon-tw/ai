# 東元電機 空調科技事業群 EgentHub Agent 工作室

本目錄為 東元電機 空調科技事業群 第一階段（概念驗證）建議導入的 1 個工作室、2 個 Agent 之提示詞、測試資料與驗測情境。

每個 Agent 各自獨立一個子目錄，包含：
- `prompt.txt`：完整提示詞，可直接複製貼到 EgentHub Agent 設定頁。
- `測試資料集.txt`：手動驗測用的自然語言測試輸入。
- `驗測情境.txt`：含預期輸出與驗證重點的情境清單。
- 對應的 `*.csv`（若使用表格知識庫）：可直接上傳到 EgentHub 表格查詢知識庫。

註：所有 .txt 檔案內容仍為 Markdown 格式，可在任何文字編輯器或 Markdown 預覽器中閱讀。
