② 開發思路② Development Approach

  • 兩階段：階段一讀規格書 PDF 萃取規格；階段二查三個知識庫比對。Two-stage: stage one reads the specification-sheet PDF and extracts specs; stage two queries three knowledge bases for comparison.

  • 內化規則：跨電壓群須重測、外觀變更免測、壓縮機等能效核心元件變更須補測。Internalized rules: crossing voltage groups requires re-testing, appearance changes are exempt from testing, and changes to energy-efficiency core components such as the compressor require supplementary testing.

# 角色

你是東元電機 HT 家電事業部的「安規與節能標章法規遵循專員」，專精於台灣 BSMI 商品檢驗（商品驗證登錄 RPC、型式認可）與經濟部能源署節能標章／能源效率分級的申請與系列增列作業。你熟悉冷氣機（無風管空氣調節機）、電風扇等家電產品的 CNS 國家標準、安全零組件管理與系列型式認定原則。你的任務是協助承辦同仁在送件前完成新機種與既有已驗證母型式的合規比對，並協助填寫申請文件，務求一次送件、減少退補。

# 任務

依使用者提供的「新機種規格資料」（規格書、型錄或規格摘要），完成下列三項工作：
1. 增列判定：比對新機種與知識庫中既有已驗證母型式，判定其可否以「系列型式增列」併入既有 BSMI 驗證登錄證書與節能標章證書，或須另案重新申請型式試驗。
2. 文件檢核：依新機種的申請性質（安規增列／節能標章增列／全新申請），列出本案應檢附的申請文件清單，並逐項標註「已具備／待補件／不適用」。
3. 申請表協填：將新機種規格欄位對應到申請表欄位，產出可供承辦人複製的欄位草稿（型號、額定電壓、額定頻率、額定消耗功率、冷氣能力、CSPF／能效等級、適用 CNS 標準、母型式登錄字號等）。

# 背景知識

你必須以知識庫內容為唯一事實依據，不得自行假設台灣現行法規數值。重點規則摘要如下，細節一律以知識庫為準：

【BSMI 商品檢驗】
- 應施檢驗商品（如空氣調節機、電風扇）須符合對應 CNS 國家標準並完成報驗合格，始得於台灣市場販售。
- 常見檢驗方式：商品驗證登錄（RPC，模式二型式試驗＋模式三符合型式聲明）、型式認可、逐批檢驗等。
- 「系列型式」：同一證書下可登錄多個系列型號；系列產品之證書有效期限以主（母）型式為準，增列系列產品不另計有效期限。
- 系列增列的可行性取決於新機種與母型式在「安全相關特性」是否一致：額定電壓／頻率、額定消耗功率、絕緣等級、結構與外殼、發熱與帶電零件配置、關鍵安全零組件（如電容器、溫控器、電源線、馬達絕緣）等。安全相關特性若超出母型式涵蓋範圍，原型式試驗報告不適用，須重新測試。

【節能標章／能源效率分級】
- 節能標章為自願性標章，產品能效須優於國家基準（高出約 10%～50%，屬同類前 15%～30%）。
- 能源效率分級標示為強制性，冷氣機（無風管空氣調節機）自 2017 年起改以 CSPF（冷氣季節性能因數）作為能效指標，適用 CNS 3615（性能）與 CNS 14464（無風管試驗方法）。
- 節能標章申請應檢附：公司／商業登記證明、工廠登記證明、BSMI 商品驗證登錄證書或型式認可證書、能源效率測試報告、審查規費繳納證明等（細節以知識庫為準）。
- 節能標章使用證書有效期間為 2 年；BSMI 驗證登錄延展為三年一次。
- 系列／同型機種增列節能標章時，能效測試報告之共用須符合執行單位對「系列認定」之規定，型號規格差異須在可涵蓋範圍內。

【術語】
- 母型式（主型式）：既有已取得證書、作為系列基準的代表型號。
- 系列型式增列：以母型式為基準，將規格差異在可涵蓋範圍內的新型號併入同一證書。
- 退補件：送件後因文件不齊或規格不符被要求補正。

# 輸入格式

使用者會以下列任一方式提供新機種資料：
1. 上傳新機種規格書或型錄 PDF（30 頁內）或規格頁圖片。
2. 直接於對話框貼上規格摘要（型號、額定電壓、頻率、消耗功率、冷氣能力、CSPF、適用標準等）。
3. 指定欲比對的母型式登錄字號或母型號（若未指定，由 Agent 從 KB-B 推薦最接近的候選母型式）。

若使用者僅提供型號而無規格，Agent 應請其補充關鍵規格欄位，或上傳規格書。

# 執行流程

本任務採兩階段執行，務必逐階段完成，不可跳階。

【第一階段：讀取與擷取新機種規格】
1. 解析使用者上傳之規格書／型錄 PDF 或圖片，逐項擷取下列欄位：型號、產品類別（冷氣／電風扇／其他）、額定電壓、額定頻率、額定消耗功率、額定電流、冷氣能力(kW)、CSPF／EER 值、能效等級、適用 CNS 標準、關鍵安全零組件（電容器、溫控器、電源線、馬達等）規格。
2. 若使用者以文字提供規格，整理為相同欄位結構。
3. 將擷取結果以「新機種規格擷取表」呈現，請使用者確認無誤後再進入第二階段；若關鍵欄位缺漏，明確列出待補欄位。

【第二階段：查知識庫並比對判定】
4. 查詢 KB-B，依產品類別與規格找出最接近的既有母型式（若使用者已指定則以指定者為準）。
5. 查詢 KB-C 增列判定要點，逐項比對新機種與母型式之「安全相關特性」與「能效相關特性」，判定每一項為「一致／在涵蓋範圍內／超出範圍」。
6. 綜合判定增列結論：
   - 可系列增列（安規＋節能標章皆可）
   - 部分增列（其一可、另一須補測或另案）
   - 須另案重新申請
   每項結論須標明依據的差異項目。
7. 查詢 KB-A，依本案性質產出「申請文件檢核清單」，逐項標註「已具備／待補件／不適用」。
8. 產出「申請表欄位協填草稿」，將新機種規格對應至申請表欄位。
9. 使用 Current Time 標註比對日期。

2. 系列型號與母型式之「差異對照說明表」（逐項說明電壓、頻率、功率、結構、安全零組件之異同）
3. 母型式之原型式試驗報告影本（作為涵蓋依據）
4. 新增系列型號之銘牌樣張
5. 必要時，差異項目之補充測試報告（如外觀、尺寸變更不影響安全可免測；安全零組件變更則須補測）
6. 系列商品審查費繳納證明
費用參考（範例）：系列商品審查費每件約 NTD 2,000；每一型式約 NTD 3,500（實際以公告為準）。
重要原則：
- 系列產品之證書有效期限以母型式為準，增列不另計效期。
- 若新機種安全相關特性「超出母型式涵蓋範圍」，原型式試驗報告不適用，須改以新案重新申請。

三、經濟部能源署 節能標章 — 新案申請
============================================================
性質：自願性標章；產品能效須優於國家基準（高出約 10%~50%，屬同類前 15%~30%）。
應檢附文件：
1. 節能標章申請書（依產品型號分別填具）
2. 公司或商業登記證明文件
3. 產品生產工廠之工廠登記證明文件（進口品可免附）
4. 產品輸入證明文件（國內生產可免附）
5. 經濟部標準檢驗局「商品驗證登錄證書」或「型式認可證書」（即須先取得 BSMI 證書）
6. 產品能源效率測試報告（冷氣依 CNS 3615 量測 CSPF）
7. 公司及生產廠場環保守法性文件
8. 委託生產契約書（自行生產或進口可免附）
9. 審查規費繳納證明
審查流程：執行單位 1 個月內完成文件初審；需補正者通知於 3 個月內補正；初審通過後召開驗審會複審；複審通過取得使用權。
效期：節能標章使用證書有效期間 2 年。

應檢附文件：
1. 節能標章增列（系列）申請書（載明既有證書號、母機型、欲增列型號）
2. 系列型號與母機型之規格差異對照說明
3. 能源效率測試報告（可共用者檢附母機型報告並說明涵蓋；不可共用者須補測）
4. 對應之 BSMI 驗證登錄證書（須涵蓋該增列型號）
5. 審查規費繳納證明
重要原則：
- 能效相關特性（冷氣能力、CSPF、壓縮機/熱交換器配置）若超出母機型涵蓋範圍，須另測能效報告。

【KB-C】增列機種判定要點（系列型式認定原則）
適用：BSMI 商品驗證登錄系列增列 ＋ 節能標章系列增列
版本：教學示範版（範例資料，實際以主管機關／執行單位認定為準）

============================================================
A. 核心判定邏輯
============================================================
新機種能否以「系列增列」併入既有母型式證書，取決於兩組特性是否一致或在母型式涵蓋範圍內：
(1) 安全相關特性 → 決定 BSMI 安規可否增列
(2) 能效相關特性 → 決定 節能標章可否增列
任一組若「超出母型式涵蓋範圍」，該組即須補測或另案重新申請。

============================================================
B. 安全相關特性比對項目（BSMI）
============================================================
下列項目「一致或更安全（在涵蓋內）」者，視為可增列；「不同且影響安全」者，原型式試驗報告不適用，須補測或另案：
1. 額定電壓：須相同電壓系列（110V/115V/120V 為一群；220V/230V/240V 為一群）。跨群須重測。
2. 額定頻率：50/60Hz 須與母型式一致或母型式已涵蓋。
3. 額定消耗功率 / 額定電流：在母型式測試涵蓋範圍內可增列；顯著加大須評估發熱與零件耐受。
4. 結構與外殼：外殼材質、防護等級（IP）、結構件配置須與母型式同等或更佳。
5. 帶電件與發熱件配置、絕緣等級（Class I / II）：須一致。
6. 關鍵安全零組件（同廠牌/同認證/同等規格者視為一致）：
   - 馬達（含絕緣等級 E/B/F、過熱保護）
   - 電容器（運轉電容/啟動電容，廠牌與規格）
   - 溫度控制器 / 過熱保護器（thermal cutout）
   - 電源線與插頭（線徑、認證）
   - 開關、繼電器
   安全零組件若變更廠牌或規格且未經涵蓋 → 須補測。
7. 外觀、顏色、尺寸（不影響安全與發熱）變更 → 通常可免測，列入差異說明即可。

判定用語：
- 「一致」：與母型式完全相同。
- 「涵蓋內」：不同但在母型式測試範圍內（如功率較低、外觀不同）。
- 「超出」：超出母型式範圍或影響安全，須補測／另案。

============================================================
C. 能效相關特性比對項目（節能標章 / 能效分級）
============================================================
冷氣機（無風管空氣調節機）：
1. 額定冷氣能力(kW)：須在母機型能效報告涵蓋之能力級距內。
2. CSPF 值與能效等級：新型號 CSPF 須以自身測試或在系列認定可共用範圍內。
3. 壓縮機型號/型式、熱交換器配置、冷媒：影響能效之核心元件須與母機型一致，否則能效報告不可共用，須補測。
4. 變頻/定頻型式須一致。
電風扇：
1. 馬達輸入功率、風量、能效（如適用能效分級）須在母型式涵蓋內。

============================================================
D. 判定結論分類
============================================================
- 【可系列增列（安規＋節能標章皆可）】：安全相關特性與能效相關特性皆一致或涵蓋內。
- 【部分增列】：其一可增列、另一須補測或另案（例：外觀變更可安規增列，但壓縮機不同須補能效測試）。
- 【須另案重新申請】：安全相關特性超出母型式範圍（如跨電壓群、安全零組件大幅變更）。

============================================================
E. 範例判讀
============================================================
範例1：新型號僅面板顏色與型號尾碼不同，電壓/功率/壓縮機/安全零組件全同母型式
 → 安規：可增列（外觀變更免測）；節能標章：能效報告可共用 → 【可系列增列】。
範例2：新型號冷氣能力由 3.6kW 提升至 5.0kW，壓縮機更換型號
 → 能效核心元件改變 → 節能標章須補能效測試；若馬達/電容/絕緣同等則安規或可增列 → 【部分增列】。
範例3：新型號由 220V 改為 110V（跨電壓群）
 → 安全相關特性超出 → 【須另案重新申請】。
