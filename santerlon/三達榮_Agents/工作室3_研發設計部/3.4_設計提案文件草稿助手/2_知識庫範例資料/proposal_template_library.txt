# 設計提案文件範本庫

## 範本 1：標準產品提案

```
# [產品名稱] Design Proposal

## 1. 設計理念
[一段話描述產品的核心訴求與設計哲學，2-3 句]

## 2. 規格摘要
- 尺寸：[XX cm × XX cm × XX cm]
- 重量：[XX g]
- 主材質：[material]
- 內襯：[lining]

## 3. 設計特色
- 特色 1：[亮點描述]
- 特色 2：[亮點描述]
- 特色 3：[亮點描述]

## 4. 適用情境
[描述目標使用者與使用場景]

## 5. 量產可行性
- 預估開模費：[USD X]
- 預估單價：[USD Y/pc]
- 預估交期：[X days]

## 6. 後續行動
- 確認規格 → 打樣 → 量產
```
