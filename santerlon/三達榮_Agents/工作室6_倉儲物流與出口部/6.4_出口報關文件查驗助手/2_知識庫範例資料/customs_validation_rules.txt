# 出口報關文件交叉驗證規則

## 三份文件必須一致的欄位

### 1. 金額一致性
- Commercial Invoice 總金額 = Packing List 總金額 = B/L 申報金額
- 若 B/L 為 Freight Collect，金額為 FOB 金額；若 Freight Prepaid，金額含運費

### 2. 數量一致性
- Invoice 總 PCS = Packing List 總 PCS
- Invoice 總 Cartons = Packing List 總 Cartons
- B/L 申報 Cartons = Packing List 總 Cartons

### 3. 品名一致性
- 三份文件的商品描述應一致或可對應
- HS Code 應一致（Invoice 與 B/L 必填）

### 4. 重量／材積
- Packing List 的 N.W. / G.W. / CBM = B/L 上的對應數字（容差 ±2%）

### 5. 客戶資訊一致性
- Invoice Buyer = Packing List Buyer = B/L Consignee（除非另有 Notify Party 約定）
- 地址完全一致

## 常見錯誤型態

1. **Invoice 金額與 PL 不符**：通常因有換貨／補貨未更新
2. **PL 入數與 Invoice 數量不符**：人為計算錯誤
3. **B/L 重量與 PL 重量差異 >5%**：通常為估重不準
4. **HS Code 不一致**：分公司報關員填寫疏忽

## 報關前 Checklist

- [ ] Invoice 總金額 = PL 總金額
- [ ] Invoice 數量 = PL 數量
- [ ] Cartons 數一致
- [ ] N.W. / G.W. 容差 ±2%
- [ ] HS Code 一致
- [ ] Buyer / Consignee 一致
- [ ] PO Number 一致
