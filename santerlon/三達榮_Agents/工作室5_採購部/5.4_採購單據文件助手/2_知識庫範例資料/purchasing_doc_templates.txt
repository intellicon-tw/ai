# 採購單據範本庫

## 範本 1：標準採購訂單 PO

```
SANDARONG CO., LTD.
PURCHASE ORDER

PO No.: PO-[YYYYMM-NNN]
Date: [YYYY-MM-DD]

Supplier: [Supplier Name]
Attn: [Contact]
Email: [email]

| Item | Material Code | Description | Qty | Unit | Unit Price (NT) | Amount (NT) |
|------|---------------|-------------|-----|------|-----------------|-------------|
|      |               |             |     |      |                 |             |

Total: NT [Total]

Delivery Date: [YYYY-MM-DD]
Delivery Place: [Sandarong 倉庫]
Payment: T/T 30 days after receipt of goods
Remarks: [Quality requirements / Special instructions]

Authorized by:
Buyer: __________
Manager: __________
```

## 範本 2：物料合約

```
原物料採購合約

甲方：三達榮股份有限公司
乙方：[供應商名]

雙方協議事項：
1. 物料規格：[詳述]
2. 月度採購量預估：[XX] [單位]
3. 單價：NT [X] / [單位]
4. 付款條件：T/T 30 天
5. 交貨條件：每月分批交貨，需配合 Sandarong 排程
6. 品質要求：依 Sandarong 品質規範
7. 違約條款：[違約方賠償條款]
8. 合約有效期：[YYYY-MM-DD] 至 [YYYY-MM-DD]

甲方簽章：__________  乙方簽章：__________
```

## 範本 3：訂金通知

```
訂金通知 Deposit Notice

PO No.: PO-[XXX]
Supplier: [Name]
Total PO Amount: NT [X]
Deposit Amount (30%): NT [Y]
Deposit Wire Date: [YYYY-MM-DD]
Remaining: NT [Z] (settled upon receipt of goods)
```
