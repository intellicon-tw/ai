# 三達榮公司 EgentHub Agent 完整 POC 開發包

本目錄為三達榮公司 AI Agent 導入的完整 POC 開發包，涵蓋 **9 個工作室、41 隻 Agent**，從生產製造、業務、研發、品管、採購、倉儲物流、管理部，到總經理辦公室與跨部門知識中心。

## 重點：教學手冊

請優先打開根目錄的 `三達榮_AI_Agent_POC_教學手冊.html`，一頁讀完全部 41 隻 Agent 的完整內容（含可一鍵複製的 Prompt、知識庫設定、驗測情境、上線指南）。

## 目錄結構

每個 Agent 各自獨立一個子目錄，包含四個檔案（依推薦閱讀順序編號）：

1. `1_prompt.txt`：完整提示詞與元資料，可直接複製貼到 EgentHub Agent 設定頁。
2. `2_知識庫範例資料/`：知識庫範例（csv 表格、txt 文本），可直接上傳到 EgentHub 對應的知識庫類型。
3. `3_驗測情境.txt`：包含測試資料集與驗測情境的完整驗證文件。
4. `4_正式上線前的準備指南.txt`：本 Agent 的上線 checklist，含知識庫設定、模型選擇、權限、驗測、監控、異常處理。

## 關於示範資料的重要說明

工作室 2-9 中，多數 Agent 的知識庫範例為「智慧方案顧問團隊預先生成的示範資料」，目的是讓客戶在工作坊上立即看到 Agent 運作效果。

**這些示範資料必須在實際上線前由三達榮對應部門替換為真實資料**，AI Agent 才能真正用於業務決策。每隻 Agent 的「4_正式上線前的準備指南.txt」中都有明確指引：
- 哪些檔案需要替換
- 替換後維持相同檔名與欄位結構
- Agent 提示詞無須修改

工作室 1 的 5 隻 Agent 使用客戶提供的真實資料（2026 年 3 月生產日報表、排程甘特、外發排程、訂單主檔等），可以直接驗測與上線。

## 建議推進節奏

- **第一波（已完成）**：工作室 1 — 全部已上線就緒
- **第二波（1-2 個月）**：工作室 2 業務部 + 工作室 4 局部 + 工作室 5 局部
- **第三波（3-6 個月）**：工作室 3 研發 + 工作室 4 其餘 + 工作室 5 其餘
- **第四波（6-12 個月）**：工作室 6 倉儲 + 工作室 7 管理部
- **長期（12 個月以上）**：工作室 8 + 工作室 9（ROF 級應用）
