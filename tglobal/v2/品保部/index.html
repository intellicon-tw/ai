文件生成型

 C · 品保-2建議候選（推測）

客訴 RMA 分類與初判助理

知識庫問答型（結構化查詢）

客服或品保接獲客訴時，需手動翻查 RMA 歷史紀錄，判斷缺陷類別與可能原因，回覆速度慢，且新進同仁容易判斷不一致。

 C · 品保-3建議候選（推測）

ISO/QMS 文件與法規查詢助理

知識庫 RAG 問答型

同仁查 ISO 條文或 RoHS/REACH 法規要求時，需翻閱大量文件，耗時且找不到正確段落，尤其面對客戶詢問時常無法快速回覆。

 C · 品保-4建議候選（推測）

承認書公版套用生成助理

文件生成型

每次新料號或新客戶需要提交承認書，品保工程師手動套公版填寫，欄位多且容易漏填，來回確認耗時。

 C · 品保-5建議候選（推測）

供應商 QCDS 月評比彙整助理

資料分析型

品保每月需彙整各供應商 QCDS（品質/成本/交期/服務）評分、產出排名表與等級分類，目前以 Excel 手動計算加權分數，格式不統一、容易算錯，月底報告趕時間。

 C · 品保-6建議候選（推測）

SPC 品質數據異常摘要助理

資料分析型

品保工程師每日需手動查看 SPC 管制圖，辨識超出 UCL/LCL 的 CTQ 項目，現在數據分散在 Excel，沒有自動摘要，異常發現不及時。
