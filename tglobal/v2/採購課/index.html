採購課任務盤點轉化的 POC 解題示範（共 3 題）

共 3 款 Agent POC分級 A🟢3 / B🟡0 / C🟠0

採購課 3 題都是典型的資料比對與通知自動化，場景具體、邏輯清楚，可直接開發。建議補上：ERP 進貨/採購單匯出範例、供應商聯絡資料與催交範本。 以下每一頁對應一個需求，點進去可看到：需求與痛點 → 開發思路 → 完整提示詞 → 知識庫設計 → 建置流程 → 驗測情境。

#01A

採購到貨資訊自動通知

採購供應與倉儲物流（A 立即可開發）

幫採購人員到貨後自動產生通知給倉庫/品保避免漏發,每次10~30分鐘、每月10~30次、1~3人。

 #02A

採購單交期追蹤與逾期提醒

採購供應與倉儲物流（A 立即可開發）

幫採購人員追蹤供應商交期、產出即將到期/逾期清單與催交範本,每次30分鐘~1小時、每月10~50次、1~3人。

 #03A

請購單轉採購單進度追蹤

採購供應與倉儲物流（A 立即可開發）

幫採購人員追蹤請購是否已轉採購/下單/到貨,產出進度表減少被反覆追問,每次30分鐘~1小時、每月10~40次、1~3人。
