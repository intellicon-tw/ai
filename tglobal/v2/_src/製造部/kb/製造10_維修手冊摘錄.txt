========================================
第一章：WC-101 / WC-102 切割機系列
========================================

【1-1 主軸系統維護】
主軸最高轉速 2500 RPM（WC-101：2000 RPM 標準、WC-102：2500 RPM 標準）。
正常電流範圍：WC-101 主軸 8-12A；WC-102 主軸 10-14A。
超過 15A 觸發 ALM-001 過電流保護，立即切斷電源。
過電流常見原因：
  1. 刀具鈍化導致切削阻力增大 → 更換刀具（每 5000 次或刀刃目視鈍化）
  2. 主軸軸承磨損 → 振動量測 > 0.1mm/s 時需更換（每季點檢）
  3. 進給速設定過高 → 進給速上限 80mm/s（TIM 類），超限觸發 ALM-006
步驟：觸發 ALM-001 後 → ①切斷主軸電源 → ②量測電流（Clamp Meter）→ ③判斷原因 → ④排除後點動試運轉確認

【1-2 刀具管理】
標準刀具壽命：5000 次切割（TIM 導熱矽膠類）；3000 次（GRP 石墨類較磨刀）。
觸發 ALM-005 時安排計畫換刀，不強制停機。
換刀步驟：①停機鎖定 → ②鬆刀把螺絲 → ③更換並鎖緊（扭力 8 N·m）→ ④試切 5 片確認邊緣 → ⑤填寫換刀記錄

【1-3 進給伺服異常 ALM-006】
原因：伺服驅動器過熱、編碼器故障、CNC 訊號中斷。
處理：①立即停機②確認驅動器指示燈（紅色=過熱/閃爍=Err）③散熱風扇確認④重新上電測試⑤若持續告警聯絡機電維修

第二章：WC-103 / WC-104 壓合機系列
========================================

【2-1 液壓系統】
標準工作壓力：WC-103（均溫板）5.0 Bar；WC-104（石墨）4.0 Bar。
ALM-002 觸發條件：壓力低於設定值 15% 持續 30 秒。
常見原因：
  1. 液壓油管洩漏 → 目視檢查各接頭，補緊或更換油管
  2. 液壓油位不足 → 確認油箱油位計，不低於 1/3
  3. 液壓泵磨損 → 排除上兩項後仍壓力不足，聯絡液壓維修
步驟：ALM-002 → ①確認允許限速繼續或停機 → ②檢查油位與管路 → ③排氣確認 → ④調整溢流閥設定值

【2-2 加熱系統】
加熱器類型：PTC 加熱板（WC-103 設定 80°C；WC-104 設定 60°C）。
ALM-003 觸發：實際溫度偏差設定值 ±8°C 超過 2 分鐘。
常見原因：
  1. 加熱器老化（輸出功率下降）→ 量測加熱器電阻，正常值 48-52Ω，異常時更換
  2. 溫控 PID 參數飄移 → 重新校正 PID（聯絡設備廠商）
  3. 熱電偶損壞 → 更換熱電偶（K 型，250°C 量程）

【2-3 真空吸附 ALM-007】
真空壓力低於 -0.6 Bar 觸發警告。
檢查項目：真空泵油位 → 吸盤磨損 → 管路破損。

========================================
第三章：WC-105 / WC-106 塗布機系列
========================================

【3-1 刮刀與塗布速度】
標準刮刀間隙：TPA 類 0.20±0.02mm；TPC 類 0.50±0.03mm。
ALM-002 類似概念：模溫偏高時塗布黏度降低，易造成不均。
超溫處置：①確認加熱板是否接觸良好 → ②清潔加熱板接點 → ③確認 PID 設定值 → ④若持續偏高停機等待降溫再開

【3-2 刮刀更換 SOP】
刮刀壽命：每 2000m 塗布長度更換（計數器讀取）。
步驟：①停機上鎖 → ②鬆開刮刀固定螺絲（M6 x 4pcs）→ ③取出舊刮刀 → ④確認刀片無缺口 → ⑤安裝新刀片鎖緊 → ⑥Feeler Gauge 確認間隙

========================================
第四章：WC-107 / WC-108 沖壓機系列
========================================

【4-1 壓力系統】
WC-107（鋁片）：標準 80 Bar；WC-108（銅片）：標準 100 Bar。
ALM-002：壓力偏差 ±10 Bar 以上觸發。
上限超標（ALM-004 型態）：立即停機，確認壓力設定輸入值是否正確。
常見手誤：換線時輸入上一個料號壓力值 → 加強換線 Double Check SOP。

【4-2 油溫管理 ALM-010】
液壓油正常溫度：35-55°C。
超過 60°C 觸發 ALM-010，確認冷卻水量與水溫（進水溫度應 < 25°C）。
冷卻水不足：聯絡廠務確認冷卻塔水量。

第五章：WC-112 雷射切割機
========================================

【5-1 雷射功率穩定性】
標準功率：50W（石墨精切）。
ALM-008 觸發：功率波動 > ±5W 持續 10 秒。
原因：雷射電源老化 → 更換雷射電源模組；光路污染（保護鏡片） → 清潔或更換保護鏡。
清潔步驟：①停機 → ②用無塵紙+異丙醇 → ③單向擦拭 → ④目視確認無殘留

【5-2 夾具定位 ALM-011】
定位感測器：磁性感應開關（NC 型）。
觸發時確認：①工件是否放正 ②感應開關是否損壞 ③夾具彈簧力道是否足夠

========================================
第六章：緊急停機共用 SOP（ALM-014）
========================================

ALM-014 為聯鎖緊急停機，所有機台皆可能觸發。
觸發來源：安全光幕被遮斷、緊急拉繩、設備連動聯鎖。
標準處理程序：
  1. 廣播確認現場人員安全
  2. 確認觸發源（查閱 PLC HMI 畫面 → 「聯鎖狀態」頁面）
  3. 消除觸發原因（人員離開光幕區、安全門關閉等）
  4. 點按「聯鎖復位」鍵後確認無異常
  5. 低速點動試運轉確認正常後恢復生產
  6. 填寫「緊急停機記錄表」
