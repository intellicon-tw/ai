高柏科技 AI Agent 開發實務工作坊 · 會計處

會計處任務盤點轉化的 POC 解題示範（共 12 題）

共 12 款 Agent POC分級 A🟢7 / B🟡2 / C🟠3

會計處盤點非常完整——12 題每題都寫清楚輸入、輸出與處理邏輯，多筆還附了附件範例，是全公司填得最扎實的部門，可直接進入開發。建議補上：各附件的實際樣本（逾期明細、請款憑證、科餘表、成本表）與會計科目對照表，AI 就能精準對帳入帳。 以下每一頁對應一個需求，點進去可看到：需求與痛點 → 開發思路 → 完整提示詞 → 知識庫設計 → 建置流程 → 驗測情境。

#01A

逾期應收帳款 Mail 發送

財務會計與成本管理（A 立即可開發）

幫會計人員每月彙整逾期應收帳款並通知各業務，每次約3~4小時、每月1次、2人；希望AI自動彙整逾期清冊並個別發信提醒負責業務。

 #02A

財報比較分析

財務會計與成本管理（A 立即可開發）

幫會計人員做各期財報差異分析與管理報告摘要，每次6~8小時、每月1次、2人。

 #03A

應收帳款收款沖帳

財務會計與成本管理（A 立即可開發）

幫會計人員每個工作日做收款沖帳，從收款單/客戶清單/應收明細產出沖帳資訊，每次0.5~3小時、4人。

 #04A

科目餘額明細表

財務會計與成本管理（A 立即可開發）

幫會計人員月結後產出科餘表，以資產負債表餘額為基礎整理各科目組成明細(刪除立沖)，每次8~16小時、5人。

 #05A

成本分析

財務會計與成本管理（A 立即可開發）

幫會計人員分析各成品料號各月單位成本(材料/人工/製費)變動並做差異說明，每次8小時、2人。

 #06A

預付費用 / 預付保險費攤提

財務會計與成本管理（A 立即可開發）

幫會計人員依每月預付費用/保險費明細，將新增項目依期數自動分攤每月金額，每次8小時、2人。

 #07A

會計憑證翻譯(越語、日語)

財務會計與成本管理（A 立即可開發）

幫會計人員翻譯越南/日本子公司請款憑證供主管審核，每次0.2小時、每月約10次、2人。

 #08B

差旅費請款單據整理與審核

財務會計與成本管理（B 需前置(數位化/介接)）

幫會計人員整理審核員工差旅憑證、分類計算並套差旅規範確認可請款金額，每次16小時、2人。

 #09B

前端請款單據入帳

財務會計與成本管理（B 需前置(數位化/介接)）

幫會計人員將前端紙本/PDF/圖片請款單據數位化、檢核合法性並產出入帳分錄建議，每次4小時、每月22次、3人。

 #10C

法令遵循 / 稅務諮詢(台灣+國外)

財務會計與成本管理（C 人機協作輔助）

幫會計人員就交易型態判斷法規適用並給合規建議,每次0.5~1小時、每月多次、2人。

 #11C

合併報表

財務會計與成本管理（C 人機協作輔助）

幫會計人員彙整子公司報表、換算幣別並做關係人/長投沖銷產出合併報表,每次4小時、1人。

 #12C

財務預測(三表)

財務會計與成本管理（C 人機協作輔助）

幫會計人員依各單位評估彙整產出損益表/資產負債表/現金流量表預測與情境模擬,每次5小時、1人。
