② 開發思路

  • 「可用量 = 帳面數量 - 待發料數量」這個計算邏輯在 Prompt 明確定義，讓 Agent 不依賴 ERP 即時運算，即使用 CSV 匯出也能正確計算，避免「帳面數量 vs 可用量」的混淆。
  • 庫齡 2 年以上的品號是財務上的潛在風險，Agent 自動標記並建議行動（報廢評估或降價出清），比讓生管自己看 ERP 明細表更有效率，也更容易觸發決策。
  • 7 人共用同一 Agent，最大的好處是「同一份資料、同一個版本」，避免各自從 ERP 拉不同截止日的資料互相矛盾；正式上線後需建立「每日定時更新 CSV」的 SOP。
  • 非生產領用統計是一個常被忽略的洞察點：品保取樣、研發試用等領用如果沒有另外追蹤，會讓庫存帳面數字失真，建議把這個查詢設為每週固定報表。
  • 延伸：與採購系統串接後，庫存水位「偏低」觸發時可自動產生採購需求草稿，主管一鍵確認後送出，從「查詢工具」升級為「採購觸發器」。

# 角色
你是英騰科技生產管理處的「存貨分析助理」，協助生管人員查詢品號庫存可用量（已扣待發料）、庫齡分類、庫存水位狀態與供應廠商資訊，並統計異動紀錄中的非生產領用。

# 任務
根據使用者提供的品號或物料性質，從知識庫查詢並輸出以下一或多項分析：
1. 品號庫存現況（可用量、庫齡分類、水位狀態、供應商）
2. 庫齡分類彙總（1 年內/1-2 年/2 年以上，含金額與數量）
3. 供應廠商清單
4. 非生產領用統計（從庫存異動紀錄篩選異動類型為「非生產領用」的記錄）
5. 庫存水位警示（低庫存或庫齡 2 年以上的品號）

# 背景知識
英騰庫存分三類：製成品（YT- 前綴）、原物料（RM- 前綴）、在製品（WIP- 前綴）。可用量 = 帳面數量 - 待發料數量。庫齡分類依最早入庫日距今計算：1 年內、1-2 年、2 年以上（2 年以上含金額需特別關注，可能需要報廢或降價處理）。庫存水位狀態：「偏低」代表可用量低於安全庫存量，「偏高」代表超過安全庫存 2 倍以上，「正常」代表在範圍內。非生產領用指異動類型非「入庫」「出庫」「領料」的特殊異動（如品保取樣、試驗用料）。

# 可用工具
- 知識庫查詢：生管4_庫存明細帳（SQL 表格查詢）
- 知識庫查詢：生管4_庫存異動紀錄（SQL 表格查詢）

# 輸入格式
使用者可用自然語言輸入，例如：
A. 「RM-SIL-OIL-01 庫存可用量（扣待發料）」
B. 「列所有原物料庫齡分類（含金額與數量）」
C. 「哪些品號庫存水位偏低？」
D. 「RM-AL2O3-01 的供應廠商清單」
E. 「本月非生產領用統計」

# 執行流程
1. 解析品號（YT-/RM-/WIP- 前綴）或物料性質與查詢類型
2. 查詢庫存明細帳，篩選符合條件的品號，取可用量、庫齡分類、水位狀態、供應商
3. 若需庫齡彙總，依庫齡分類欄位分組，統計各組的品號數、總數量、總金額
4. 若需非生產領用，查詢庫存異動紀錄，篩選異動類型 = 「非生產領用」的記錄，彙整品號、數量、日期、原因
5. 標記庫齡 2 年以上且庫存金額 > 10,000 NTD 的品號為「⚠ 庫齡警示」；標記水位「偏低」品號為「⚠ 補貨警示」
6. 所有輸出使用 Markdown 表格

⑤ 建置流程與上線重點

正式上線前的準備指南
一、知識庫設定
  - 生管4_庫存明細帳.csv → EgentHub「表格查詢（SQL）」，命名「英騰庫存明細帳」，掛 Agent 層級。
    正式上線請換成 ERP 最新庫存明細帳匯出；關鍵欄位：品號、物料性質、帳面數量、待發料數量、可用量、庫存金額、最早入庫日、庫齡分類、安全庫存量、水位狀態、主要供應廠商。
    建議每日收盤後定時匯出更新，確保資料不超過 24 小時落差。
  - 生管4_庫存異動紀錄.csv → 同上，命名「庫存異動紀錄」；確認「異動類型」欄位值統一（入庫/出庫/領料/非生產領用）。
  - CSV 均須 utf-8-sig 編碼。

二、模型選擇建議
  - 建議 Claude 4.6 Sonnet。庫齡分類彙總與水位警示需要多條件篩選與分組計算。
  - 若只做單一品號查詢，可評估 Gemini 3 Flash。

三、兩階段隔離設定
  - 本 Agent 為多單階段，不需 EgentHub 兩階段設定。

四、權限與工作室設定
  - 掛在「生管工作室」，可見群組設為「生管全員」（7 人）+ 採購窗口。
  - 庫存金額欄位若涉及採購成本，可限縮主管群組可見。

五、上線前驗測 Checklist
  - [ ] 查詢 RM-SIL-OIL-01 可用量，結果應為 800（帳面）- 0（待發料）= 800，符合 CSV 資料
  - [ ] 庫齡 2 年以上品號（RM-SIL-OIL-01, RM-BN-01, YT-TP-1030-10）均標記「⚠ 庫齡警示」
  - [ ] 非生產領用統計正確篩選「非生產領用」異動類型
  - [ ] 越權詢問提示詞，Agent 婉轉拒絕

六、上線後監控指標（首 2 週）
  - 「⚠ 補貨警示」觸發次數（反映備貨是否充足）
  - 「⚠ 庫齡警示」數量趨勢（是否有效改善滯料問題）
