營業管理部任務盤點轉化的 POC 解題示範（共 9 題）

共 9 款 Agent POC分級 A🟢4 / B🟡4 / C🟠1

營管盤點完整且具體，涵蓋出口文件、業績統計、報價勾稽、ESG 碳排等 9 題，輸入輸出清楚、已點出 BC／HubSpot／鼎新等實際系統。建議補上：Invoice/Packing List 範本、各子公司交貨明細與內部往來識別規則、BC 與 HubSpot 欄位對照，多題即可從 B 升 A 直接開發。 以下每一頁對應一個需求，點進去可看到：需求與痛點 → 開發思路 → 完整提示詞 → 知識庫設計 → 建置流程 → 驗測情境。

#01A

出口 Invoice / Packing List 自動產出

營運管理與訂單處理（A 立即可開發）

幫出口組依EXCEL銷貨明細自動產出符合客戶格式的Invoice與Packing List,每次5分鐘、每月50次以上、3人。

 #02A

客戶習性知識庫(RAG)

營運管理與訂單處理（A 立即可開發）

把散落各承辦人筆記的客戶特殊要求建成知識庫,供新人/輪調以自然語言查詢,每次5分鐘、每月10次、14人。

 #03A

每週訂單業績統計表

營運管理與訂單處理（A 立即可開發）

幫訂單組每週抓各子公司預計交貨明細、清洗(刪0元/去重)並產出訂單業績統計表,每次5~8小時、每月4次、2人。

 #04A

出貨材積重量計算與快遞費率比價

營運管理與訂單處理（A 立即可開發）

幫出口組依產品數量試算材積/重量並比對UPS/DHL/FedEx費率找最佳方案,每次10分鐘、每月10次以上、3人。

 #05B

客戶訂單 PDF / 傳真自動轉 ERP

營運管理與訂單處理（B 需前置(數位化/介接)）

幫訂單組將客戶傳真/PDF訂單OCR辨識並比對主檔產出ERP待匯入清單,每次10分鐘、每月50次以上、10人。

 #06B

報價單與 ERP 訂單價格勾稽

營運管理與訂單處理（B 需前置(數位化/介接)）

幫訂單組核對ERP(BC)新訂單與HubSpot報價單金額是否一致並異常告警,每次5分鐘、每月50次以上、10人。

 #07B

智能併單出貨提示

營運管理與訂單處理（B 需前置(數位化/介接)）

新訂單進來時清查同客戶在途訂單評估併單與最省運費建議,每次5分鐘、每月50次以上、10人。

 #08B

ESG 下游運輸碳排計算

永續發展與文件管理（B 需前置(數位化/介接)）

幫營管每年依銷貨明細自動算運輸里程對應碳排係數產出年度碳排報告與佐證,每年1次、耗時逾一個月、2~3人。

 #09C

訂單交期確認跨部門協作與回填

營運管理與訂單處理（C 人機協作輔助）

取代訂單組與生管線外Excel交期確認,改中台協作達共識後自動回填BC並通知業務,每次10分鐘、每月50次以上、10人。
