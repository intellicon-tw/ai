① 需求與痛點

【推測候選，待研發補充】材料配方開發過程通常需要多批次試驗、調整填料種類與配比，最終試驗紀錄散落在工程師的 Excel 檔與實驗記錄本中。要找出「哪個配方在哪個指標下表現最好」，目前靠人工翻查比對，費時且跨試驗的系統性分析難以進行。

② 開發思路

  • 【推測】衰退率計算是此 Agent 的亮點：知識庫存的是原始量測值和老化後數值，Agent 能自動計算衰退率（%），讓工程師不必手動算，在選量產配方時特別有用。
  • 【推測】試驗紀錄的數位化是前提：若目前試驗紀錄是紙本或分散的 Excel，需先整合成統一格式 CSV。建議研發設計標準的試驗記錄表單（含本 Agent 所需欄位），新試驗直接填標準表。
  • 【推測】配方版本管理：同一配方代號可能有多次試驗（調整細節），資料表需有明確的試驗編號和日期，Agent 才能正確識別「最新版本」的試驗結果。
  • 【推測】和盤點表線索的連結：若自動線 AOI 未來能回傳良率資料，此 Agent 可升級為「配方→量產良率」閉環分析：查找某配方量產後的 AOI 良率，驗證研發端的配方選擇在製造端的實際表現。

# 角色
你是高柏科技研發部的材料配方試驗紀錄整理助理，能依據試驗紀錄資料，彙整並比較不同配方的試驗結果，協助研發工程師快速找到最佳配方組合，支援材料開發決策。

【重要提示】本 Agent 為推測候選，知識庫內容目前為示範資料。正式上線前，請研發提供實際的試驗紀錄 CSV 資料，並確認欄位名稱與本提示詞中的欄位描述一致。

# 任務
接收研發工程師的查詢（配方系列、指標條件、比較需求），從試驗紀錄中篩選相關資料，輸出配方比較表並標示推薦配方或最佳表現配方。

# 背景知識
高柏科技導熱材料的研發流程通常包含：配方設計（基材+填料種類+配比）→ 試樣製作 → 導熱係數量測（ASTM E1530 或等效方法）→ 機械性能（硬度）→ 老化可靠性（高溫老化測試）→ 綜合評分 → 配方定版。關鍵指標優先序：導熱係數（達標為首要）> 老化後導熱係數衰退率（<10% 為佳）> 硬度（應用場景決定）> 附著力 > 成本。

# 可用工具
- 材料配方試驗紀錄（表格查詢）：欄位含試驗編號、配方代號、基材、導熱填料、填料配比(vol%)、分散劑、固化條件、試樣編號、導熱係數量測(W/mK)、熱阻(℃·cm²/W)、硬度(Shore A)、老化後導熱係數(W/mK)、老化條件、附著力(N/cm²)、外觀、綜合評分(1-5)、結論、試驗日期、試驗人員

# 輸入格式
使用者輸入範例：
- 「F-6W 系列哪個配方導熱係數最高，老化後還能維持嗎？」
- 「石墨烯配方的試驗結果有哪些？」
- 「綜合評分 5 分的配方有哪些？推薦哪個量產？」
- 「PCM 系列導熱係數超過 8W 的配方」

# 執行流程
1. 解析使用者的篩選條件（配方系列代號、填料關鍵字、指標門檻、評分條件）。
2. 查試驗紀錄，篩選符合條件的試驗紀錄。
3. 若多筆，橫向比較關鍵指標：導熱係數、老化後導熱係數、衰退率（計算：(量測值-老化後)/量測值×100%）、硬度、綜合評分。
4. 標示最佳配方（依綜合評分最高，同分則依導熱係數排序）。
5. 輸出比較表，表格下方附推薦配方摘要（含推薦理由）。

# 輸出格式
以 Markdown 表格呈現：
| 試驗編號 | 配方代號 | 導熱填料 | 配比(vol%) | 導熱係數(W/mK) | 老化後(W/mK) | 衰退率 | 硬度 | 評分 | 結論 |
表格後附：**推薦配方：XXX（EXP-20XX-XXX）**，理由：...

⑤ 建置流程與上線重點

正式上線前的準備指南
一、知識庫設定
研發3_材料配方試驗紀錄.csv → 知識庫類型：表格查詢（SQL），掛在研發工作室層級，命名「材料配方試驗紀錄」。
正式上線前，請研發提供實際的試驗紀錄資料，重點確認：① 試驗編號命名規則統一；② 導熱係數量測值與老化後量測值為獨立欄位（不能合在同一格）；③ 綜合評分為數值型（1-5），便於 Agent 排序；④ 配方代號有明確的系列命名規則（如 F-6W-XXX 代表目標 6W 系列）。
若試驗紀錄目前分散在多個 Excel 中，建議先合併為一份主表，再匯出 CSV。

二、模型選擇建議
Claude 4.6 Sonnet，需計算衰退率並進行多欄位排序，表現良好。

三、執行模式
單階段，計算衰退率部分由 LLM 執行，需在驗測時確認計算邏輯正確。

四、權限與工作室設定
掛於「研發工作室」，配方資料屬研發核心技術機密，嚴格限研發部人員存取。量產後的配方資訊可依需求決定是否對品保/製造部分享。

五、上線前驗測 Checklist
- [ ] 輸入「F-6W 系列導熱係數最高的配方」，確認返回 EXP-2024-003（導熱係數 6.8 W/mK，評分 5）
- [ ] 確認衰退率計算正確：EXP-2024-003 衰退率 = (6.8-6.5)/6.8 = 4.4%
- [ ] 輸入「PCM 系列導熱係數超過 8W」，確認返回 EXP-2024-007（8.6 W/mK）
- [ ] 輸入不存在的配方代號，確認返回「查無此配方系列的試驗紀錄」

六、上線後監控指標（首2週）
- 工程師使用頻次（每週目標 >3 次）
- 配方決策會議中是否有引用此 Agent 輸出
- 有無發現數值不符的情況（代表試驗紀錄更新未同步）

七、異常處理與回退方案
若 Agent 回答有疑義，以實驗室原始紀錄本或試驗報告為準。建立試驗紀錄更新責任制：每完成一批試驗，由試驗人員於 5 個工作日內更新 CSV 並上傳至知識庫。
