C · 研發-1建議候選（推測）

熱模擬報告摘要與比較助理

資料分析型

【推測候選，待研發補充】研發人員在進行材料選型時，往往需要橫向比較同一客戶案件下多次熱模擬的結果（不同材料、不同厚度），才能判斷哪個方案最佳。目前比較過程多靠人工整理散落的模擬報告，費時且容易遺漏關鍵數值（熱阻、最高溫、溫差）。

 C · 研發-2建議候選（推測）

研發技術文件與材料規格知識庫問答

知識庫問答型（全文查找 + 向量 RAG）

【推測候選，待研發補充】研發工程師需要快速查詢材料規格（導熱係數、耐溫範圍、應用限制）與過往技術文件（客戶選型紀錄、測試報告結論），目前依賴個人記憶或翻找資料夾，新進同仁特別吃力，跨案件的知識累積也難以系統化。

 C · 研發-3建議候選（推測）

材料配方與試驗紀錄整理助理

資料分析型

【推測候選，待研發補充】材料配方開發過程通常需要多批次試驗、調整填料種類與配比，最終試驗紀錄散落在工程師的 Excel 檔與實驗記錄本中。要找出「哪個配方在哪個指標下表現最好」，目前靠人工翻查比對，費時且跨試驗的系統性分析難以進行。

 C · 研發-4建議候選（推測）

競品散熱規格比對助理

知識庫問答型（結構化查詢 + 比較分析）

【推測候選，待研發補充】業務接到客戶詢問「你們的產品跟 Bergquist GP6000 比有什麼優勢？」或「Honeywell PTM7950 的熱阻是多少，你們有沒有對標產品？」時，需要研發工程師協助查找競品規格並整理比較，費時且回覆速度慢，影響商機把握。
