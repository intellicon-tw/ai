C · 行銷-1建議候選（推測）

行銷預算合約 PDF 解析與建檔助理

文件解析型（PDF 上傳→結構化建檔，兩階段）

每次收到廠商合約或報價單 PDF，需人工翻閱抄錄活動起訖日、付款截止日、未稅含稅金額、各細項，再手動填入預算追蹤表，容易漏填或輸錯，遇展覽旺季同時處理多份文件壓力更大。

 C · 行銷-2建議候選（推測）

行銷預算執行追蹤與超支預警助理

資料分析型（彙整執行率、超支預警、費用佔比）

行銷各活動的預算與實際花費分散在不同簽呈與記帳系統，月底要彙整執行率時需人工加總計算，超支風險往往到月底才發現，且各通路費用佔比難以快速視覺化，影響下次預算分配決策。

 C · 行銷-3建議候選（推測）

產品 DM 與行銷文案生成助理

生成型（依規格與賣點生成行銷文案／社群貼文）

【推測題】行銷撰寫散熱產品的 DM 文案或社群貼文時，需要先查散熱係數、規格、認證等技術資料，再轉化為目標客群看得懂的語言；往往工程術語翻譯費時，也難以保持各產品文案風格一致，旺季同時準備多款產品更新 DM 時壓力倍增。

 C · 行銷-4建議候選（推測）

展覽籌備時程與待辦清單助理

生成型（依展期反推籌備節點，產出甘特待辦）

【推測題】每次準備展覽，行銷需要從零開始整理籌備清單：攤位費截止、大圖送件、展品出貨、訂機票飯店…各有不同的前置天數，仰賴資深同仁的記憶或過去零散的 Excel，容易遺漏關鍵節點，尤其是首次參展的展覽或新人接手時風險更高。
