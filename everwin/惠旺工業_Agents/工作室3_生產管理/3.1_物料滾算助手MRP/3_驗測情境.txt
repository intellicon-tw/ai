# 物料滾算助手MRP - 驗測情境

本檔案分為兩部分：
- **Part A — 測試資料集**：3 筆可直接貼到 EgentHub 對話框做手動驗測的自然語言輸入。
- **Part B — 驗測情境**：3 個情境的預期輸出與驗證重點，作為手動或半自動回歸驗測依據。

---

## Part A — 測試資料集

### 測試 #1：依訂單滾算

```
幫我滾算訂單 A20260101 的物料需求，未交數量 1000 PCS，需求交期 2026-05-30。
```

### 測試 #2：多成品同時滾算

```
下個月 (2026-06) 生產計畫：1000 把 EW-DX5 + 800 把 EW-3000-Pro，請列出採購清單與最遲下單日。
```

### 測試 #3：庫存足夠驗測

```
成品 EW-T50-15GA 要生產 500 PCS，會缺料嗎？
```

## Part B — 驗測情境

### 情境 #1：單訂單物料展開含採購最遲日

**輸入描述**

> 生管詢問訂單 A20260101 (EW-T50-15GA 未交 1000 PCS) 的物料需求。

**預期輸出**

> 輸出 5 個子件（P-FRAME-T50/P-BARREL-T50/P-SPRING-T50/P-TRIGGER-T50/P-SCREW-M4 各乘以 1000 + 損耗），對比庫存後標示淨需求與最遲下單日。

**驗證重點**

> 損耗率是否正確帶入、可用庫存是否含在途、最遲下單日推算是否扣除安全緩衝。

### 情境 #2：供應商集中度風險揭露

**輸入描述**

> 生管要求下個月 1000 EW-DX5 + 800 EW-3000-Pro 的採購清單。

**預期輸出**

> 正確展開兩個成品的子件需求，識別出旭東金屬 (V001) 同時被分配多個機身料號，標示集中度風險。

**驗證重點**

> 金額試算是否合理、是否正確識別瓶頸供應商。

### 情境 #3：庫存足夠不需採購

**輸入描述**

> 生管詢問 EW-T50-15GA 生產 500 PCS 是否會缺料。

**預期輸出**

> 展開後比對庫存，輸出「庫存足夠」清單為主，無需採購建議。

**驗證重點**

> 是否正確判斷不需採購、是否仍提供結餘數字給生管參考。
