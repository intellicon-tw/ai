# 物料滾算助手 MRP - 正式上線前的準備指南

本文件提供將「物料滾算助手 MRP」正式部署到 EgentHub、供生管團隊每週備料規劃使用的逐步準備清單。

## 1. 知識庫設定

### 1.1 BOM 展開表（表格查詢知識庫）

- **來源檔案**：`3_知識庫範例資料/BOM展開表.csv`
- **建議名稱**：「BOM 展開表 (BOM Master)」
- **層級**：Agent 層級
- **正式資料替換**：上線前由 PDM 系統匯出最新版本，並建立每月同步流程

### 1.2 零件主檔 / 庫存餘額表 / 在手訂單明細

- 同前述格式設定，皆為表格查詢知識庫。
- 庫存與在手訂單建議每日同步一次。

## 2. 模型選擇建議

Claude 4.6 Sonnet — 涉及 BOM 多層展開、庫存比對、最遲下單日推算的複合計算任務。

## 3. 兩階段隔離設定

本 Agent 為單階段執行，無須額外設定。

## 4. 權限與工作室設定

- 工作室：生產管理
- 可見群組：生管、採購（採購可查詢以提前掌握需求節奏）
- 資料敏感度：BOM 屬產品核心機密，限定生產相關人員

## 5. 上線前驗測 Checklist

- [ ] **情境 #1（單訂單物料展開含採購最遲日）**：必須通過
- [ ] **情境 #2（供應商集中度風險揭露）**：必須揭露瓶頸供應商
- [ ] **情境 #3（庫存足夠不需採購）**：必須正確判斷不需採購

## 6. 上線後監控指標（首 2 週）

- 每週滾算次數
- 缺料預警的命中率（提前發現 vs 事後火燒屁股）
- 採購最遲下單日的遵守率

## 7. 異常處理與回退方案

- 若 BOM 展開漏項，先檢查 BOM 表是否有新成品未建檔。
- 維運窗口：生管主管擔任 owner。
- 回退：失效時回到既有 ERP MRP 模組。

## 8. 種子團隊接手提示

延伸應用：「生產排程 AI 副駕駛」「跨成品物料共用優化 Agent」。本 Agent 為生管 ROI 級應用的核心，建議第二階段強化串接。
