# 客戶報價管理助手 - 正式上線前的準備指南

本文件提供將「客戶報價管理助手」從 POC 套件正式部署到 EgentHub 平台、在惠旺業務同仁日常查價與報價單初稿產出作業中穩定使用的逐步準備清單。

## 1. 知識庫設定

### 1.1 客戶報價單價表（表格查詢知識庫）

- **來源檔案**：`3_知識庫範例資料/客戶報價單價表.csv`
- **知識庫類型**：表格查詢知識庫（SQL）
- **建議名稱**：「客戶報價單價表 (Customer Price List)」
- **層級**：Agent 層級
- **欄位確認**：客戶代碼、客戶名稱、品號、品名、規格、幣別、標準單價、促銷單價、有效起日、有效迄日、業務負責人、報價備註
- **正式資料替換**：建議由業務助理在每次新報價簽核完成後同步更新，並保留歷史報價版本以供查詢

## 2. 模型選擇建議

- **主要推理**：Claude 4.6 Sonnet — 對自然語言查詢、折扣計算、合約到期提醒的綜合任務最穩。
- **大量輕量查詢場景**：可降配至 Gemini 3 Flash。

## 3. 兩階段隔離設定

本 Agent 為單階段執行，無須額外設定。

## 4. 權限與工作室設定

- 工作室：業務部
- 可見群組：業務同仁 + 業務助理 + 業務主管；建議**禁止**其他部門可見（報價屬機密）
- 算力點數初期建議：依業務同仁日查價次數估算，預期每人每日 5–10 次查詢

## 5. 上線前驗測 Checklist

- [ ] **情境 #4（單一客戶單品項查詢含到期提醒）**：必須通過
- [ ] **情境 #5（折扣試算正確性）**：必須通過，且必須同時揭露促銷價
- [ ] **情境 #6（即將到期合約查詢）**：必須通過

## 6. 上線後監控指標（首 2 週）

- 每日查價次數（目標：穩定增長到取代 Excel 翻找）
- 業務同仁滿意度（每週簡短問卷）
- 報價單初稿產出時間（基準 vs 上線後）

## 7. 異常處理與回退方案

- 若回傳價格與業務記憶不符，先確認知識庫版本是否最新。
- 維運窗口：業務助理 owner，負責每週同步報價變動。
- 回退：失效時回到既有 Excel 查價流程。

## 8. 種子團隊接手提示

延伸應用：「報價單初稿自動產出 Agent」「年度報價回顧 Agent」皆可以本 Agent 為起點。當客戶提出 RFQ 時，可結合此 Agent 與訂單解析 Agent 形成完整詢報價工作流。
