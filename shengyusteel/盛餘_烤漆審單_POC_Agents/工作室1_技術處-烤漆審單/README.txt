# 工作室1_技術處-烤漆審單

本工作室包含 4 個 Agent，共 4 個使用表格知識庫，2 個使用文件知識庫。

## Agent 清單

| 編號 | Agent 名稱 | 知識庫類型 | POC 方式 | 兩階段 |
|---|---|---|---|---|
| 1.1 | 烤漆訂單審查助手 | Agent 向量知識庫（解讀指南）＋表格查詢知識庫（解讀對照表/防呆規則/相似訂... | csv 資料表 | 否 |
| 1.2 | 業務備註解讀助手 | Agent 向量知識庫（解讀指南）＋表格查詢知識庫（解讀對照表） | csv 資料表 | 否 |
| 1.3 | 防呆規則查詢助手 | 表格查詢知識庫（防呆規則表） | csv 資料表 | 否 |
| 1.4 | 相似訂單比對助手 | 表格查詢知識庫（相似訂單歷史檔） | csv 資料表 | 否 |

## 各 Agent 子目錄

- `1.1_烤漆訂單審查助手/` — 1_prompt + 2_知識庫範例資料 + 3_驗測情境 + 4_正式上線前的準備指南
- `1.2_業務備註解讀助手/` — 1_prompt + 2_知識庫範例資料 + 3_驗測情境 + 4_正式上線前的準備指南
- `1.3_防呆規則查詢助手/` — 1_prompt + 2_知識庫範例資料 + 3_驗測情境 + 4_正式上線前的準備指南
- `1.4_相似訂單比對助手/` — 1_prompt + 2_知識庫範例資料 + 3_驗測情境 + 4_正式上線前的準備指南
