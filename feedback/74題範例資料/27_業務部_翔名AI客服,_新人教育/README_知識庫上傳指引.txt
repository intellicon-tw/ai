## 輸入（客戶提供）
翔名網頁可以有一個AI客服，協助回覆基本資訊。
 例如: 哪種產品詢價要聯絡哪位業務、公司販售產品...等資訊。

## 預期輸出（客戶提供）
針對特定關鍵字(詢價、銷售項目..) AI自動判讀並回覆資訊。

## 處理邏輯（客戶提供）
先建立一個專門的小程式，並建立特定的搜尋關鍵字，AI自動判讀並回覆。

## EgentHub 知識庫上傳指引

### 建議的知識庫檔案
- 公司規章辦法.pdf（員工手冊、SOP、流程說明）／類型：向量檢索
- 問答集 FAQ.pdf（歷年常見問題與回答）／類型：全文查找
- 聯絡通訊錄.xlsx（各部門窗口與分機）／類型：SQL 表格

### 上傳步驟
1. 登入 EgentHub 工作室
2. 進入「知識庫管理」
3. 依上述清單上傳對應檔案
4. 設定每個檔案的處理類型（向量／SQL／全文）
5. 在 Agent 設定中綁定本知識庫

### 注意事項
- L0 級機密圖檔（客戶 CAD、製程參數）請走地端 only 路由
- SQL 表格類請先整理為 CSV / XLSX 格式
- 全文文件建議 PDF / DOCX
- 向量檢索適合長文件、案例庫、SOP
