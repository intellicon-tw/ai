# 合約審核助手（題號 #15）知識庫範例

## 基本資訊
- 部門：光能事業部
- 提案人：陳琇琳
- 架構：A ‧ 文件解析／生成
- 建議知識庫類型：全文查找
- 出席時段：S8 15:00-15:40
- 共通專題：合約 ‧ 套用

## 輸入（客戶提供）
1.合約交期、履約特殊要求提醒(如TAF、熱處理…等)
 2.藍圖版次比對

## 預期輸出（客戶提供）
1.預交日前1週~1個月警示提醒(依合約執行要求而定)
 2.依採購項次提醒須含製程，以及需求時間
 3.確認已承製過以及目前是否有在製；以及零配件屬性與要求

## 處理邏輯（客戶提供）
1.依照合約內容建立schedule
 2.依schedule提醒各製程進度
 3.搭配schedule篩選在製與否產品之藍圖版次。

## 客戶提供的資料來源
依現行客戶資安要求，合約需嚴格保密，無法上拋，僅能以人工建立資料

## EgentHub 知識庫上傳指引

### 建議的知識庫檔案
- 翔名標準合約範本.pdf（NDA、Master、PO、Quality Agreement 等標準範本）／類型：全文查找
- 過往簽約案件庫.txt（歷史簽約案例含協商重點與踩雷經驗）／類型：向量檢索
- 法律風險條款庫.pdf（危險條款解釋、判例、業界經驗）／類型：向量檢索
- 客戶通訊錄與簽約主檔.xlsx（客戶代號、過往合約、法務窗口）／類型：SQL 表格

### 上傳步驟
1. 登入 EgentHub 工作室
2. 進入「知識庫管理」
3. 依上述清單上傳對應檔案
4. 設定每個檔案的處理類型（向量／SQL／全文）
5. 在 Agent 設定中綁定本知識庫

### 注意事項
- L0 級機密圖檔（客戶 CAD、製程參數）請走地端 only 路由
- SQL 表格類請先整理為 CSV / XLSX 格式
- 全文文件建議 PDF / DOCX
- 向量檢索適合長文件、案例庫、SOP
