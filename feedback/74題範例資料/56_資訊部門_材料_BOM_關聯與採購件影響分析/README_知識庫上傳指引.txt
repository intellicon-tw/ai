# 材料 BOM 關聯與採購件影響分析（題號 #56）知識庫範例

## 基本資訊
- 部門：資訊部門
- 提案人：羅銘輝
- 架構：C ‧ 資料分析
- 建議知識庫類型：SQL 表格
- 出席時段：S8 15:00-15:40
- 共通專題：— ‧ 獨立題

## 輸入（客戶提供）
- 輸入材料料號後，需人工查詢被哪些成品使用
 - 再由成品 BOM 往下展開採購件
 - 容易漏掉替代料或共用料
 - 缺料影響範圍難快速判斷

## 預期輸出（客戶提供）
- 輸入材料料號即可自動往上展第一階成品
 - 自動往下展該成品所有採購件
 - 顯示每項品號最新平均成本
 - 顯示材料用量與成本占比
 - 缺料影響成品分析
 - 可輸出 Excel BOM 關聯報表

## 處理邏輯（客戶提供）
1. 使用者輸入材料料號
 2. 系統查詢 Where-Used BOM（往上展）
 3. 找出第一階成品料號
 4. 由成品 BOM 往下展開採購件，並帶出用量資訊
 5. 彙整材料對成品的佔比
 6. 輸出分析報表

## 客戶提供的資料來源
- ERP BOM 資料表
 - 材料主檔
 - 採購件料號資料
 -品號最新平均成本資料
