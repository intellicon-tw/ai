# 市購標準件詢價（題號 #44）知識庫範例

## 基本資訊
- 部門：採購部
- 提案人：吳予茿
- 架構：D ‧ 流程自動化
- 建議知識庫類型：向量+SQL 混合
- 出席時段：XT 11:00-12:00
- 共通專題：手工重複 ‧ 套用

## 輸入（客戶提供）
客人BOM表

## 預期輸出（客戶提供）
EXCEL報價表

## 處理邏輯（客戶提供）
根據客人BOM表辨識規格，對照廠內料號，依近一年歷史進價報價，針對未曾詢價過的料號，搜尋該原廠網站並發出詢價EMAIL或於該公司網站留下詢價資訊，收到報價後，EMAIL收到報價後，整理成EXCEL表並發出給詢價人員

## EgentHub 知識庫上傳指引

### 建議的知識庫檔案
- 採購系統 MCP（採購單、供應商評等 API）／類型：MCP 工具
- 供應商主檔.xlsx（供應商代號、過往交期、品保）／類型：SQL 表格
- 詢價範本.pdf（詢價單格式、條件項目）／類型：全文查找

### 上傳步驟
1. 登入 EgentHub 工作室
2. 進入「知識庫管理」
3. 依上述清單上傳對應檔案
4. 設定每個檔案的處理類型（向量／SQL／全文）
5. 在 Agent 設定中綁定本知識庫

### 注意事項
- L0 級機密圖檔（客戶 CAD、製程參數）請走地端 only 路由
- SQL 表格類請先整理為 CSV / XLSX 格式
- 全文文件建議 PDF / DOCX
- 向量檢索適合長文件、案例庫、SOP
