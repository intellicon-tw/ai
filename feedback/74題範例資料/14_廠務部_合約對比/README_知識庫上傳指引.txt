# 合約對比（題號 #14）知識庫範例

## 基本資訊
- 部門：廠務部
- 提案人：廖雅茹
- 架構：A ‧ 文件解析／生成
- 建議知識庫類型：全文查找
- 出席時段：S4 13:00-13:20
- 共通專題：合約 ‧ 套用

## 輸入（客戶提供）
將廠商提供的合約或保養合約等.能夠與去年合約做對比並提出異處

## 預期輸出（客戶提供）
協助提供:
 1. 相異處
 2. 建議如何改善

## 處理邏輯（客戶提供）
匯入廠商提供合約→確認合約內容→標示合約異常或修改處

## 客戶提供的資料來源
1.年度純水系統保養合約
 2.年度空壓系統保養合約
 3.年度冰水系統保養合約

## EgentHub 知識庫上傳指引

### 建議的知識庫檔案
- 翔名標準合約範本.pdf（NDA、Master、PO、Quality Agreement 等標準範本）／類型：全文查找
- 過往簽約案件庫.txt（歷史簽約案例含協商重點與踩雷經驗）／類型：向量檢索
- 法律風險條款庫.pdf（危險條款解釋、判例、業界經驗）／類型：向量檢索
- 客戶通訊錄與簽約主檔.xlsx（客戶代號、過往合約、法務窗口）／類型：SQL 表格

### 上傳步驟
1. 登入 EgentHub 工作室
2. 進入「知識庫管理」
3. 依上述清單上傳對應檔案
4. 設定每個檔案的處理類型（向量／SQL／全文）
5. 在 Agent 設定中綁定本知識庫

### 注意事項
- L0 級機密圖檔（客戶 CAD、製程參數）請走地端 only 路由
- SQL 表格類請先整理為 CSV / XLSX 格式
- 全文文件建議 PDF / DOCX
- 向量檢索適合長文件、案例庫、SOP
