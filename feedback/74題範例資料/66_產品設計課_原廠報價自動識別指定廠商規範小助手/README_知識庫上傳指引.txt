# 原廠報價自動識別指定廠商規範小助手（題號 #66）知識庫範例

## 基本資訊
- 部門：產品設計課
- 提案人：葉于萱
- 架構：A ‧ 文件解析／生成
- 建議知識庫類型：全文查找
- 出席時段：S7 14:20-15:00
- 共通專題：圖面辨識 ‧ 套用

## 輸入（客戶提供）
AI辨識原廠圖SPEC，依SPEC需求製作對應委外供應商報價圖面

## 預期輸出（客戶提供）
匯入圖面>產出指定供應商委外圖

## 處理邏輯（客戶提供）
1.匯入圖面(PDF檔)
 2.抓取圖面所有SPEC
 3.反查各項SPEC是否有指定供應商
 4.依照各個指定供應商，生成指定格式的詢價圖面

## 客戶提供的資料來源
1.原廠規範指定供應商歸類
 2.詢價圖面指定格式與觸發條件

## EgentHub 知識庫上傳指引

### 建議的知識庫檔案
- 原廠報價自動識別指定廠商規範小助手_範本庫.pdf（標準範本與格式定義）／類型：全文查找
- 歷史案例庫.txt（過往處理案例）／類型：向量檢索
- 客戶/業務主檔.xlsx（代號、聯絡資訊、特殊規定）／類型：SQL 表格

### 上傳步驟
1. 登入 EgentHub 工作室
2. 進入「知識庫管理」
3. 依上述清單上傳對應檔案
4. 設定每個檔案的處理類型（向量／SQL／全文）
5. 在 Agent 設定中綁定本知識庫

### 注意事項
- L0 級機密圖檔（客戶 CAD、製程參數）請走地端 only 路由
- SQL 表格類請先整理為 CSV / XLSX 格式
- 全文文件建議 PDF / DOCX
- 向量檢索適合長文件、案例庫、SOP
