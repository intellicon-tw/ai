# 交期回覆幫手（題號 #36）知識庫範例

## 基本資訊
- 部門：業務部
- 提案人：王思敏
- 架構：D ‧ 流程自動化
- 建議知識庫類型：SQL 表格
- 出席時段：XT 11:00-12:00
- 共通專題：手工重複 ‧ 首示範

## 輸入（客戶提供）
客戶自行輸入訂單資料，讓AI自動回覆目前ERP最新的交期。(在公司網頁設置AI小幫手)
 助理僅需維護特定欄位，或是利用公式維護資料。

## 預期輸出（客戶提供）
(1)客戶輸入特定訂單資料後(訂單單號/序號/品號…)，AI至ERP內的特定欄位，帶出該訂單產品的最新交期、未交數量…等資訊，並自動回覆客戶。 (2)為確保安全性，客戶端可能需要有自己專屬的帳號密碼。

## 處理邏輯（客戶提供）
先建立一個專門的小程式，並且提供資料欄位(單號/序號/品號...)給客戶填寫。
 AI到ERP訂單帶出 交期、未交數量的資料，並回覆給客戶。

## 客戶提供的資料來源
ERP訂單資料、開發程式。

## EgentHub 知識庫上傳指引

### 建議的知識庫檔案
- ERP 訂單表 MCP（訂單狀態、預計出貨日查詢）／類型：MCP 工具
- 生產排程系統 MCP（工單進度、瓶頸工序查詢）／類型：MCP 工具
- 客戶主檔.xlsx（客戶代號、語言偏好、付款條件）／類型：SQL 表格
- 客戶郵件範本.pdf（中/英/日三語回信範本）／類型：全文查找

### 上傳步驟
1. 登入 EgentHub 工作室
2. 進入「知識庫管理」
3. 依上述清單上傳對應檔案
4. 設定每個檔案的處理類型（向量／SQL／全文）
5. 在 Agent 設定中綁定本知識庫

### 注意事項
- L0 級機密圖檔（客戶 CAD、製程參數）請走地端 only 路由
- SQL 表格類請先整理為 CSV / XLSX 格式
- 全文文件建議 PDF / DOCX
- 向量檢索適合長文件、案例庫、SOP
