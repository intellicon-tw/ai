# 製造工時核對（題號 #12）知識庫範例

## 基本資訊
- 部門：財務部
- 提案人：江心怡
- 架構：C ‧ 資料分析
- 建議知識庫類型：SQL 表格
- 出席時段：S1 09:00-10:00
- 共通專題：— ‧ 獨立題

## 輸入（客戶提供）
自動檢查人機時不符及報工異常的資料並通知相關人員修改

## 預期輸出（客戶提供）
依照圖面及業務提供資訊評估產品分數

## 處理邏輯（客戶提供）
依指令條件將人機時不符符的資料找出,寄給相關主管回覆正確人機時資料後,透過MIS寫的程式將正確人機時資料更新至ERP系統內

## 客戶提供的資料來源
1.移轉單明細表

## EgentHub 知識庫上傳指引

### 建議的知識庫檔案
- ERP 財務模組.sql（AP/AR、總分類帳、成本明細）／類型：SQL 表格
- 預算表.xlsx（各部門預算 vs 實際）／類型：SQL 表格
- 財務分析 SOP.pdf（毛利計算規則、成本攤提原則）／類型：向量檢索

### 上傳步驟
1. 登入 EgentHub 工作室
2. 進入「知識庫管理」
3. 依上述清單上傳對應檔案
4. 設定每個檔案的處理類型（向量／SQL／全文）
5. 在 Agent 設定中綁定本知識庫

### 注意事項
- L0 級機密圖檔（客戶 CAD、製程參數）請走地端 only 路由
- SQL 表格類請先整理為 CSV / XLSX 格式
- 全文文件建議 PDF / DOCX
- 向量檢索適合長文件、案例庫、SOP
