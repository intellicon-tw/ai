# 泡泡圖及品檢表生成（題號 #64）知識庫範例

## 基本資訊
- 部門：QA
- 提案人：許婉琳
- 架構：A ‧ 文件解析／生成
- 建議知識庫類型：SQL 表格
- 出席時段：S5 13:20-13:45
- 共通專題：圖面辨識 ‧ 套用

## 輸入（客戶提供）
1.自動辨識客戶提供的圖面，抓取尺寸、公差與圖區資訊。
 2.依規則自動產生球標編號，並依左上到右下或順時針邏輯排序。
 3.自動將球標、尺寸、公差等資料整理並帶入指定 Excel 表格。
 4.避免人工框選、漏標與輸入錯誤。
 5.若人工修改泡泡圖/球標編號，AI 可自動連動調整後續編號。

## 預期輸出（客戶提供）
1. 自動完成圖面球標
 2. 自動將球標序號、圖區、尺寸、公差填入 Excel
 3. 自動依小數點位數帶入對應公差
 4. 支援人工微調與序號自動重排
 5. 後續可進行 AI 圖面與報告審查

## 處理邏輯（客戶提供）
1. 匯入 PDF / CAD 圖面
 2. AI 自動辨識所有尺寸與視角區塊
 3. 依規則自動產生球標並排序
 4. 擷取尺寸、公差與圖區資訊
 5. 若圖面未標示公差，依小數點位數自動比對公差表
 6. 自動寫入 Excel 對應欄位
 7. 支援人工修改與球標連動更新
 8. 最後進行漏標與規格檢查

## 客戶提供的資料來源
1. 客戶工程圖（PDF / CAD）
 2. 圖面球標 Excel 範本
 3. 公差規格表
 4. 圖區對照規則
 5. 已完成的歷史球標資料

## EgentHub 知識庫上傳指引

### 建議的知識庫檔案
- 翔名圖面範本庫.pdf（標準圖面格式、註記、規範）／類型：全文查找
- 客戶 SPEC 規範庫.pdf（SEMI、ASME、ISO 規範摘要）／類型：向量檢索
- 歷史加工注意事項.txt（資深前輩經驗、特殊製程注意）／類型：向量檢索
- 料號主檔.xlsx（翔名料號、客戶料號對照）／類型：SQL 表格

### 上傳步驟
