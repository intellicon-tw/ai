# 原廠圖NOTES助手（題號 #65）知識庫範例

## 基本資訊
- 部門：機構設計課
- 提案人：葉于萱
- 架構：A ‧ 文件解析／生成
- 建議知識庫類型：全文+向量 混合
- 出席時段：S7 14:20-15:00
- 共通專題：圖面辨識 ‧ 首示範

## 輸入（客戶提供）
AI提供註記英翻中並提供注意事項

## 預期輸出（客戶提供）
匯入圖面>產生中文版註記&注意事項

## 處理邏輯（客戶提供）
1.匯入圖面(PDF檔)
 2.辨識原廠註記&註記中的SPEC
 3.產生中文版本註記&注意事項
 (格式依照原廠圖NOTES排序即可)

## 客戶提供的資料來源
1.常用註記翻譯
 2.SPEC注意事項整理

## EgentHub 知識庫上傳指引

### 建議的知識庫檔案
- 翔名圖面範本庫.pdf（標準圖面格式、註記、規範）／類型：全文查找
- 客戶 SPEC 規範庫.pdf（SEMI、ASME、ISO 規範摘要）／類型：向量檢索
- 歷史加工注意事項.txt（資深前輩經驗、特殊製程注意）／類型：向量檢索
- 料號主檔.xlsx（翔名料號、客戶料號對照）／類型：SQL 表格

### 上傳步驟
1. 登入 EgentHub 工作室
2. 進入「知識庫管理」
3. 依上述清單上傳對應檔案
4. 設定每個檔案的處理類型（向量／SQL／全文）
5. 在 Agent 設定中綁定本知識庫

### 注意事項
- L0 級機密圖檔（客戶 CAD、製程參數）請走地端 only 路由
- SQL 表格類請先整理為 CSV / XLSX 格式
- 全文文件建議 PDF / DOCX
- 向量檢索適合長文件、案例庫、SOP
