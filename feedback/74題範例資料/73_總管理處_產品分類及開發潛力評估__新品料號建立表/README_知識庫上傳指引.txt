# 產品分類及開發潛力評估  新品料號建立表（題號 #73）知識庫範例

## 基本資訊
- 部門：總管理處
- 提案人：吳予彤
- 架構：C ‧ 資料分析
- 建議知識庫類型：SQL+向量 混合
- 出席時段：S3 10:30-11:00
- 共通專題：商機研究 ‧ 套用

## 輸入（客戶提供）
評估此新產品的開發潛力，作為開發及生產優先順序評估用

## 預期輸出（客戶提供）
依照圖面及業務提供資訊評估產品分數，並提供建議，如產品價格或數量提升多少則可以接案，或是雖然非主流產品，但是為新的產品類別有戰術價值。或是目前產品資訊不足，尚缺那些資訊，以及資訊哪裡可以取得

## 處理邏輯（客戶提供）
依與客戶討論後提供的年用量(市場規模)及報價來評估產品的開發效益
 將產品以分數標示以利後段開發時的排序
 未達平均值時，請與主管討論是否開發
 單看項目在單一客戶年用量小時，可收集產品整體市場規模，如能在客戶群中如能創造量的效益時可以考慮開發(如年保產品)
 策略價值可以再加入(但必須考慮公司的量能及現在與未來的技術規劃)
