## EgentHub 知識庫上傳指引

### 建議的知識庫檔案
- 報關文件範本.pdf（進/出口報單、Invoice、Packing List 範本）／類型：全文查找
- 海關品名分類碼表.xlsx（HS Code 與翔名產品對照）／類型：SQL 表格
- 客戶通訊錄.xlsx（客戶國別、報關資訊、特殊規定）／類型：SQL 表格
- 出口管制法規.pdf（ITAR、EAR 等出口管制要點）／類型：向量檢索

### 上傳步驟
1. 登入 EgentHub 工作室
2. 進入「知識庫管理」
3. 依上述清單上傳對應檔案
4. 設定每個檔案的處理類型（向量／SQL／全文）
5. 在 Agent 設定中綁定本知識庫

### 注意事項
- L0 級機密圖檔（客戶 CAD、製程參數）請走地端 only 路由
- SQL 表格類請先整理為 CSV / XLSX 格式
- 全文文件建議 PDF / DOCX
- 向量檢索適合長文件、案例庫、SOP
