- 建議知識庫類型：SQL 表格

## 輸入（客戶提供）
廠商接單後，產品之排程、回貨時間及數量

## 預期輸出（客戶提供）
廠商回貨日期及數量提醒，陸續回貨，累加至需求數量以及預交日比對

## 處理邏輯（客戶提供）
廠商載(到)料時間至預交日排程比對；實際回貨日與預交日計算扣抵

## 客戶提供的資料來源
廠商報價單、訂單、派工單

## EgentHub 知識庫上傳指引

### 建議的知識庫檔案
- 採購系統 MCP（採購單、供應商評等 API）／類型：MCP 工具
- 供應商主檔.xlsx（供應商代號、過往交期、品保）／類型：SQL 表格
- 詢價範本.pdf（詢價單格式、條件項目）／類型：全文查找

### 上傳步驟
1. 登入 EgentHub 工作室
2. 進入「知識庫管理」
3. 依上述清單上傳對應檔案
4. 設定每個檔案的處理類型（向量／SQL／全文）
5. 在 Agent 設定中綁定本知識庫

### 注意事項
- L0 級機密圖檔（客戶 CAD、製程參數）請走地端 only 路由
- SQL 表格類請先整理為 CSV / XLSX 格式
- 全文文件建議 PDF / DOCX
- 向量檢索適合長文件、案例庫、SOP
