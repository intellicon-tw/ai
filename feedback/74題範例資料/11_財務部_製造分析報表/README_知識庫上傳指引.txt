# 製造分析報表（題號 #11）知識庫範例

## 基本資訊
- 部門：財務部
- 提案人：江心怡
- 架構：C ‧ 資料分析
- 建議知識庫類型：SQL 表格
- 出席時段：S1 09:00-10:00
- 共通專題：— ‧ 獨立題

## 輸入（客戶提供）
因為要大量下載相關報表來彙總編製製造分析的報表,因為都是每月重複性的工作,所以希望能程式化以減少人為判斷的錯誤及提昇時效性.

## 預期輸出（客戶提供）
一鍵生成增加時效性與人為判斷或公式錯誤的發生

## 處理邏輯（客戶提供）
將右側相關報表彙整在一起完成製造分析的報表,每個新增欄位都要依相關分類來賦予其資料,並依結果完成各項報表呈相關主管

## 客戶提供的資料來源
1.毛利分析資料
 2.產品入庫成本明細表
 3.明細分類帳
 4.相關欄位的賦予

## EgentHub 知識庫上傳指引

### 建議的知識庫檔案
- ERP 財務模組.sql（AP/AR、總分類帳、成本明細）／類型：SQL 表格
- 預算表.xlsx（各部門預算 vs 實際）／類型：SQL 表格
- 財務分析 SOP.pdf（毛利計算規則、成本攤提原則）／類型：向量檢索

### 上傳步驟
1. 登入 EgentHub 工作室
2. 進入「知識庫管理」
3. 依上述清單上傳對應檔案
4. 設定每個檔案的處理類型（向量／SQL／全文）
5. 在 Agent 設定中綁定本知識庫

### 注意事項
- L0 級機密圖檔（客戶 CAD、製程參數）請走地端 only 路由
- SQL 表格類請先整理為 CSV / XLSX 格式
- 全文文件建議 PDF / DOCX
- 向量檢索適合長文件、案例庫、SOP
