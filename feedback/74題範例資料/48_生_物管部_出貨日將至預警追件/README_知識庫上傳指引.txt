# 出貨日將至預警追件（題號 #48）知識庫範例

## 基本資訊
- 部門：生/物管部
- 提案人：鄭佳儀
- 架構：D ‧ 流程自動化
- 建議知識庫類型：SQL 表格
- 出席時段：XT 11:00-12:00
- 共通專題：手工重複 ‧ 套用

## 輸入（客戶提供）
依照我們提供的客戶簡稱、指定日期區間(例如:2026/6/1~2026/6/15)，撈出訂單報表內的出貨日符合日期區間內，尚未入庫存的產品預警(需扣除在庫庫存)，並產出一份其產品由工序最大遞減排序的工單，包含其工單是否有尚未領用的材料品號(需確認其未領用的材料品號是否有庫存)的報表

## 預期輸出（客戶提供）
依照產出的報表，快速得知該客戶目前交期將至但尚未完工的產品列表

## 處理邏輯（客戶提供）
1.開啟「訂單報表」
 2.開啟篩選→填入「客戶簡稱」
 3.逐筆確認個產品的出貨日，使否庫存已滿足此出貨日的產品訂單數量
 4.若還有在線上加工的，查詢下一支「WIP」報表
 5.開啟篩選→填入品號→列出需請製造安排加工的製令排序
 6.查詢該製令是否有未領用的材料(鼎新ERP→製令/託外管理系統→*製造命令建立作業MCP)
 7.如有尚未到廠的材料/半成品/採購件，請物管確認到料時間/請製造加工半成品/請採購確認到料時間

## 客戶提供的資料來源
1.訂單報表
 2.WIP報表
 3.鼎新ERP→製令/託外管理系統→*製造命令建立作業MCP
 4.鼎新ERP系統→庫存管理系統→*庫存狀況查詢作業

## EgentHub 知識庫上傳指引

### 建議的知識庫檔案
- ERP 訂單表 MCP（訂單狀態、預計出貨日查詢）／類型：MCP 工具
- 生產排程系統 MCP（工單進度、瓶頸工序查詢）／類型：MCP 工具
- 客戶主檔.xlsx（客戶代號、語言偏好、付款條件）／類型：SQL 表格
- 客戶郵件範本.pdf（中/英/日三語回信範本）／類型：全文查找

### 上傳步驟
1. 登入 EgentHub 工作室
2. 進入「知識庫管理」
3. 依上述清單上傳對應檔案
4. 設定每個檔案的處理類型（向量／SQL／全文）
5. 在 Agent 設定中綁定本知識庫

### 注意事項
- L0 級機密圖檔（客戶 CAD、製程參數）請走地端 only 路由
- SQL 表格類請先整理為 CSV / XLSX 格式
- 全文文件建議 PDF / DOCX
- 向量檢索適合長文件、案例庫、SOP
