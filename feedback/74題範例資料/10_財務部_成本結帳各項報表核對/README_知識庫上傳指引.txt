## EgentHub 知識庫上傳指引

- ERP 財務模組.sql（AP/AR、總分類帳、成本明細）／類型：SQL 表格

### 上傳步驟
1. 登入 EgentHub 工作室
2. 進入「知識庫管理」
3. 依上述清單上傳對應檔案
4. 設定每個檔案的處理類型（向量／SQL／全文）
5. 在 Agent 設定中綁定本知識庫

### 注意事項
- L0 級機密圖檔（客戶 CAD、製程參數）請走地端 only 路由
- SQL 表格類請先整理為 CSV / XLSX 格式
- 全文文件建議 PDF / DOCX
- 向量檢索適合長文件、案例庫、SOP
