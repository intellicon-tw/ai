# 領刀流程（題號 #74）知識庫範例

## 基本資訊
- 部門：製造

- 架構：D ‧ 流程自動化
- 建議知識庫類型：SQL 表格

## 輸入（客戶提供）
建立線上領刀系統
進化領刀流程
節省領刀時間
提升換線效率
改善機台稼動率

## 預期輸出（客戶提供）
1.自動帶出NC程式刀單
2.自動比對ERP庫存狀態
3.自動通知刀具室發刀
4.缺刀或特殊刀具自動向刀具室提出請購需求
5.完成配刀自動扣除庫存

## 處理邏輯（客戶提供）
1.加工者拿到工單後事先進行線上領刀作業
2. A I 進行庫存比對，若庫存不足或特殊刀具需求則直接向刀具室發出需求警示
3.刀具室將依據需求及缺刀部分進行發刀及請購作業
4.領用完成核准後自動ERP扣料，節省人員一筆一筆Keyin工時。

## EgentHub 知識庫上傳指引

### 建議的知識庫檔案
- 刀具庫存系統 MCP（庫存查詢、領用記錄 API）／類型：MCP 工具
- 刀具主檔.xlsx（型號、規格、適用機台）／類型：SQL 表格

### 上傳步驟
1. 登入 EgentHub 工作室
2. 進入「知識庫管理」
3. 依上述清單上傳對應檔案
4. 設定每個檔案的處理類型（向量／SQL／全文）
5. 在 Agent 設定中綁定本知識庫

### 注意事項
- L0 級機密圖檔（客戶 CAD、製程參數）請走地端 only 路由
- SQL 表格類請先整理為 CSV / XLSX 格式
- 全文文件建議 PDF / DOCX
- 向量檢索適合長文件、案例庫、SOP
