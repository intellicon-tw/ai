# 鎖定在製期末約量（題號 #9）知識庫範例

## 基本資訊
- 部門：財務部
- 提案人：江心怡
- 架構：D ‧ 流程自動化
- 建議知識庫類型：SQL 表格
- 出席時段：S1 09:00-10:00
- 共通專題：— ‧ 獨立題

## 輸入（客戶提供）
將人員逐筆檢查改為系統直接判斷及修改期末在製約量的數字

## 預期輸出（客戶提供）
一鍵生成自動將正確的期末約量數字更新至ERP內.

## 處理邏輯（客戶提供）
針對本月有入庫的在製工單檢查其期末在製約量異常的工單(期末約量>已領餘量者)並鎖定其期末在製約量,
 因為要逐筆檢查故需花費大量時間

## 客戶提供的資料來源
1.製令
 2.製令成本建立作業
 3.生產進度表

### 上傳步驟
1. 登入 EgentHub 工作室
2. 進入「知識庫管理」
3. 依上述清單上傳對應檔案

5. 在 Agent 設定中綁定本知識庫

- SQL 表格類請先整理為 CSV / XLSX 格式
- 全文文件建議 PDF / DOCX
- 向量檢索適合長文件、案例庫、SOP
