# ESG 碳盤查與永續資料蒐集整合助手（題號 #20）知識庫範例

## 基本資訊
- 部門：廠務部
- 提案人：詹錢翔
- 架構：A ‧ 文件解析／生成
- 建議知識庫類型：全文+SQL 混合
- 出席時段：S4 13:00-13:20
- 共通專題：— ‧ 獨立題

## 輸入（客戶提供）
1.廠區內分散於各處的能源數據（台電電費單PDF、公務車油耗 Excel、水費單PDF等） 。
 2.外部供應商格式不一的碳盤查資料、產品碳足跡（PCF）宣告與清運廠商的廢棄物聯單 。
 3.人工輸入單位的錯誤攔截（例如：將公升誤填為加侖，或抄表數字多打一個零），導致碳排計算暴增或暴跌。

## 預期輸出（客戶提供）
1.符合國際標準（ISO 14064-1 / 14067）的企業碳盤查 Excel 整合總表 。
 2.管理階層 ESG 營運決策戰情週報（Mail 草稿與數據摘要） 。
 3.企業永續報告書（ESG Report）合規文本環境績效章節草稿 。

## 處理邏輯（客戶提供）
1.自主數據擷取： AI Agent 定期自動讀取指定的 Excel、Google Sheets，並利用文字識別（OCR）功能讀取跨廠區的水電瓦斯帳單 PDF 與外部廠商回傳的碳排表單 。
 2.異常值攔截： 自動比對該廠區的歷史能耗基期，若發現某月份數據異常飆高，自動標記為潛在異常，並主動草發信給負責窗口進行二次確認。
 3.動態係數匹配： 自動連接並檢索政府主管機關或國際最新公布的碳排係數資料庫，將異質數據精準轉換為二氧化碳當量（CO2e）。
 4.主動式催收： 比對應繳清單，若發現特定供應商漏填範疇三數據，自動撰寫並發送客製化的催繳與確認郵件 。

## 客戶提供的資料來源
1.歷年台電電費單 PDF 範本
 2.公務車油耗統計 Excel 表
 3.環境部碳排放係數清冊網址
 4.供應商碳足跡回報單

- ESG 報告範本（GRI / SASB）.pdf（國際標準框架與翔名 ESG 模板）／類型：全文查找

3. 依上述清單上傳對應檔案

5. 在 Agent 設定中綁定本知識庫
