# Agent 04 — 電商文案 SEO 助手（行銷部 / 王鈺婷）

## 一、可行性
**可做，是 LLM 強項。** 唯一需要的是「品牌語氣指南」「平台特性指南」「過去成功文案範例」這三個知識庫，能讓文案產出符合「印表大王」的調性，而不是泛泛的 LLM 風格。

## 二、模型建議
- 預設：`claude-sonnet-4-6`（中文文案表達自然、結構穩、SEO 關鍵字嵌入流暢）

## 三、知識庫設定

| 知識庫名稱 | 類型 | 狀態 |
|-----------|------|------|
| 印表機機器規格表 | 表格查詢 | ✅ 已備齊（與 Agent 03 共用） |
| 印表機耗材規格表 | 表格查詢 | ✅ 已備齊（與 Agent 03 共用） |
| 品牌語氣指南 | 全文查找 | **待王鈺婷補**（已附範本供修改） |
| 平台特性指南 | 全文查找 | **已附初稿**（可立即使用，建議王鈺婷再校對） |
| 過去成功文案範例 | 全文查找 | **待王鈺婷補**（建議貼 5–10 篇高互動貼文） |

## 四、兩階段執行機制

當使用者上傳 **PDF 原廠規格書** 時，Prompt 已設計為「步驟一讀 PDF / 步驟二查知識庫 + 撰寫文案」，避免 PDF 視覺理解與表格查詢的 context 混淆。
若使用者直接提供「文字版產品規格」，則可一步完成。

## 五、待釐清事項
1. 「短影音文案」原始描述較模糊 — 確認指的是「腳本／口播稿」（Prompt 採用此假設），還是「貼文文案」？
2. YouTube 短影音／TikTok 是否要產出「字幕配文」？目前 Prompt 含「分鏡腳本」與「口播」，沒含逐字字幕。
3. LINE@ 是否有規定的 CTA 連結格式？

## 六、驗測情境建議
1. 上傳 EPSON M310 規格書 PDF + 「6 月畢業季活動，重點打學生族群」 → 預期：兩步驟完成，產出 7 平台文案。
2. 直接給文字「BROTHER HL-L3280CDW 彩色雷射，活動：618 折 $50 副廠碳粉」 → 預期：跳過步驟一，直接寫文案。
3. 只指定 IG + Threads → 預期：只產出這兩個平台，不多寫。
