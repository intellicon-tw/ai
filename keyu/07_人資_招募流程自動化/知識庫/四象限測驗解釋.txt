四象限風格測驗解釋（依上傳測驗圖建構，請黃靖涵校對）

═══════════════════════════════════════
測驗結構
═══════════════════════════════════════
本測驗測量兩個維度：
- X 軸：步調 - 慢步調 A ←→ 快步調 B
- Y 軸：導向 - 任務導向 C ←→ 人際導向 D

座標計算：
- X = B - A（正值偏快步調、負值偏慢步調）
- Y = C - D（正值偏任務導向、負值偏人際導向）

═══════════════════════════════════════
四象限與類型
═══════════════════════════════════════

【第二象限｜分析型】（慢步調 + 任務導向）
- 核心優勢：邏輯嚴謹、注重細節、品質要求高、長期規劃能力強
- 行為風格：先思考再行動，避免錯誤；偏好獨立工作
- 可能盲點：決策速度慢、過度完美主義、難以彈性調整、不太擅長社交場合
- 適合職務：研發、稽核、資料分析、品保、財務、技術文件撰寫

【第一象限｜支配型】（快步調 + 任務導向）
- 核心優勢：決策果斷、目標導向、執行力強、敢於挑戰
- 行為風格：直接、效率優先、不愛繞圈
- 可能盲點：可能忽略他人感受、太強勢、容易給團隊壓力
- 適合職務：業務主管、專案經理、營運主管、新事業 BD

【第三象限｜說話型】（慢步調 + 人際導向）
- 核心優勢：關懷他人、值得信任、團隊穩定器、傾聽能力強
- 行為風格：溫和、不急、重視關係與和諧
- 可能盲點：避免衝突、決策依賴他人、面對壓力時退縮
- 適合職務：客服、HR、行政、客戶關係維護、教育訓練

【第四象限｜表達型】（快步調 + 人際導向）
- 核心優勢：熱情、感染力強、創意豐富、樂於分享
- 行為風格：活潑、樂觀、喜歡互動與被關注
- 可能盲點：細節掌握度低、虎頭蛇尾、太愛說話、組織度不足
- 適合職務：行銷企劃、活動公關、業務開發、社群經營、講師

═══════════════════════════════════════
象限邊界情況
═══════════════════════════════════════
- 若 |X| < 3 或 |Y| < 3：座標靠近中央，視為混合型，需綜合判斷。
- 兩個分數很接近的特質（例如 C=22、D=21）：在面試時可雙向觀察。

═══════════════════════════════════════
給招募同仁的使用建議
═══════════════════════════════════════
- 此測驗只是初步參考，不能單獨決定錄取與否。
- 同一類型不同人差異仍大，面試時還要評估專業能力、經驗、文化契合度。
- 跨象限組合（如業務+分析）的人才相對稀有，可額外關注。
