# 工作室4_品保

本工作室包含 7 個 Agent，共 2 個使用表格知識庫，5 個使用文件知識庫。

## Agent 清單

| 編號 | Agent 名稱 | 知識庫類型 | POC 方式 | 兩階段 |
|---|---|---|---|---|
| 4.1 | 可靠度測試數據圖表助手 | 表格查詢知識庫（測試數據） | csv 資料表 | 否 |
| 4.2 | 8D客訴改善報告助手 | 全文查找（8D範本/歷史報告） | RAG 文件 | 否 |
| 4.3 | 進貨檢驗規範查詢助手 | 全文查找（IQC規範） | RAG 文件 | 否 |
| 4.4 | 檢驗報告生成助手 | 全文查找（報告範本） | RAG 文件 | 否 |
| 4.5 | PFMEA初稿助手 | 全文查找（PFMEA/歷史） | RAG 文件 | 否 |
| 4.6 | 量檢具校驗管理助手 | 表格查詢知識庫（校驗台帳） | csv 資料表 | 否 |
| 4.7 | 客戶稽核比對應答助手 | 全文查找（品質文件/認證） | RAG 文件 | 否 |
