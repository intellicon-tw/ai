# 工作室5_採購資材

本工作室包含 5 個 Agent，共 3 個使用表格知識庫，2 個使用文件知識庫。

## Agent 清單

| 編號 | Agent 名稱 | 知識庫類型 | POC 方式 | 兩階段 |
|---|---|---|---|---|
| 5.1 | 供應商評鑑彙整助手 | 表格查詢知識庫（供應商紀錄） | csv 資料表 | 否 |
| 5.2 | 採購合約審查助手 | 全文查找（標準範本/風險清單） | RAG 文件 | 否 |
| 5.3 | 料件規格查詢助手 | 表格查詢知識庫（料件主檔） | csv 資料表 | 否 |
| 5.4 | 物料環保證明檢核助手 | 全文查找（應備文件清單） | RAG 文件 | 否 |
| 5.5 | 庫存呆滯料分析助手 | 表格查詢知識庫（庫存紀錄） | csv 資料表 | 否 |

## 各 Agent 子目錄

- `5.1_供應商評鑑彙整助手/` — 1_prompt + 2_知識庫範例資料 + 3_驗測情境 + 4_正式上線前的準備指南
- `5.2_採購合約審查助手/` — 1_prompt + 2_知識庫範例資料 + 3_驗測情境 + 4_正式上線前的準備指南
- `5.3_料件規格查詢助手/` — 1_prompt + 2_知識庫範例資料 + 3_驗測情境 + 4_正式上線前的準備指南
- `5.4_物料環保證明檢核助手/` — 1_prompt + 2_知識庫範例資料 + 3_驗測情境 + 4_正式上線前的準備指南
- `5.5_庫存呆滯料分析助手/` — 1_prompt + 2_知識庫範例資料 + 3_驗測情境 + 4_正式上線前的準備指南
