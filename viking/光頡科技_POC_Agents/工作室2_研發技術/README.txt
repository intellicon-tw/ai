# 工作室2_研發技術

本工作室包含 8 個 Agent，共 2 個使用表格知識庫，5 個使用文件知識庫。

## Agent 清單

| 編號 | Agent 名稱 | 知識庫類型 | POC 方式 | 兩階段 |
|---|---|---|---|---|
| 2.1 | 技術文件檢索助手 | 全文查找（內部技術文件）+ Agent 向量（外部論文/專利） | RAG 文件 | 否 |
| 2.2 | 技術文件翻譯助手 | 表格查詢知識庫（術語對照表 glossary） | csv 資料表 | 否 |
| 2.3 | 可行性判斷報告初稿助手 | 全文查找（技術文件庫） | RAG 文件 | 否 |
| 2.4 | 技術方案情境比較助手 | 全文查找（技術文件） | RAG 文件 | 否 |
| 2.5 | 圖面與Datasheet規格擷取助手 | - | 純提示工程 | 條件式（上傳檔案時啟用） |
| 2.6 | 電性參數工程計算助手 | 表格查詢知識庫（料號電性規格） | csv 資料表 | 否 |
| 2.7 | 專利與論文檢索摘要助手 | Agent 向量知識庫（外部公開資訊） | RAG 文件 | 否 |
| 2.8 | 資深製程專家知識傳承系統 | 全文查找 + Agent 向量（製程經驗庫） | RAG 文件 | 否 |

- `2.1_技術文件檢索助手/` — 1_prompt + 2_知識庫範例資料 + 3_驗測情境 + 4_正式上線前的準備指南

- `2.5_圖面與Datasheet規格擷取助手/` — 1_prompt + 2_知識庫範例資料 + 3_驗測情境 + 4_正式上線前的準備指南
