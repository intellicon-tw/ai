# 工作室1_業務

本工作室包含 6 個 Agent，共 3 個使用表格知識庫，3 個使用文件知識庫。

## Agent 清單

| 編號 | Agent 名稱 | 知識庫類型 | POC 方式 | 兩階段 |
|---|---|---|---|---|
| 1.1 | 客戶詢價規格比對助手 | 表格查詢知識庫（料號規格表）+ 全文查找（客戶規格書） | csv 資料表 | 否 |
| 1.2 | 報價單草擬助手 | 表格查詢知識庫（料號價表/歷史報價） | csv 資料表 | 否 |
| 1.3 | 國際客戶多語回覆助手 | Agent 向量知識庫（回覆樣本/FAQ）+ 全文查找（規格） | RAG 文件 | 否 |
| 1.4 | 客戶滿意度調查分析助手 | 表格查詢知識庫（問卷資料） | csv 資料表 | 否 |
| 1.5 | 業務週報生成助手 | - | 純提示工程 | 否 |
| 1.6 | 市場與競品情報摘要助手 | Agent 向量知識庫（情報來源） | RAG 文件 | 否 |

## 各 Agent 子目錄

- `1.1_客戶詢價規格比對助手/` — 1_prompt + 2_知識庫範例資料 + 3_驗測情境 + 4_正式上線前的準備指南

- `1.4_客戶滿意度調查分析助手/` — 1_prompt + 2_知識庫範例資料 + 3_驗測情境 + 4_正式上線前的準備指南
- `1.5_業務週報生成助手/` — 1_prompt + 2_知識庫範例資料 + 3_驗測情境 + 4_正式上線前的準備指南
- `1.6_市場與競品情報摘要助手/` — 1_prompt + 2_知識庫範例資料 + 3_驗測情境 + 4_正式上線前的準備指南
