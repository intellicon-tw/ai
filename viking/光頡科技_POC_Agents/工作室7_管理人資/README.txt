# 工作室7_管理人資

本工作室包含 5 個 Agent，共 1 個使用表格知識庫，3 個使用文件知識庫。

## Agent 清單

| 編號 | Agent 名稱 | 知識庫類型 | POC 方式 | 兩階段 |
|---|---|---|---|---|
| 7.1 | 會議紀錄整理助手 | - | 純提示工程 | 否 |
| 7.2 | 法規遵循文件查詢助手 | 全文查找（內控/法規） | RAG 文件 | 否 |
| 7.3 | ESG資料彙整助手 | 表格查詢知識庫（各廠數據） | csv 資料表 | 否 |
| 7.4 | 教育訓練教材生成助手 | 全文查找（既有教材/SOP） | RAG 文件 | 否 |
| 7.5 | 人事制度HR問答助手 | 全文查找（員工手冊/規章） | RAG 文件 | 否 |

## 各 Agent 子目錄

- `7.1_會議紀錄整理助手/` — 1_prompt + 2_知識庫範例資料 + 3_驗測情境 + 4_正式上線前的準備指南

- `7.3_ESG資料彙整助手/` — 1_prompt + 2_知識庫範例資料 + 3_驗測情境 + 4_正式上線前的準備指南
- `7.4_教育訓練教材生成助手/` — 1_prompt + 2_知識庫範例資料 + 3_驗測情境 + 4_正式上線前的準備指南
- `7.5_人事制度HR問答助手/` — 1_prompt + 2_知識庫範例資料 + 3_驗測情境 + 4_正式上線前的準備指南
