# 工作室3_工程製造

本工作室包含 6 個 Agent，共 2 個使用表格知識庫，4 個使用文件知識庫。

## Agent 清單

| 編號 | Agent 名稱 | 知識庫類型 | POC 方式 | 兩階段 |
|---|---|---|---|---|
| 3.1 | 工程變更單比對助手 | 全文查找（歷史 ECN，可選） | RAG 文件 | 條件式（上傳檔案時啟用） |
| 3.2 | 工程變更單草擬助手 | 全文查找（ECN範本/歷史紀錄） | RAG 文件 | 否 |
| 3.3 | 製程特徵比對助手 | 全文查找（製程條件/SOP） | RAG 文件 | 否 |
| 3.4 | 製程品質異常分析助手 | 表格查詢知識庫（異常紀錄） | csv 資料表 | 否 |
| 3.5 | 製造工單與SOP生成助手 | 全文查找（工單範本/SOP） | RAG 文件 | 否 |
| 3.6 | 生產排程衝突偵測助手 | 表格查詢知識庫（排程/訂單） | csv 資料表 | 否 |

## 各 Agent 子目錄

- `3.1_工程變更單比對助手/` — 1_prompt + 2_知識庫範例資料 + 3_驗測情境 + 4_正式上線前的準備指南
- `3.2_工程變更單草擬助手/` — 1_prompt + 2_知識庫範例資料 + 3_驗測情境 + 4_正式上線前的準備指南
- `3.3_製程特徵比對助手/` — 1_prompt + 2_知識庫範例資料 + 3_驗測情境 + 4_正式上線前的準備指南
- `3.4_製程品質異常分析助手/` — 1_prompt + 2_知識庫範例資料 + 3_驗測情境 + 4_正式上線前的準備指南
- `3.5_製造工單與SOP生成助手/` — 1_prompt + 2_知識庫範例資料 + 3_驗測情境 + 4_正式上線前的準備指南
- `3.6_生產排程衝突偵測助手/` — 1_prompt + 2_知識庫範例資料 + 3_驗測情境 + 4_正式上線前的準備指南
