# 產品規格查詢助手 — 正式上線指南

本文件協助您把 Prompt Designer 產出的 Agent 從「概念驗證」推向「正式上線」。
建議由種子團隊以一週為單位推進：第一週 KB 準備、第二週 Agent 建置與驗測、第三週試營運、第四週正式上線。

---

## 一、部署前置作業

### 1. 知識庫準備

1. 從 ERP 或 PLM 匯出最新產品規格表。
2. 標準化欄位（料號、品名、類別、規格、認證、應用領域、建議售價、庫存、最小起訂量、交期）。
3. 確認沒有機密欄位（如成本、利潤率）誤上傳。
4. 編碼採 UTF-8 BOM（utf-8-sig），避免中文欄位在 Excel 中亂碼。

### 2. 確認權限與工作室

- 在 EgentHub 後台建立或選擇對應的工作室（例如：人資、品保、業務 BU）。
- 確認您的角色為「工作室管理員」或「開發者」，才能建立 Agent 與上傳 KB。
- 確認工作室點數額度足夠（建議至少預留一個月用量緩衝）。

---

## 二、在 EgentHub 建立 Agent

### Step 1：建立 Agent 並選擇模型
登入 EgentHub 後台 → 選擇工作室 → 「建立 Agent」 →
- Agent 名稱：產品規格查詢助手
- 預設模型：建議 Claude Sonnet 或 Gemini Pro
- 描述：（複製需求盤點檔案中第一段「角色定位」）

### Step 2：貼入 Prompt Designer 產出的提示詞
將 Prompt Designer 產出的提示詞完整複製貼上，注意保留「問答指引」段落不要修改。

### Step 3：上傳知識庫
1. 進入 Agent 編輯頁 → 「表格知識庫」 → 「上傳 CSV」。
2. 上傳產品規格表.csv。
3. 確認 EgentHub 自動辨識的欄位類型正確（文字 / 數字）。
4. 測試查詢「PROD-A-001 規格」確認 Agent 能讀取。

### Step 4：設定使用者與群組可見性
- 開發階段：限定種子團隊試用。
- 試營運階段：擴大到部門 5-10 位種子使用者。
- 正式上線：開放給目標部門全員。

---

## 三、提示詞調校建議

- 若 Agent 經常忽略「庫存與交期」欄位，請在「執行流程」加上：「每次回覆都需主動標註庫存與交期狀態」。
- 若使用者反映回覆過於冗長，請在「輸出格式」加上：「若僅查單一料號，用 7 行內表格呈現」。
- 若希望主動推薦替代品，請在「執行流程」加上：「若使用者查的料號缺貨或不符需求，主動列出 1-2 個替代料號」。

---

## 四、驗測情境（建議至少跑過以下案例）

1. **正向案例 - 單一料號**: 輸入「PROD-A-001 的規格與認證？」
   - 預期：回覆完整規格、認證、價格、庫存表格
2. **正向案例 - 屬性查詢**: 輸入「我要找通過 AEC-Q 認證的產品」
   - 預期：列出 PROD-A-001、PROD-A-003、PROD-B-005、PROD-C-003、PROD-E-001、PROD-E-003、PROD-F-003
3. **邊界案例 - 缺貨替代**: 輸入「PROD-A-005 怎麼買不到？」
   - 預期：說明缺貨並推薦 PROD-A-001 或 PROD-A-002 等替代品
4. **邊界案例 - 料號不存在**: 輸入「PROD-Z-999 規格？」
   - 預期：誠實告知無此料號，建議聯繫產品 PM

---

## 五、上線檢核清單

- [ ] CSV 內容已由產品 PM 確認最新版本
- [ ] 已剃除機密欄位（成本、利潤率）
- [ ] 報價標註為「建議售價區間」並提醒實際以業務報價為準
- [ ] 提示詞中已加上「不得虛構料號」的保護機制
- [ ] 業務、FAE 種子使用者試用 1 週

---

## 六、上線後維運建議

- 建立每月 CSV 同步機制（建議由 PM 助理每月初匯出更新）。
- 庫存與交期欄位變動頻繁，可考慮後續透過 MCP 串接 ERP 即時查詢。
- 蒐集業務反饋的「沒問到」案例，逐步補充缺漏欄位。
- 進階版：與「RFQ 應答草稿助手」整合，達到端到端自動化。

---

## 七、常見問題

**Q1: Agent 回答不準確？**
先檢查 KB 內容是否完整、條文是否清晰；其次調整提示詞中的「執行流程」與「輸出格式」。

**Q2: 同事抱怨 Agent 太囉嗦？**
在提示詞「輸出格式」中明確要求字數限制或結構（例如：條列三點以內、不超過 200 字）。

**Q3: 找不到對應內容時應如何處理？**
在提示詞中加上：「若知識庫中無法找到答案，請明確回覆『此問題未在現有資料中，建議您聯繫 [窗口] 進一步確認』，切勿自行推測。」
