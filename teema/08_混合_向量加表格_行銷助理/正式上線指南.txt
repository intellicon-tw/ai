# 智造行銷助理 — 正式上線指南

本文件協助您把 Prompt Designer 產出的 Agent 從「概念驗證」推向「正式上線」。
建議由種子團隊以一週為單位推進：第一週 KB 準備、第二週 Agent 建置與驗測、第三週試營運、第四週正式上線。

---

## 一、部署前置作業

### 1. 知識庫準備

1. **向量 KB(市場報告)**:整理近期產業市場研究報告(可從 Yole、TrendForce、IEK 等公開報告抽取摘要)。
2. **表格 KB(客戶名單)**:從 CRM 匯出,**客戶名稱務必脫敏為代碼**,並標註客群類別(A/B/C/D)、產業、規模、地區。
3. **向量 KB(品牌指南)**:整理公司過往轉換率高的文案範例 + 品牌風格指南。
4. **Few-shot 範例**:在 Prompt 中內嵌 2-3 個完美範例(EDM、社群貼文等),讓 AI 模仿風格。
5. 編碼 UTF-8 BOM。

### 2. 確認權限與工作室

- 在 EgentHub 後台建立或選擇對應的工作室（例如：人資、品保、業務 BU）。
- 確認您的角色為「工作室管理員」或「開發者」，才能建立 Agent 與上傳 KB。
- 確認工作室點數額度足夠（建議至少預留一個月用量緩衝）。

---

## 二、在 EgentHub 建立 Agent

### Step 1：建立 Agent 並選擇模型
登入 EgentHub 後台 → 選擇工作室 → 「建立 Agent」 →
- Agent 名稱：智造行銷助理
- 預設模型：建議 Claude Sonnet 或 Gemini Pro
- 描述：（複製需求盤點檔案中第一段「角色定位」）

### Step 2：貼入 Prompt Designer 產出的提示詞
將 Prompt Designer 產出的提示詞完整複製貼上，注意保留「問答指引」段落不要修改。

### Step 3：上傳知識庫
1. 進入 Agent 編輯頁。
2. 上傳市場報告:「向量知識庫」 → 上傳 市場研究報告.txt。
3. 上傳客戶名單:「表格查詢知識庫」 → 上傳 客戶名單.csv。
4. 上傳品牌指南(若有):「向量知識庫」 → 上傳 品牌風格指南.txt。
5. **在 Prompt 中放入 Few-shot 範例**:於「執行流程」結尾加上 1-2 個完整範例,讓 AI 學習。
6. 整合測試:輸入「為 PROD-A-001 寫 EDM,目標 OEM 大廠」,確認多 KB 整合與語氣匹配。

### Step 4：設定使用者與群組可見性
- 開發階段：限定種子團隊試用。
- 試營運階段：擴大到部門 5-10 位種子使用者。
- 正式上線：開放給目標部門全員。

---

## 三、提示詞調校建議

- 若文案語氣不分客群,請在「角色定位」加上四大客群語氣對照表(A 專業嚴謹/B 務實直接/C 親切白話/D 專業國際化)。
- 若 Agent 自行虛構規格,請在「限制條件」加上:「所有產品規格須引用知識庫,不得自行虛構」。
- 若 Agent 嘗試報價,請在「任務邊界」明確列出「不可處理:報價、議價」並引導至業務窗口。
- 若文案缺乏結構(EDM 沒有 CTA、社群貼文沒有 hashtag),請在「輸出格式」明確規範各任務的結構。
- **Few-shot 是本題重點**:在 Prompt 中放入 2-3 個完整範例,讓 AI 模仿風格。

---

## 四、驗測情境（建議至少跑過以下案例）

1. **情境 1 - 新品 EDM**: 輸入「為 PROD-A-001 寫一封 EDM,目標客群 A(OEM 大廠),行銷目的:新品發表」
   - 預期：5 段 EDM 結構,語氣專業嚴謹,引用認證(UL/AEC-Q),含 CTA
2. **情境 2 - 客群拓展分析**: 輸入「PROD-D-001 想拓展中小製造商,做客群分析」
   - 預期：從客戶名單篩選 C 類,列出配適度與渠道建議
3. **情境 3 - 展會邀約**: 輸入「2026 COMPUTEX 展會,主打 LED 系列,寫攤位邀約文」
   - 預期：中英標語、三產品亮點、EDM + LinkedIn 雙版本
4. **情境 4 - LinkedIn 貼文**: 輸入「為 PROD-E-001 溫度感測器寫 LinkedIn 技術分享」
   - 預期：200-300 字,語氣專業但不生硬,有實例,有 Hashtags
5. **情境 5 - 邊界測試(報價拒答)**: 輸入「我們可以給客戶 50% 折扣嗎?草擬報價單」
   - 預期：明確拒答,語氣友善,引導至業務窗口

## 五、上線檢核清單

- [ ] 市場報告已篩選最新版本,移除過時資訊
- [ ] 客戶名單已脫敏,客群類別正確標示
- [ ] 品牌指南已由行銷主管核可
- [ ] Few-shot 範例已內嵌於 Prompt 並測試
- [ ] 提示詞已加上「不可虛構規格、不可報價」保護
- [ ] 提示詞已要求依客群代碼匹配語氣
- [ ] 至少 3 位行銷 / 業務種子使用者試用

## 六、上線後維運建議

- 本題目是『AI 替代重複性內容、人力做策略決策』的最佳示範。
- 蒐集使用者實際送出的最終版本,持續優化 Few-shot 範例。
- 進階版可結合廣告投放系統(Meta、Google),自動推薦目標客群。
- 與『B2B 智慧客服』整合,行銷文案 → 客戶詢問 → 業務跟進形成閉環。
- 建議行銷部門每月分享『AI 寫的最佳案例』,推廣使用文化。

## 七、常見問題

**Q1: Agent 回答不準確？**
先檢查 KB 內容是否完整、條文是否清晰；其次調整提示詞中的「執行流程」與「輸出格式」。

**Q2: 同事抱怨 Agent 太囉嗦？**
在提示詞「輸出格式」中明確要求字數限制或結構（例如：條列三點以內、不超過 200 字）。

**Q3: 找不到對應內容時應如何處理？**
在提示詞中加上：「若知識庫中無法找到答案，請明確回覆『此問題未在現有資料中，建議您聯繫 [窗口] 進一步確認』，切勿自行推測。」
