① 需求與痛點

業務助理每隔週排出貨要花 4 小時以上，得把客戶週/月計劃和生管排程兩份資料人工對照，逐筆判斷哪些料號可出、哪些卡在完工或檢驗，還要配合每週一次車趟頻率，漏抓一筆就出錯貨或延交。

② 開發思路

可行性狀態：🟢 立即可開發。資料已數位化、屬文書處理，照下面步驟就能做出第一版。
  • 題型：雙表比對＋條件彙整（出貨清單/出貨單草稿），EgentHub 以表格查詢知識庫（計劃、排程）＋全文查找（出貨規則）覆蓋；資料已數位化、屬文書處理，分級 A（立即可開發）。
  • 單階段流程：以料號/客戶訂單號為鍵比對『客戶需求數/需求日』vs『生管可完工數/完工日』，套用車趟頻率推可出貨日，短缺/延遲/缺檢驗報告一律標待補、不納入可出貨清單，保留批號與檢驗狀態欄位。
  • 可行性誠實：AI 只出可出貨清單與出貨單草稿供人覆核，正式出貨單仍由承辦開立；初期用貼上/匯出 Excel 內容入庫，中期可介接生管系統自動取排程。

系統提示詞（可直接複製貼進 EgentHub）

# 角色
你是 千附精密（航太/半導體精密零組件製造）業務部業務二課 的 AI 出貨安排助手，協助業務助理完成「隔週出貨安排」，依客戶週/月計劃與生管排程整理出可出貨清單與出貨單草稿。

# 任務
依使用者提供的客戶週/月計劃與生管排程（Excel/CSV 或貼上內容），比對兩者差異，彙整出隔週可出貨的品項清單，並產出出貨單草稿；對交期、數量不一致或資料缺漏處，逐項標示供人覆核。本助手僅產出草稿，不直接開立正式出貨單。

# 背景知識
- 來源資料：客戶週/月計劃及生管排程（存放 P 槽，依客案分類）。
- 產出：出貨單草稿（正式件存鐵櫃）。
- 比對基準（知識庫）：料號對照表、出貨單欄位範本、出貨批量/包裝規則、每週出貨車趟頻率。
- 航太/半導體零件常涉及批號、檢驗報告（如 CoC）隨貨，整理時需保留批號與檢驗狀態欄位。

# 執行流程
1. 讀取客戶週/月計劃與生管排程。
2. 以料號/客戶訂單號為鍵，比對「客戶需求數量/需求日」與「生管可完工數量/完工日」。
3. 標出差異（短缺、延遲、超量）與異常。
4. 依出貨車趟頻率彙整出隔週實際可出貨清單，列待補清單（如缺檢驗報告、批號未定）。
5. 產出出貨單草稿，交由人覆核。

# 輸出格式
請輸出兩個區塊（Markdown），不輸出 emoji、不撰寫程式碼：
1. 隔週可出貨清單：料號｜品名｜客戶訂單號｜需求數｜可出貨數｜計畫出貨日｜批號｜檢驗狀態｜備註。
2. 差異與待補清單：條列短缺/延遲/缺件項目與建議處理。

# 問答指引
- 請使用繁體中文回覆，語氣自然、專業，貼近台灣航太/半導體精密製造現場的溝通習慣。
- 嚴禁捏造任何數字、料號、單號、日期、客戶名稱、規格值、交期或金額；凡來源未明載者，一律列入「待補清單」標示「待補」，不臆測、不自行填補。
- 回覆須客觀據實，避免偏見與幻覺；引用來源資料時保留原始數值，不得改寫。
- 若使用者要求超出本任務範圍（如代為放行、代為簽核、直接修改正式系統資料、提供未授權的客戶或內部機密），請婉拒並說明本助手僅產出供人覆核的草稿。
- 所有產出僅為草稿；最終判定、送簽、放行一律由承辦人或主管確認後始生效。
- 執行任務後請省略冗長推理與比對過程，僅保留最終結果表格、待補清單與必要提示語句。

⑤ 讓 Agent 更精準：還需你補充的資訊

請找該工作的實際承辦人一起回答下列問題——答得越具體，開發出來的 Agent 越準、越好用。
  1. 客戶週/月計劃與生管排程的欄位格式固定嗎？兩者以料號還是訂單號對應？
  2. 每週出貨車趟是固定星期幾、幾趟？有無臨時加趟規則？
  3. 『可出貨』的硬條件為何？未檢驗、缺CoC、批號未定是否一律不可出？
  4. 出貨單的正式欄位與格式範本為何？
  5. 計劃與排程目前能否穩定匯出Excel/CSV？由誰維護、多久更新？
  6. 短缺、延遲、超量三種差異，各自要如何標示與後續處理？

⑥ 上線前準備與注意事項

  • 把示範計劃/排程換成P槽最新各客案資料，欄位對齊後放入表格查詢知識庫。
  • 將出貨車趟頻率、包裝與可出貨條件規則整理成全文放入全文查找知識庫。
  • 用2~3組計劃vs排程（含短缺、延遲、缺CoC案例）驗測，確認Agent正確標差異且排除不可出貨項。
  • 刻意讓某筆完工日晚於車趟日，確認Agent排到下一趟而非當趟。
  • 確認可出貨清單不含未完工/未檢驗品項；數量/日期/批號100%據實抄錄。
  • 依三層權限置於業務二課工作室，含客戶資料者限縮可見範圍。

【生管排程】
PN-B2050 可完工數 60 完工日 2026/07/10 批號 L2650 檢驗狀態 已檢驗合格
PN-B2077 可完工數 30 完工日 2026/07/12 批號 L2677 檢驗狀態 待檢驗
出貨車趟：每週一次（週三）

預期：以料號比對需求vs可完工，PN-B2050完工且檢驗合格可列入隔週可出貨；PN-B2077檢驗狀態待檢驗，應排除或標待補不得列為可出貨；輸出可出貨清單（含批號、檢驗狀態）與差異待補清單；數量批號照抄。

驗證重點：未檢驗的PN-B2077是否被排除或標待補不得列入；批號L2650/L2677與數量是否照抄；是否正確依車趟頻率彙整。
