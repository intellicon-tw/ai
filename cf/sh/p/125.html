# 暫不適用 EgentHub（純實體作業）

本題「倉管作業ー進料整理、入庫、出庫、運送」屬千附廠務工程駐廠（台積等高科技廠）之純實體倉儲物流作業（依工程部預留單（助理 ERP）進行進料整理、入庫、出庫與運送）。進料整理、入庫上架、出庫揀貨與運送均為實體倉管作業，AI 無法代為執行，暫不適用 EgentHub 直接開發。

可考慮就周邊文書另立題目，例如：預留單欄位標準化與備料清單彙整、入出庫明細登錄草稿、料件對照預留單差異標示、領料/退料紀錄整理草稿。建議搭配 WMS／條碼／RFID 等實體流程數位化工具，先把出料/入庫資料變成結構化紀錄，再讓 Agent 接手後段文書彙整。
