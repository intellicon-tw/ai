# 跨境供應鏈追蹤助理 - 驗測情境

以下 3 個情境,可直接複製測試輸入到 EgentHub 對話框驗測。
每個情境含:測試輸入、預期輸出、驗證重點。

## 情境 #1:單批追蹤

**測試輸入**

```
Rootopia Brew 出貨到日本的批次號 RB20260420-JP,目前狀態如何?預計幾號到?
```

**預期輸出**

> 目前階段 + 預計到貨時間 + 是否異常

**驗證重點**

> 資料引用準確;階段正確;異常標註

## 情境 #2:期間總覽

**測試輸入**

```
未來 14 天有哪些跨境出貨?有沒有遲交風險?
```

**預期輸出**

> 出貨清單表 + 風險評估 + 建議行動

**驗證重點**

> 資料完整;風險判斷合理

## 情境 #3:異常處理

**測試輸入**

```
上週出到 LA 的批次卡在洛杉磯港口,可能原因?怎麼處理?
```

**預期輸出**

> 可能原因 + 歷史相似案例 + 建議與物流商溝通要點

**驗證重點**

> 原因分析合理;引用歷史;建議具體
