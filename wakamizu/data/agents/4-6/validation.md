# 空間品牌體驗AI化 - 驗測情境

以下 3 個情境,可直接複製測試輸入到 EgentHub 對話框驗測。
每個情境含:測試輸入、預期輸出、驗證重點。

## 情境 #1:靜謐自然

**測試輸入**

```
(旅客)我喜歡靜靜在自然中,可不可以推薦溪頭附近的隱藏路線?
```

**預期輸出**

> 深度體驗推薦 + 搭配影片 + 搭配風味

**驗證重點**

> 推薦扣到「靜謐」;結合三模組;不過度商業化

## 情境 #2:職人文化

**測試輸入**

```
(旅客)我對日本職人文化有興趣,熊本人吉旅居時可以怎麼安排?
```

**預期輸出**

> 職人體驗推薦 + 搭配 Content 影片 + Rootopia 風味

**驗證重點**

> 深度元素;與職人故事連結;不堆景點

## 情境 #3:知識庫不足

**測試輸入**

```
(旅客)我想看看與那國島深夜的星空與當地民俗音樂
```

**預期輸出**

> 誠實告知部分資訊不足,建議現場管家進一步協助

**驗證重點**

> 不捏造;轉介明確
