# 內容企劃助理 - 正式上線前的準備指南

本指南協助若水金禾相關部門在 EgentHub 上線此 Agent 前,完成資料準備、提示詞校對、驗測與權限設定。

---

## 一、知識庫資料準備

1. **頻道歷史數據**(2 週)
   - 100+ 部影片的觀看、留存、訂閱、互動資料
   - 結構化標籤:主題、商業目標、達成度

2. **品牌調性指南**

3. **競品頻道分析**
   - 至少 5 個競品的內容策略

4. **Google Trends/關鍵字工具資料**
   - 月更新

## 二、提示詞校對重點

- 提案是否扣到商業目標
- 標題是否避免「網紅化」過度
- 是否會推薦過於小眾或無素材的方向

## 三、驗測通過標準

- 3 個情境通過
- 頻道製作人試用 1 個月,實際拍攝 5 支經 Agent 啟發的影片

## 四、EgentHub 平台設定建議

- 模型:Claude 4.6 Sonnet
- 知識庫:Agent 向量
- 權限:斑米多媒體企劃、製作人、品牌經理
- 算力配額:每月 60,000 點

## 五、第一週試運轉觀察重點

- 提案是否被採用
- 標題建議的點擊率
- SEO 關鍵字的搜尋量

## 六、風險提醒與避坑建議

- **創意被替代風險**:Agent 提案可能讓企劃過度依賴,失去原創力,建議「Agent 啟發、人類定案」
- **競品資料時效**:競品策略會變化,需定期更新
