# 內容企劃助理 - 驗測情境

以下 3 個情境,可直接複製測試輸入到 EgentHub 對話框驗測。
每個情境含:測試輸入、預期輸出、驗證重點。

## 情境 #1:新品上市企劃

**測試輸入**

```
Rootopia Brew 即將推出 2026 春摘 4 款新品,我想拍一支系列 3 集介紹影片,目標導流到電商,目標受眾 30-45 歲咖啡與茶愛好者。
```

**預期輸出**

> 3 個提案,每個含大綱、3 版標題、SEO、商業效益、可重用素材

**驗證重點**

> 提案扣商業目標;標題具點擊力;SEO 合宜

## 情境 #2:民宿開幕系列

**測試輸入**

```
溪頭四季會館 2026 年 7 月試營運,我們要在開幕前 3 個月開始預告,規劃 5 支影片,目標是訂房預約導流。
```

**預期輸出**

> 5 支系列影片企劃,有連貫敘事、季節性考量

**驗證重點**

> 系列邏輯合理;每支可獨立看;順序合理

## 情境 #3:下半年規劃

**測試輸入**

```
請幫我規劃 2026 下半年(7-12 月)的 8 支影片企劃,要平衡內容、風味、空間三大模組。
```

**預期輸出**

> 8 個提案分布合理,扣季節、含三模組

**驗證重點**

> 考慮季節;涵蓋三模組;搭配商業活動
