# 茶園履歷與溯源助理 - 驗測情境

以下 3 個情境,可直接複製測試輸入到 EgentHub 對話框驗測。
每個情境含:測試輸入、預期輸出、驗證重點。

## 情境 #1:結構化履歷查詢

**測試輸入**

```
請整理 NO.3 阿里山石棹茶園的完整履歷,要附海拔、品種、有機認證狀態、年產量。
```

**預期輸出**

> 完整履歷卡(編號、地理位置、海拔、品種、認證、製程、年產量),並引用知識庫文件

**驗證重點**

> 資料是否完整;表格格式正確;引用清楚

## 情境 #2:包裝盒產地故事

**測試輸入**

```
為 Rootopia Brew 春摘批次寫一段適合包裝盒的產地故事,約 150 字,符合根本在旅行品牌調性,別寫太浮誇。
```

**預期輸出**

> 符合品牌調性的故事文案,150 字左右,扣到產地與職人精神

**驗證重點**

> 字數準確;符合根本在旅行語氣;不浮誇

## 情境 #3:多茶園比較

**測試輸入**

```
比較 NO.1 杉林溪、NO.5 鹿谷、NO.9 蘭陽三座茶園的差異,並說明各自適合搭配什麼咖啡品牌做組合包。
```

**預期輸出**

> 對照表 + 推薦使用情境(如組合包、活動主打)

**驗證重點**

> 對照欄位一致;推薦情境具商業可行性
