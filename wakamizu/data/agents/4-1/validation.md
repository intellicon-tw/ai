# 房務SOP助理 - 驗測情境

以下 4 個情境,可直接複製測試輸入到 EgentHub 對話框驗測。
每個情境含:測試輸入、預期輸出、驗證重點。

## 情境 #1:據點+房型查詢

**測試輸入**

```
宜蘭根本家民宿寵物房入住前的標準清潔流程?需要特別注意什麼備品?
```

**預期輸出**

> 結構化操作步驟、備品清單、預估工時、注意事項

**驗證重點**

> 據點與房型對應正確;步驟完整;含寵物房特殊備品

## 情境 #2:日文自動切換

**測試輸入**

```
熊本人吉旅館の備品規格について教えてください。
```

**預期輸出**

> 整段以日文回應,品牌專有名詞保留原文

**驗證重點**

> 語言切換準確;品牌名未亂譯

## 情境 #3:客訴處理

**測試輸入**

```
客人剛 check-in 後反映冷氣有異味,標準處理流程?
```

**預期輸出**

> 提示先安撫客人、再進行操作步驟、最後回報主管

**驗證重點**

> 強調安撫優先;含回報主管;符合 SOP

## 情境 #4:知識庫未涵蓋據點

**測試輸入**

```
請問東京共享辦公室的清潔 SOP 是?
```

**預期輸出**

> 誠實告知尚未建置完整 SOP,建議聯絡駐東京同仁

**驗證重點**

> 不捏造;轉介明確
