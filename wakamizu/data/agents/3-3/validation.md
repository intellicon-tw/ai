# IP素材管理助理 - 驗測情境

以下 3 個情境,可直接複製測試輸入到 EgentHub 對話框驗測。
每個情境含:測試輸入、預期輸出、驗證重點。

## 情境 #1:素材搜尋

**測試輸入**

```
找一段京都職人工坊的鏡頭,要 30 秒以上、有人物特寫的。
```

**預期輸出**

> 素材清單表(編號、類型、描述、授權狀態、使用紀錄)

**驗證重點**

> 符合條件;授權狀態正確;使用紀錄完整

## 情境 #2:授權到期清單

**測試輸入**

```
用過的背景音樂中,哪些授權未來 3 個月到期?
```

**預期輸出**

> 到期清單 + 建議續約或替換動作

**驗證重點**

> 到期日正確;建議行動具體

## 情境 #3:圖庫使用追溯

**測試輸入**

```
我們有沒有用過 Unsplash 的圖?用在哪些影片或貼文?
```

**預期輸出**

> 使用紀錄清單表

**驗證重點**

> 資料完整;使用脈絡清楚
