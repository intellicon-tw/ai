# IP素材管理助理 - 正式上線前的準備指南

本指南協助若水金禾相關部門在 EgentHub 上線此 Agent 前,完成資料準備、提示詞校對、驗測與權限設定。

---

## 一、知識庫資料準備

1. **既有素材清查**(4 週)
   - 100+ 部影片所用的素材回追
   - 結構化整理為素材資料庫

2. **授權合約掃描**(2 週)
   - 所有授權合約存檔(PDF)
   - 重點欄位提取(來源、期限、限制)

3. **使用紀錄補建**
   - 每筆素材何時用在何處

## 二、提示詞校對重點

- 授權到期日計算是否準確
- 是否會誤判「商用授權」與「個人授權」
- 對 CC0 等開放授權的識別

## 三、驗測通過標準

- 3 個情境通過
- 剪輯師試用 2 週,搜尋成功率 95% 以上
- 至少抓到 3 個過去未注意的即將到期授權

## 四、EgentHub 平台設定建議

- 模型:Claude 4.6 Sonnet
- 知識庫:表格 + 全文查找
- 權限:斑米多媒體企劃、剪輯師、製作人
- 算力配額:每月 30,000 點

## 五、第一週試運轉觀察重點

- 是否真的能避免授權誤用
- 對複雜授權條款的解讀準確度

## 六、風險提醒與避坑建議

- **法律風險**:授權誤用會有法律後果,Agent 提供查詢協助,但最終確認需法務或外部顧問
- **資料完整性**:若舊素材使用紀錄缺失,會有授權盲區
