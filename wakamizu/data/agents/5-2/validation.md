# 跨境法規與合規助理 - 驗測情境

以下 3 個情境,可直接複製測試輸入到 EgentHub 對話框驗測。
每個情境含:測試輸入、預期輸出、驗證重點。

## 情境 #1:稅務申報時程

**測試輸入**

```
日本沖繩貿易子公司每年要繳哪些稅?何時繳?有沒有特殊優惠?
```

**預期輸出**

> 稅種清單 + 申報時程 + 沖繩特別優惠(如有)+ 提示需會計師確認

**驗證重點**

> 稅種完整;時程正確;標註會計師確認

## 情境 #2:勞動法跨國比較

**測試輸入**

```
比較台灣勞基法與日本勞動法在加班費的差異,對我們的跨國同仁影響?
```

**預期輸出**

> 比較表 + 對若水金禾跨國勞動成本的影響 + 建議行動

**驗證重點**

> 比較準確;影響評估合理;建議具體

## 情境 #3:移轉訂價提示

**測試輸入**

```
母公司向沖繩貿易子公司收取品牌授權金,這算移轉訂價嗎?要怎麼處理?
```

**預期輸出**

> 移轉訂價基本概念 + 三地稅局立場 + 建議由會計師設計合理價格

**驗證重點**

> 概念清楚;不給出具體比例;建議專業確認
