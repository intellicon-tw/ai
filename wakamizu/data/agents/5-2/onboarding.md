# 跨境法規與合規助理 - 正式上線前的準備指南

本指南協助若水金禾相關部門在 EgentHub 上線此 Agent 前,完成資料準備、提示詞校對、驗測與權限設定。

---

## 一、知識庫資料準備

1. **三地法規條文**(4 週)
   - 公司法、稅法、勞動法、商業法規
   - 由各地法律顧問提供認可摘要

2. **法律顧問意見書**
   - 過去 3 年的所有顧問意見書(去敏感化)

3. **稅務時程表**
   - 每年所有申報截止日

4. **移轉訂價政策**
   - 既有政策文件

## 二、提示詞校對重點

- 是否會自行給出具體稅率(應引用知識庫,並提示確認)
- 是否會自行給出法律意見(必須轉介專業)
- 跨地比較是否平衡

## 三、驗測通過標準

- 3 個情境通過
- 法務與財務試用 4 週,實際解決 5 個跨境問題
- 所有對外決策仍經法律顧問確認

## 四、EgentHub 平台設定建議

- 模型:Claude 4.6 Sonnet
- 知識庫:全文查找 + 向量
- 權限:執行長、法務、財務、貿易子公司負責人
- 算力配額:每月 50,000 點

## 五、第一週試運轉觀察重點

- 是否會給過於具體的法律建議
- 顧問轉介機制是否被遵守
- 法規時效性(會更新)

## 六、風險提醒與避坑建議

- **法律責任**:Agent 僅供查詢,不可作為法律意見依據
- **時效性**:法規會修訂,需每季校對
- **跨地差異**:法規實務可能與條文差距大,需在地顧問確認
