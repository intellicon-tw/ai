# 國際通路拓展助理 - 正式上線前的準備指南

本指南協助若水金禾相關部門在 EgentHub 上線此 Agent 前,完成資料準備、提示詞校對、驗測與權限設定。

---

## 一、知識庫資料準備

1. **既有通路紀錄整理**(2 週)
   - 過去接觸過的所有通路商紀錄
   - 包含失敗案例與原因

2. **通路條件對照**
   - 各通路的進場費、抽成、結算週期、退貨規則

3. **競品鋪貨情況**
   - 日本台灣茶競品(石垣島茶葉、新光茶等)
   - 美國亞洲茶競品

4. **展會紀錄**
   - 過去三年參展紀錄、效益、合作機會

## 二、提示詞校對重點

- 通路條件是否會更新(會變動,需定期復核)
- 聯繫信是否會過度承諾
- 是否會推薦不適合的通路

## 三、驗測通過標準

- 3 個情境通過
- 海外業務試用 4 週,新接觸 5 個通路
- 業務開發信經品牌經理認可

## 四、EgentHub 平台設定建議

- 模型:Claude 4.6 Sonnet
- 知識庫:向量 + 表格
- 權限:海外業務、品牌經理、貿易子公司
- 算力配額:每月 40,000 點

## 五、第一週試運轉觀察重點

- 開發信是否符合各國商業書信慣例
- 是否會洩漏其他通路商的條件
- 競品分析是否合理

## 六、風險提醒與避坑建議

- **商業機密**:通路條件與競品定價屬敏感資料
- **跨國法規**:各國通路法規差異大,Agent 提供概要,實際合約需法務審核
- **品牌一致性**:不同市場的訊息需統一,避免品牌錯亂
