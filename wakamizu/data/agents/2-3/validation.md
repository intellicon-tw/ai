# 國際通路拓展助理 - 驗測情境

以下 3 個情境,可直接複製測試輸入到 EgentHub 對話框驗測。
每個情境含:測試輸入、預期輸出、驗證重點。

## 情境 #1:日本選品店拓展

**測試輸入**

```
我想在日本東京尋找適合 Rootopia Brew 進駐的高端選品店,請給我目標清單與聯繫建議。
```

**預期輸出**

> 目標選品店清單(已蒐集者)+ 進場條件 + 聯繫信草稿

**驗證重點**

> 是否引用既有資料;聯繫信是否符合品牌調性

## 情境 #2:美國電商通路

**測試輸入**

```
想了解 Whole Foods 線上電商與亞洲超市網路電商的差異,我們應該先進哪個?
```

**預期輸出**

> 通路對照表 + 進場難度 + 建議優先順序

**驗證重點**

> 通路資料具體;建議邏輯清楚

## 情境 #3:競品定價蒐集

**測試輸入**

```
彙整日本市場上 14 款台灣茶品牌的定價區間。
```

**預期輸出**

> 競品定價對照表 + 定位分析

**驗證重點**

> 定價資訊真實(若無則註明需補充);分析合理
