# 體驗行程設計助理 - 驗測情境

以下 3 個情境,可直接複製測試輸入到 EgentHub 對話框驗測。
每個情境含:測試輸入、預期輸出、驗證重點。

## 情境 #1:宜蘭兩日行程

**測試輸入**

```
幫我設計宜蘭根本家民宿的春季兩日行程,客群是高端親子,預算每組 NT$15,000 左右(含住宿)。
```

**預期輸出**

> 兩日完整行程 + 在地餐飲 + 體驗活動 + 定價

**驗證重點**

> 行程符合季節與客群;定價合理;含合作方確認項

## 情境 #2:熊本一日體驗

**測試輸入**

```
為熊本人吉旅館客人設計『日本職人探訪』一日體驗,客群是咖啡愛好者,預算 ¥15,000/人。
```

**預期輸出**

> 一日行程,含在地咖啡店與烘豆師訪問,搭配 Rootopia Brew 體驗

**驗證重點**

> 在地元素具體;與品牌結合;預算合理

## 情境 #3:與那國島深度旅居

**測試輸入**

```
與那國島 9 棟開幕後,設計三日深度旅居體驗,客群是想暫離都會的高端旅人,預算每人 ¥45,000/三天。
```

**預期輸出**

> 三日深度行程,結合自然、職人、料理、海洋

**驗證重點**

> 扣到旅居感;節奏不過密;符合品牌靜謐調性
