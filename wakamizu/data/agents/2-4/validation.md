# 產品製造知識傳承系統 - 驗測情境

以下 3 個情境,可直接複製測試輸入到 EgentHub 對話框驗測。
每個情境含:測試輸入、預期輸出、驗證重點。

## 情境 #1:工藝問題

**測試輸入**

```
春茶萎凋時長要怎麼判斷?今天溫度 22 度、濕度 65%。
```

**預期輸出**

> 資深師傅判斷邏輯 + 建議萎凋時長 + 觀察重點

**驗證重點**

> 是否引用實際師傅原話;建議是否具操作性;標註需確認項

## 情境 #2:配方調整

**測試輸入**

```
金萱茶最新批次風味偏弱,想加強花香,該怎麼調?
```

**預期輸出**

> 可能原因 + 配方調整建議 + 歷史相似案例

**驗證重點**

> 原因分析全面;建議謹慎(配方屬機密);引用歷史案例

## 情境 #3:知識庫不足提示

**測試輸入**

```
請問有沒有針對極端氣候(如颱風後)的特殊製程處理?
```

**預期輸出**

> 誠實告知此議題知識庫資料不足,建議直接諮詢資深師傅

**驗證重點**

> 不捏造;轉介明確
