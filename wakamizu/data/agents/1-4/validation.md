# 產品規格查詢助理 - 驗測情境

以下 3 個情境,可直接複製測試輸入到 EgentHub 對話框驗測。
每個情境含:測試輸入、預期輸出、驗證重點。

## 情境 #1:品牌精準查詢

**測試輸入**

```
豆香洞 Tokado 最近的淺焙水洗豆有哪幾款?客人想找帶花香、酸度低的咖啡。
```

**預期輸出**

> 回傳結構化規格表(品牌、產品名、產地、處理法、烘焙度、風味、沖煮),附推薦理由

**驗證重點**

> 品牌名拼寫正確;規格欄位齊全;推薦理由符合品牌調性

## 情境 #2:搭配建議

**測試輸入**

```
我們有客人來門市要找 Rootopia Brew 中適合搭配抹茶蛋糕的款式,請推薦 2-3 款並說明搭配理由。
```

**預期輸出**

> 回傳 2-3 款最相關產品 + 搭配理由 + 補充說明

**驗證重點**

> 推薦數量控制 5 項以內;搭配理由扣風味描述;不浮誇

## 情境 #3:防幻覺

**測試輸入**

```
請問 Blue Mountain 牙買加咖啡豆我們有代理嗎?規格如何?
```

**預期輸出**

> 誠實告知未代理,建議洽風味商品部

**驗證重點**

> 不捏造資訊;回應委婉專業
