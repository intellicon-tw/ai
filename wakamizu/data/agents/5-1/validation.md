# 跨國團隊協作翻譯助理 - 驗測情境

以下 3 個情境,可直接複製測試輸入到 EgentHub 對話框驗測。
每個情境含:測試輸入、預期輸出、驗證重點。

## 情境 #1:會議紀錄中翻日

**測試輸入**

```
請翻譯這份會議紀錄為日文(雙語對照),內容是 Rootopia Brew 與日本通路商的會議:『今日確認下半年訂單為 Q3 約 8000 包、Q4 約 12000 包,付款週期月結 60 天。』
```

**預期輸出**

> 中日雙語對照,商業用語得體

**驗證重點**

> 日文是否符合敬語標準;品牌名保留;月結 60 天有正確翻譯

## 情境 #2:合約條款日翻中

**測試輸入**

```
翻譯這段日文合約條款為中文:『本契約は、契約期間満了の 60 日前までに書面による解約通知がない限り、自動的に 1 年間更新されるものとする。』
```

**預期輸出**

> 翻譯為中文,並標註高敏感條款建議法務確認

**驗證重點**

> 翻譯準確;有提示法務確認

## 情境 #3:Email 中翻英

**測試輸入**

```
翻譯這封給 LA 物業管理的 Email 為英文:『您好,關於 LA Rootopia Brew 展店的物業管理合約,我們希望增加一條關於品牌標識物使用的條款,可否安排下週視訊討論?』
```

**預期輸出**

> 得體的英文 Email,符合美國商業書信慣例

**驗證重點**

> 英文流暢;商業用語得體;邀約句型清楚
