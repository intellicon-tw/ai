# 多語系旅客服務助理 - 驗測情境

以下 4 個情境,可直接複製測試輸入到 EgentHub 對話框驗測。
每個情境含:測試輸入、預期輸出、驗證重點。

## 情境 #1:中文常見問題

**測試輸入**

```
旅客以中文問:『請問宜蘭根本家可以帶寵物嗎?有沒有專屬房型?』
```

**預期輸出**

> 中文回覆草稿 + 個人化建議

**驗證重點**

> 回覆準確;符合品牌調性;含個人化建議

## 情境 #2:日文 check-in 詢問

**測試輸入**

```
旅客以日文問:『熊本人吉旅館のチェックインは何時からですか?早めにチェックインできますか?』
```

**預期輸出**

> 日文回覆草稿 + 是否能早 check-in 之政策

**驗證重點**

> 日文流暢自然;政策說明清楚

## 情境 #3:VIP 旅客特殊需求

**測試輸入**

```
VIP 旅客以英文問:『We will arrive at 11pm, can the staff prepare welcome tea?』(住宿熊本人吉)
```

**預期輸出**

> 英文回覆草稿 + 標註建議轉真人主管確認

**驗證重點**

> 英文得體;標註轉接;不直接承諾

## 情境 #4:複雜客訴

**測試輸入**

```
旅客以中文抱怨:『冷氣噪音太大,昨天整晚睡不好,要求退費!』
```

**預期輸出**

> 標註複雜客訴,建議直接由主管處理

**驗證重點**

> 識別出複雜客訴;不直接給退費承諾
