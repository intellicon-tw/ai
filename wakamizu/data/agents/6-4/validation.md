# 投資人簡報助理 - 驗測情境

以下 3 個情境,可直接複製測試輸入到 EgentHub 對話框驗測。
每個情境含:測試輸入、預期輸出、驗證重點。

## 情境 #1:Q1 投資人更新

**測試輸入**

```
請更新 2026 Q1 投資人簡報,涵蓋三大模組進展與跨國佈局。
```

**預期輸出**

> 簡報草稿(里程碑、財務、模組、跨國、展望)+ 可能 Q&A

**驗證重點**

> 資料引用準確;結構完整;Q&A 預想合理

## 情境 #2:募資 Pitch

**測試輸入**

```
下一輪募資準備中,請幫我設計重點訴求的 5 頁 Pitch Deck 內容。
```

**預期輸出**

> 5 頁重點(問題、方案、市場、實績、團隊)的內容草稿

**驗證重點**

> 扣到 Airbnb × Blue Bottle × Netflix 願景;實績援引正確;不浮誇

## 情境 #3:高敏感數字提醒

**測試輸入**

```
Q1 我們的營收成長多少?
```

**預期輸出**

> 引用最新財務,並標註「需執行長/財務長最終確認後對外」

**驗證重點**

> 資料引用真實;有確認提示
