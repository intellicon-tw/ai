# 投資人簡報助理 - 正式上線前的準備指南

本指南協助若水金禾相關部門在 EgentHub 上線此 Agent 前,完成資料準備、提示詞校對、驗測與權限設定。

---

## 一、知識庫資料準備

1. **歷史簡報整理**(2 週)
   - 過去所有對外簡報
   - 標準化模板

2. **里程碑時間軸**
   - 2022 至今所有重要事件

3. **財務報表整合**
   - 月、季、年報

4. **歷史投資人問題清單**

註:此 Agent 為 L3b 長期任務,建議在 ROE/ROI 階段成熟後啟動。

## 二、提示詞校對重點

- 數字引用是否真實
- 是否會誇大實績
- 高敏感數字是否要求確認

## 三、驗測通過標準

- 3 個情境通過
- 執行長與財務長共同認可
- 至少 2 場投資人會議實際使用

## 四、EgentHub 平台設定建議

- 模型:Claude 4.6 Sonnet
- 知識庫:Agent 向量 + 表格
- 權限:**僅執行長、財務長、董事會**
- 算力配額:每月 15,000 點

## 五、第一週試運轉觀察重點

- 此 Agent 上線預估在 ROE/ROI 完成後
- 上線前需執行長、財務長、外部 IR 顧問共同確認

## 六、風險提醒與避坑建議

- **財務機密**:涉及未公開財務數字,嚴格權限
- **法律合規**:對外揭露需符合公開發行規定(若日後 IPO)
- **IR 一致性**:對不同投資人的訊息需一致,Agent 需確保訊息統一
