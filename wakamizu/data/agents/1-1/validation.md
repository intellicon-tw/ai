# 出口法規合規助理 - 驗測情境

以下 3 個情境,可直接複製測試輸入到 EgentHub 對話框驗測。
每個情境含:測試輸入、預期輸出、驗證重點。

## 情境 #1:日本市場標示設計

**測試輸入**

```
Rootopia Brew 日月潭紅玉紅茶要外銷日本,請依日本食品衛生法產出標示範本與所需認證清單。
```

**預期輸出**

> 完整的日文標示範本(原料、營養成分、有效期限格式)、JAS 與有機認證需求、進口許可申請清單、可能需事前諮詢日本顧問的項目

**驗證重點**

> 是否引用日本食品衛生法具體條文;標示範本是否符合日本格式;是否標出敏感項需專家確認

## 情境 #2:美國 FDA FSVP 流程

**測試輸入**

```
我們新代理的衣索比亞咖啡豆要進美國,請說明 FDA FSVP 完整流程與我們需準備的文件。
```

**預期輸出**

> FSVP 流程步驟、Foreign Supplier 審查重點、Hazard Analysis 需求、所需文件清單

**驗證重點**

> 流程是否完整;是否包含台灣端供應商審查文件;是否提示 FDA 申請費用差異

## 情境 #3:MRL 跨國比較

**測試輸入**

```
比較台灣、日本、美國三地茶葉農藥殘留 MRL 標準的差異,以包裝茶為主。
```

**預期輸出**

> 三地 MRL 對照表(常見農藥)、最嚴格的標準、對 Rootopia Brew 茶園採購的建議

**驗證重點**

> 是否實際引用三地法規;是否點出最嚴格的市場;是否給出操作建議
