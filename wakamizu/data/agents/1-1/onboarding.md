# 出口法規合規助理 - 正式上線前的準備指南

本指南協助若水金禾相關部門在 EgentHub 上線此 Agent 前,完成資料準備、提示詞校對、驗測與權限設定。

---

## 一、知識庫資料準備

1. **法規文件蒐集**(預估 2-3 週)
   - 台灣:食品安全衛生管理法、食品標示法、CAS 認證辦法
   - 日本:食品衛生法、食品表示法、有機 JAS 規範、各都道府縣特殊規定
   - 美國:FDA FSMA、FSVP、Prop 65、各州標示要求

2. **農藥殘留標準對照表**
   - 整理茶葉、咖啡常用農藥的三地 MRL
   - 建議由貿易子公司同仁與外部顧問聯合確認

3. **既有產品標示檔案**
   - 提供 Rootopia Brew 14 款現有中文標示
   - 既有外銷產品的英文/日文標示範本

4. **匯入 EgentHub**:法規條文用「全文查找知識庫」,MRL 用「表格查詢知識庫」

## 二、提示詞校對重點

- 確認知識庫覆蓋的國家清單與實際業務地區一致
- 確認回應時是否會主動提示「需專業顧問確認」(避免成為法律意見)
- 測試多語提問是否能正確處理(中文 / 日文 / 英文)

## 三、驗測通過標準

- 完成 2_驗測情境.txt 的全部 3 個情境
- 至少 2 名業務人員實際使用 1 週,正確率達 90% 以上
- 任何高風險回應(如關稅金額、罰則具體比例)都應建議「請洽外部顧問」

## 四、EgentHub 平台設定建議

- 模型:Claude 4.6 Sonnet
- 知識庫:全文查找(法規條文)+ 表格查詢(MRL 對照)
- 權限:貿易子公司、Rootopia Brew 品牌經理、外銷業務、管理部
- 算力配額:每月 100,000 點(中度使用)

## 五、第一週試運轉觀察重點

- 觀察是否有業務同仁將 Agent 回應當成「最終法律意見」直接傳給客戶(應為禁止行為)
- 收集前 50 個提問樣本,評估知識庫盲區
- 注意是否有捏造法規引用(請所有使用者標註可疑回應)

## 六、風險提醒與避坑建議

- **法律責任風險**:Agent 僅提供查詢協助,不可作為合規認定依據,所有對外回應仍需法律顧問確認
- **法規時效性**:法規會更新,知識庫每季校對一次,建議由貿易子公司負責
- **跨國差異**:不同地區海關實務可能與條文差異大,提示業務勿過度仰賴 Agent
