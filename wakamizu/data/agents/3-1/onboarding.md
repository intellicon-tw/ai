# 多語系字幕與文案產生器 - 正式上線前的準備指南

本指南協助若水金禾相關部門在 EgentHub 上線此 Agent 前,完成資料準備、提示詞校對、驗測與權限設定。

---

## 一、知識庫資料準備

1. **品牌術語對照表**(已附範本,需擴充至 100+ 詞)
2. **頻道風格指南**(範本已附,需斑米多媒體共識後定稿)
3. **過往高互動貼文**(IG/FB/LINE 各取 50 篇)
4. **既有翻譯案例**(校對過的中日英對照)

## 二、提示詞校對重點

- 品牌術語是否未直譯
- 字幕字數是否合宜
- 多平台調性差異是否清楚

## 三、驗測通過標準

- 3 個情境通過
- 斑米多媒體小編試用 2 週,可獨立產出影片字幕
- 翻譯經外籍同仁校對後對外

## 四、EgentHub 平台設定建議

- 模型:Claude 4.6 Sonnet
- 知識庫:Agent 向量
- 權限:斑米多媒體小編、剪輯師、企劃、品牌經理
- 算力配額:每月 100,000 點

## 五、第一週試運轉觀察重點

- 品牌術語錯譯率
- 各平台貼文是否有調性差異
- 是否會自動加入不適合的 hashtag

## 六、風險提醒與避坑建議

- **品牌一致性**:多語版本若調性偏離會傷品牌
- **法律用語**:涉及 NDA、合作條款等不應自動翻譯
- **文化差異**:某些梗在不同國家失效,需人工最後校對
