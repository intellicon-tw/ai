# 多語系字幕與文案產生器 - 驗測情境

以下 3 個情境,可直接複製測試輸入到 EgentHub 對話框驗測。
每個情境含:測試輸入、預期輸出、驗證重點。

## 情境 #1:中翻日 SRT

**測試輸入**

```
請把以下中文字幕翻譯成日文 SRT(時間軸保留):

00:00:01,000 --> 00:00:04,500
今天我們來到了京都的 Weekenders 咖啡

00:00:04,500 --> 00:00:08,200
這是當地咖啡迷的口袋名單
```

**預期輸出**

> 保留原時間軸的日文 SRT,單行不超過 42 字元

**驗證重點**

> 時間軸正確;格式 SRT;品牌名保留

## 情境 #2:多平台貼文套組

**測試輸入**

```
我們剛拍完豆ポレポレ 2025 日本烘豆冠軍的影片,影片重點是冠軍豆處理法。請幫我產出 IG/FB/LINE 三個平台版本(中文)。
```

**預期輸出**

> 三個分區清楚標題;IG 開頭吸睛 + hashtag;FB 故事感;LINE 簡短 + CTA

**驗證重點**

> 各平台調性符合;hashtag 相關;CTA 清楚

## 情境 #3:中日英三語預告

**測試輸入**

```
為熊本人吉旅館 2026 春開幕,寫一段三語社群預告文,各 100-150 字。
```

**預期輸出**

> 三語版本敘事一致,品牌術語保留

**驗證重點**

> 字數準確;品牌術語正確;語氣符合品牌
