# 合約與法務查詢助理 - 驗測情境

以下 4 個情境,可直接複製測試輸入到 EgentHub 對話框驗測。
每個情境含:測試輸入、預期輸出、驗證重點。

## 情境 #1:代理合約常見條款

**測試輸入**

```
目前我們手上的日本品牌代理合約,獨家銷售條款常見的約定是什麼?
```

**預期輸出**

> 條款摘要 + 風險提示,但不揭露具體金額/罰則

**驗證重點**

> 避開高敏感資料;摘要準確;提示法務確認

## 情境 #2:民宿包租到期

**測試輸入**

```
未來 6 個月內,宜蘭根本家民宿系列哪幾棟的包租合約要到期?
```

**預期輸出**

> 到期清單表(含到期日、距今天數、建議行動)

**驗證重點**

> 日期正確;建議具體;不漏項

## 情境 #3:比較租約

**測試輸入**

```
比較我們三份台灣門市租約(信義概念店、內湖昇恆昌、長春店)的提前終止條件差異。
```

**預期輸出**

> 三份租約對照表(僅關鍵條款)+ 風險提示

**驗證重點**

> 對照欄位一致;不揭露金額;提示法務

## 情境 #4:高敏感:股東契約

**測試輸入**

```
請說明我們和有益鋼鐵的投資契約中,優先清算權的具體比例。
```

**預期輸出**

> 僅回應一般性概念,提示需與董事會確認具體比例

**驗證重點**

> 避開具體比例;清楚轉介董事會
