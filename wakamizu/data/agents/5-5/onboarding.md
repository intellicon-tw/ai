# 國際市場情報系統 - 正式上線前的準備指南

本指南協助若水金禾相關部門在 EgentHub 上線此 Agent 前,完成資料準備、提示詞校對、驗測與權限設定。

---

## 一、知識庫資料準備

1. **資料源建立**(6-12 個月)
   - 產業研究報告訂閱
   - 競品官網與社群監測
   - 政策追蹤(各地公告)
   - 展會清單與年度排程

2. **資料更新機制**
   - 每週抓取、每月分析

3. **第二階段考慮工具串接**(產業 RSS、Google Alerts)

【提醒】此 Agent 為 L3 長期任務,適合在 ROE/ROI 階段成熟後啟動。

## 二、提示詞校對重點

- 資料是否真實(避免捏造競品)
- 啟示是否扣到若水金禾策略
- 行動建議具體可行

## 三、驗測通過標準

- 至少 3 個月資料累積
- 執行長與品牌策略團隊認可月報品質
- 每月實際採用 1-2 個建議

## 四、EgentHub 平台設定建議

- 模型:Claude 4.6 Sonnet
- 知識庫:Agent 向量
- 權限:執行長、品牌策略、業務拓展
- 算力配額:每月 30,000 點

## 五、第一週試運轉觀察重點

- 此 Agent 上線預估在 ROE/ROI 完成後
- 上線初期供董事會與管理層使用

## 六、風險提醒與避坑建議

- **資料時效**:競品策略變化快,需固定更新節奏
- **資料來源可信度**:需區分公開資訊與商業情報的差異
- **不可侵權**:不可使用付費報告未授權資料
