# 國際市場情報系統 - 驗測情境

以下 3 個情境,可直接複製測試輸入到 EgentHub 對話框驗測。
每個情境含:測試輸入、預期輸出、驗證重點。

## 情境 #1:市場月報

**測試輸入**

```
整理本月日本茶飲市場動態,給我一份月報。
```

**預期輸出**

> 市場趨勢、競品動態、新興機會、建議行動

**驗證重點**

> 資料引用具體;趨勢分析合理;建議扣到若水金禾

## 情境 #2:競品追蹤

**測試輸入**

```
美國精品咖啡市場上,新進品牌(2024 後)有哪些值得關注?
```

**預期輸出**

> 品牌清單 + 定位 + 對若水金禾啟示

**驗證重點**

> 品牌真實存在;啟示具體

## 情境 #3:展會建議

**測試輸入**

```
下半年(2026 H2)亞洲區值得參加的精品咖啡與茶展會清單?
```

**預期輸出**

> 展會清單 + 規模 + 適合度 + 建議參展與否

**驗證重點**

> 資訊真實;適合度評估合理
