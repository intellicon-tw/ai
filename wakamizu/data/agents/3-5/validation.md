# AI內容飛輪引擎 - 驗測情境

以下 3 個情境,可直接複製測試輸入到 EgentHub 對話框驗測。
每個情境含:測試輸入、預期輸出、驗證重點。

## 情境 #1:帶貨歸因分析

**測試輸入**

```
上一季哪些影片最會帶 Rootopia Brew 的訂單?ROI 排名?
```

**預期輸出**

> Top 帶貨內容表 + ROI 排名 + 共通特徵分析

**驗證重點**

> 資料引用準確;歸因邏輯清楚;分析有洞察

## 情境 #2:下季主題推薦

**測試輸入**

```
下個季度應推什麼帶貨主題?根據過去歸因。
```

**預期輸出**

> 3-5 個主題 + 預期帶貨效益 + 建議製作節奏

**驗證重點**

> 扣到實際歸因;建議具體可執行

## 情境 #3:資料缺口提示

**測試輸入**

```
為何上週某支影片帶貨明顯比預期低?
```

**預期輸出**

> 分析原因 + 標註資料缺口 + 建議補充追蹤

**驗證重點**

> 誠實標註不確定性;建議合理
