## 內容飛輪資料模型(L2 規劃稿)

### 1. 資料來源
- YouTube Analytics(觀看、留存、CTR、訂閱)
- IG/FB Insights(互動、保存、分享)
- 電商後台(訂單、商品、UTM)
- 廣告投放(花費、轉換)

### 2. 歸因模型
- 第一接觸 First Touch
- 最後接觸 Last Touch
- 線性 Linear
- 時間衰減 Time Decay

### 3. 計算欄位
- 內容 ROI = (歸因訂單金額 - 內容製作成本) / 內容製作成本
- 帶貨係數 = 影片觀看 → 商品點擊 → 加購 → 結帳的轉換率
- 內容半衰期 = 影片帶貨高峰至帶貨減半的時間

### 4. 飛輪指標
- 內容產出 → 流量 → 訂單 → 數據回饋 → 主題優化
- 各環節 KPI 與監控頻率

(此 Agent 為 L2 長期任務,需 12 個月以上資料累積。)
