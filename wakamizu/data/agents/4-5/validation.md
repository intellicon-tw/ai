# 據點展店評估助理 - 驗測情境

以下 3 個情境,可直接複製測試輸入到 EgentHub 對話框驗測。
每個情境含:測試輸入、預期輸出、驗證重點。

## 情境 #1:新城市評估

**測試輸入**

```
我們考慮在台中市火車站周邊開直營門市,可行性如何?
```

**預期輸出**

> 區域分析、成本估算、與台北既有店比較、風險、整體建議

**驗證重點**

> 資料引用合理;比較公平;風險不漏項

## 情境 #2:海外擴展

**測試輸入**

```
想在京都開設選品店 + 茶體驗空間,目標客群是年輕世代與華人遊客,評估可行性。
```

**預期輸出**

> 京都區域分析、競品定位、進場成本、文化挑戰、建議規模

**驗證重點**

> 京都特性具體;文化挑戰具體;成本合理

## 情境 #3:風險評估

**測試輸入**

```
與那國島 9 棟建置中,要怎麼評估地緣政治風險?
```

**預期輸出**

> 地緣政治風險清單 + 業務影響 + 緩解方案

**驗證重點**

> 風險清單具體;緩解方案合理;不過度悲觀也不過度樂觀
