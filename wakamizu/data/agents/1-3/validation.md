# 生產排程與品管助理 - 驗測情境

以下 3 個情境,可直接複製測試輸入到 EgentHub 對話框驗測。
每個情境含:測試輸入、預期輸出、驗證重點。

## 情境 #1:單品庫存比對

**測試輸入**

```
Rootopia Brew 春摘紅玉下個月要 5,000 包,目前原料夠嗎?代工廠 A 可以排得上嗎?
```

**預期輸出**

> 比對庫存與產能,給出排程建議與風險提示;若庫存不足,建議補貨數量與時程

**驗證重點**

> 是否引用配方表;是否考慮代工廠產能上限;補貨建議是否合理

## 情境 #2:品管異常處理

**測試輸入**

```
代工廠 B 反映上週批次的東方美人蜜香茶葉色澤偏暗,客訴 3 件,該怎麼處理?
```

**預期輸出**

> 對照 SOP 給出檢驗步驟、批次回收建議、與代工廠的後續溝通要點

**驗證重點**

> 處理流程是否完整;是否提示主管確認;是否引用過往相似案例

## 情境 #3:全局報表

**測試輸入**

```
未來 8 週的生產排程總覽,哪些產品有缺料風險?
```

**預期輸出**

> 彙整 14 支產品 8 週排程表,標出缺料風險產品與建議補貨時程

**驗證重點**

> 排程表完整;風險產品標註明確;建議行動具體可執行
