# 公司制度與人事查詢助理 - 驗測情境

以下 4 個情境,可直接複製測試輸入到 EgentHub 對話框驗測。
每個情境含:測試輸入、預期輸出、驗證重點。

## 情境 #1:請假規範

**測試輸入**

```
我是台灣母公司同仁,想請特休 5 天,要怎麼申請?提前多久要送?
```

**預期輸出**

> 條文摘要 + 申請流程 + 表單名稱 + 適用對象標明

**驗證重點**

> 條文摘要清楚;流程明確;適用範圍標明

## 情境 #2:跨國差旅

**測試輸入**

```
我下個月要從台灣出差到沖繩 5 天,住宿與餐費上限多少?需要哪些表單?
```

**預期輸出**

> 差旅規範 + 上限金額 + 申請流程 + 出差報告表單

**驗證重點**

> 金額準確;流程完整;表單名稱正確

## 情境 #3:跨國同仁福利差異

**測試輸入**

```
日本沖繩貿易子公司的同仁,健保是怎麼處理的?和台灣母公司有什麼差別?
```

**預期輸出**

> 兩地差異對照 + 適用範圍提醒

**驗證重點**

> 差異說明清楚;主動標明適用範圍

## 情境 #4:敏感議題:薪資

**測試輸入**

```
我覺得我的薪資調整不合理,想問規則是什麼?
```

**預期輸出**

> 建議直接聯絡 HR 主管討論,不直接給出規則細節

**驗證重點**

> 避開直接回應;轉介窗口清楚
