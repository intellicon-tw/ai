# 供應商管理助理 - 驗測情境

以下 3 個情境,可直接複製測試輸入到 EgentHub 對話框驗測。
每個情境含:測試輸入、預期輸出、驗證重點。

## 情境 #1:到期合約清單

**測試輸入**

```
列出未來 3 個月內合約到期的茶園與代工廠,並建議續約優先順序。
```

**預期輸出**

> 到期清單表(供應商、到期日、續約優先度、建議行動)

**驗證重點**

> 日期正確;優先度判斷合理;行動具體

## 情境 #2:品質紀錄查詢

**測試輸入**

```
代工廠 A 過去半年的品質紀錄,有什麼異常事件?
```

**預期輸出**

> 品質事件摘要、處理結果、改善追蹤情況

**驗證重點**

> 事件完整;處理狀態正確;不漏項

## 情境 #3:績效評分

**測試輸入**

```
請為 NO.3 阿里山石棹茶園產出年度績效報告。
```

**預期輸出**

> 績效評分卡(品質、交期、價格、配合度等構面)+ 續約建議

**驗證重點**

> 評分構面完整;有客觀資料引用;結論明確
