# 供應商管理助理 - 正式上線前的準備指南

本指南協助若水金禾相關部門在 EgentHub 上線此 Agent 前,完成資料準備、提示詞校對、驗測與權限設定。

---

## 一、知識庫資料準備

1. **供應商主檔整理**(2-3 週)
   - 範本已附 CSV,補充每筆完整資訊
   - 包含茶園、代工廠、日本品牌、包材廠、物流商

2. **歷史溝通紀錄**
   - 過去 12 個月的會議紀錄、Email 來往(需經採購主管授權)
   - 重要合約變更歷史

3. **品質檢驗紀錄**
   - 每批次進料檢驗結果(已有則匯入,無則需建立)

4. **匯入 EgentHub**:基本資料用表格、紀錄用向量、合約用全文查找

## 二、提示詞校對重點

- 到期日計算是否準確
- 績效評分是否會引用真實資料而非捏造
- 對 C 級供應商的處理建議是否符合內規

## 三、驗測通過標準

- 3 個情境通過
- 採購同仁試用 2 週,可獨立完成季度供應商盤點
- 績效報告經主管認可後才對供應商揭露

## 四、EgentHub 平台設定建議

- 模型:Claude 4.6 Sonnet
- 知識庫:表格 + 向量 + 全文查找(三類混用)
- 權限:採購、品牌經理、品管組、管理部
- 算力配額:每月 50,000 點

## 五、第一週試運轉觀察重點

- 是否會洩漏不同供應商的敏感比較(議價方面)
- 評分結果是否與資深採購直覺一致
- 觀察是否會建議「終止合約」這種高風險決策

## 六、風險提醒與避坑建議

- **商業機密**:供應商資料、價格、合約屬高敏感,Agent 回應需限內部使用
- **關係風險**:Agent 給出的供應商評分若外流會傷害合作關係
- **權限分級**:不同角色看到不同欄位(如採購看價格、品管看品質)
