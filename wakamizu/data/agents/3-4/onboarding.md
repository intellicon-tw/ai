# 內容數據分析師 - 正式上線前的準備指南

本指南協助若水金禾相關部門在 EgentHub 上線此 Agent 前,完成資料準備、提示詞校對、驗測與權限設定。

---

## 一、知識庫資料準備

1. **數據匯出流程設計**(2 週)
   - YouTube Analytics 每週匯出 CSV
   - IG/FB 洞察報告每週匯出
   - 統一彙整檔案格式

2. **歷史數據補建**(4 週)
   - 過去 12 個月所有影片數據

3. **競品數據蒐集**
   - 至少 5 個競品頻道的公開數據

4. **可選擴充**:第二階段可考慮 API 串接

## 二、提示詞校對重點

- 數據引用是否準確(避免捏造)
- 主題建議是否扣到實際表現
- 異常指標的判定門檻是否合理

## 三、驗測通過標準

- 3 個情境通過
- 製作人試用 4 週,週報節省 3 小時/週

## 四、EgentHub 平台設定建議

- 模型:Claude 4.6 Sonnet
- 知識庫:表格(主)
- 權限:製作人、品牌經理、行銷統籌
- 算力配額:每月 40,000 點

## 五、第一週試運轉觀察重點

- 數據引用準確率
- 建議是否被採用

## 六、風險提醒與避坑建議

- **數據時效性**:若匯出延遲,分析失準
- **API 串接風險**:若日後串接,需注意 YouTube API 配額與隱私規範
- **過度數據導向**:創意型內容可能不適合純數據驅動
