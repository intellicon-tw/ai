# 內容數據分析師 - 驗測情境

以下 3 個情境,可直接複製測試輸入到 EgentHub 對話框驗測。
每個情境含:測試輸入、預期輸出、驗證重點。

## 情境 #1:週報產出

**測試輸入**

```
彙整上週(2026/04/22-04/28)的內容表現,給我一份週報。
```

**預期輸出**

> Top 3 影片、觀眾輪廓、異常指標、主題建議

**驗證重點**

> 資料正確;趨勢分析合理;建議具體

## 情境 #2:觀眾分析

**測試輸入**

```
近三個月我們觀眾的年齡與地區分布有什麼變化?
```

**預期輸出**

> 趨勢對照表 + 變化分析 + 啟示

**驗證重點**

> 資料引用準確;分析有洞察

## 情境 #3:下月主題建議

**測試輸入**

```
根據近期表現,下個月應該推什麼主題?
```

**預期輸出**

> 3 個主題建議(扣到表現好的類型) + 預期效益

**驗證重點**

> 建議扣到實際數據;邏輯清楚
