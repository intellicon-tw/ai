# 旅宿營運數據助理 - 驗測情境

以下 3 個情境,可直接複製測試輸入到 EgentHub 對話框驗測。
每個情境含:測試輸入、預期輸出、驗證重點。

## 情境 #1:據點訂房率比較

**測試輸入**

```
上週宜蘭、溪頭、與那國島的訂房率比較。
```

**預期輸出**

> 三據點對照表 + 趨勢分析 + 差異原因推測

**驗證重點**

> 資料準確;對照清楚;分析合理

## 情境 #2:評價趨勢

**測試輸入**

```
3 月份各據點評價趨勢,有沒有負評集中的點?
```

**預期輸出**

> 評價彙整表 + 主要負評類別 + 改善建議

**驗證重點**

> 評價來源多元;負評歸納合理

## 情境 #3:客訴熱點

**測試輸入**

```
上週哪個據點客訴最多?是什麼類型?
```

**預期輸出**

> 客訴排行 + 類型分析 + 緊急度判斷

**驗證重點**

> 資料引用準確;類型分類合理
