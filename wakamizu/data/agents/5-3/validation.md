# 多幣別財務對帳助理 - 驗測情境

以下 3 個情境,可直接複製測試輸入到 EgentHub 對話框驗測。
每個情境含:測試輸入、預期輸出、驗證重點。

## 情境 #1:月結對帳

**測試輸入**

```
請產出 2026 年 4 月份四家子公司的合併對帳報表,基準幣別為新台幣。
```

**預期輸出**

> 多幣別對帳表 + 費用分攤明細 + 匯率損益 + 異常項

**驗證重點**

> 資料引用準確;匯率換算正確;分攤規則一致

## 情境 #2:關聯交易對帳

**測試輸入**

```
母公司 4 月開給沖繩貿易子公司的品牌授權金 NT$200,000,沖繩有沒有正確認列?
```

**預期輸出**

> 雙邊認列對照 + 差異 + 建議行動

**驗證重點**

> 認列對照清楚;差異邏輯合理

## 情境 #3:匯率損益分析

**測試輸入**

```
Q1 期間日圓匯率波動,我們的匯率損益是多少?哪個子公司影響最大?
```

**預期輸出**

> 匯率損益計算 + 影響度排序 + 建議避險方向

**驗證重點**

> 計算正確;分析合理;不給具體投資建議
