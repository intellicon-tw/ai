# 新品開發知識庫助理 - 驗測情境

以下 3 個情境,可直接複製測試輸入到 EgentHub 對話框驗測。
每個情境含:測試輸入、預期輸出、驗證重點。

## 情境 #1:新品方向評估

**測試輸入**

```
我們想開發冷萃柚子茶,目標市場是夏季年輕女性,請幫我評估可行性。
```

**預期輸出**

> 歷史相似配方、消費者回饋、競品定位、可行性評估、建議下一步

**驗證重點**

> 是否引用過去開發紀錄;是否考慮競品;建議是否具操作性

## 情境 #2:歷史配方查詢

**測試輸入**

```
過去三年中,我們開發過哪些蜂蜜風味的配方?最終量產的有哪幾款?
```

**預期輸出**

> 歷史配方清單表 + 量產與否原因

**驗證重點**

> 配方查詢完整;原因分析合理

## 情境 #3:競品差異化

**測試輸入**

```
比較我們現有的紅玉紅茶與市場上的競品,有哪些差異化空間?
```

**預期輸出**

> 競品分析表 + 我們的差異點 + 開發建議

**驗證重點**

> 競品資料具體;差異點扣到品牌定位
