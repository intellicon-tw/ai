# 補助計畫申請助理 - 驗測情境

以下 3 個情境,可直接複製測試輸入到 EgentHub 對話框驗測。
每個情境含:測試輸入、預期輸出、驗證重點。

## 情境 #1:茶葉外銷補助

**測試輸入**

```
我們要申請農業部茶葉外銷補助,主軸是 Rootopia Brew 拓展日本市場,請幫我產出計畫書初稿,並列出佐證文件。
```

**預期輸出**

> 六大段(摘要、簡介、目標、執行、效益、預算)完整初稿,並附佐證清單

**驗證重點**

> 六段完整;品牌實績有用;佐證清單合理

## 情境 #2:新創事業獎

**測試輸入**

```
我們想再申請經濟部新創事業獎,主訴 AI 中控台與跨國地方創生,請給我計畫書架構與重點訴求建議。
```

**預期輸出**

> 計畫書架構 + 重點訴求 + 既有獎項援引建議

**驗證重點**

> 架構符合公告;重點訴求扣到若水金禾差異化;有歷史獎項援引

## 情境 #3:ESG 補助

**測試輸入**

```
請幫我規劃 ESG 主題的補助計畫初稿,主軸是台日地方創生與在地茶農合作。
```

**預期輸出**

> 符合 ESG 訴求的計畫初稿,扣到既有實績(2025 ESG 創新獎)

**驗證重點**

> ESG 訴求清楚;在地連結具體;不過度誇大
