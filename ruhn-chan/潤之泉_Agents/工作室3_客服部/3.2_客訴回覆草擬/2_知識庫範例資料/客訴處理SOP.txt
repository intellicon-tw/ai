# 潤之泉客訴處理 SOP

## 客訴分類
1. **產品類**：包裝破損、異物、口感異常、品質懷疑
2. **物流類**：延遲、地址錯誤、包裝外觀
3. **客服態度類**：回覆慢、口氣不佳
4. **帳務類**：發票錯誤、退款延遲、優惠未套用
5. **期望不符類**：產品功效未達期望、口味不喜歡
6. **法規類**：標示質疑、認證疑慮
7. **媒體曝光風險類**：公開抱怨、負評、爆料

## 嚴重度評估
- **低**：單一案件、客戶情緒平穩、內部可處理
- **中**：需主管知悉、有重複發生風險
- **高**：含媒體風險、可能引起連鎖反應、需董事會層級回應

## 補償方案層級
- **基本**：致歉 + 退款或換貨
- **加碼**：基本 + 額外贈品（小品項）
- **VIP**：加碼 + 主管致電 + 大品項贈品

## 必通報情況
- 產品異常涉及食品安全 → 通報品管、主管機關
- 媒體曝光風險 → 通報董事會
- 客戶申訴消保官／衛福部 → 通報法務
