# 工作室8_經營決策數據中心

本工作室包含 6 個 Agent，共 5 個使用表格知識庫，1 個使用文件知識庫。

## Agent 清單

| 編號 | Agent 名稱 | 知識庫類型 | POC 方式 | 兩階段 |
|---|---|---|---|---|
| 8.1 | 銷售數據洞察 | 表格查詢（銷售明細+產品）+ Agent 向量（方法論） | csv 資料表 | 否 |
| 8.2 | 通路績效分析 | 表格查詢知識庫 | csv 資料表 | 否 |
| 8.3 | 營業預測 | 表格查詢知識庫 | csv 資料表 | 否 |
| 8.4 | 庫存與供需平衡 | 表格查詢知識庫 | csv 資料表 | 否 |
| 8.5 | 業績成長策略 | -（後續可掛上往策略決議記錄） | 純提示工程 | 否 |
| 8.6 | 會員行為分析 | 表格查詢知識庫 | csv 資料表 | 否 |
