# 工作室1_行銷企劃部

本工作室包含 4 個 Agent，共 1 個使用表格知識庫，2 個使用文件知識庫。

## Agent 清單

| 編號 | Agent 名稱 | 知識庫類型 | POC 方式 | 兩階段 |
|---|---|---|---|---|
| 1.1 | FB粉絲專頁小編 | Agent 向量（品牌語氣+貼文範例）+ 表格查詢（產品）+ 全文查找（合規） | csv 資料表 | 否 |
| 1.2 | 保健食品合規檢查官 | 全文查找知識庫（合規禁字） | RAG 文件 | 否 |
| 1.3 | EDM與LINE推播文案 | 多源知識庫（語氣+產品+合規） | 純提示工程 | 否 |
| 1.4 | 媒體新聞稿草擬 | 多源知識庫 | 純提示工程 | 否 |

## 各 Agent 子目錄

- `1.1_FB粉絲專頁小編/` — 1_prompt + 2_知識庫範例資料 + 3_驗測情境 + 4_正式上線前的準備指南
- `1.2_保健食品合規檢查官/` — 1_prompt + 2_知識庫範例資料 + 3_驗測情境 + 4_正式上線前的準備指南
- `1.3_EDM與LINE推播文案/` — 1_prompt + 2_知識庫範例資料 + 3_驗測情境 + 4_正式上線前的準備指南
- `1.4_媒體新聞稿草擬/` — 1_prompt + 2_知識庫範例資料 + 3_驗測情境 + 4_正式上線前的準備指南
