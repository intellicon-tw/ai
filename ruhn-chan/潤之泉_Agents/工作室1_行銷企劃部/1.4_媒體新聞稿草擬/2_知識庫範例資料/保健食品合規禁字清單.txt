# 保健食品 FB 貼文合規禁字清單（精簡版）

## 一、絕對禁用（依食安法第28條、健康食品管理法）
- 醫療效能類：治療、治癒、療效、根治、改善XX症狀、緩解XX、舒緩XX、預防XX疾病、防止XX
- 抗XX類：抗癌、抗腫瘤、抗發炎、抗病毒、抗菌
- 生理機能改變類：降血壓、降血糖、降膽固醇、降三高、減肥、瘦身、燃脂、消脂、塑身、增強免疫力、提升免疫力、增加抵抗力
- 暗示醫療：適合XX病患食用、取代藥物、不必再吃藥

## 二、需小心使用
- 「保健」「養生」可以用但不可與具體病名連用
- 「滋補」「強身」可以用但不可誇大
- 「術後修復」建議改為「術後輕鬆補」
- 「飲控」可以用但不可宣稱「減重」

## 三、合規替代寫法對照表
- 改善體質 → 日常營養補給
- 增強免疫力 → 天然營養支持
- 補身體 → 輕鬆補給
- 降血壓 → 低鈉配方
- 預防骨鬆 → 富含鈣質、植物蛋白
- 治療失眠 → 晚間放鬆茶飲、舒適入睡
- 抗氧化 → 含有多酚、富含天然多酚
- 減肥 → 飲控期間的輕盈選擇
- 美白 → 由內而外的好氣色

## 四、安全可用白名單
- 成分敘述：胺基酸、植物蛋白、低鈉、零脂肪、零普林
- 食用情境：日常營養補給、術後輕鬆補、飲控期間替代選擇
- 製程與原料：花蓮在地有機黑豆、190小時溫控發酵
- 認證：有機認證、TOC慈心、HALAL、TAIWAN ORGANIC、綠保標章
