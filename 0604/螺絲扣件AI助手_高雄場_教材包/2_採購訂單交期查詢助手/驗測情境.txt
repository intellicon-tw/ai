# 採購訂單交期查詢助手 - 驗測情境

## 情境 #1：逾期訂單標注驗證

**輸入描述** 使用者查詢「目前有哪些訂單已逾期」

**預期輸出** 列出所有狀態為逾期的訂單，含訂單編號、廠商、料號、未交數量，並加上⚠️標示

**驗證重點** 確認逾期標注正確，來自資料表的「訂單狀態」欄位，不自行推算

## 情境 #2：廠商維度彙整

**輸入描述** 使用者查詢「東捷鍍鋅有哪些訂單還沒交齊」

**預期輸出** 列出東捷鍍鋅所有未結案訂單，含未交數量與預計交期

**驗證重點** 確認只顯示東捷鍍鋅的訂單，不混入其他廠商

## 情境 #3：料號查詢

**輸入描述** 使用者查詢「料號 SWCH18A-Φ8 目前的採購狀況」

**預期輸出** 列出該料號所有訂單，含狀態與預計交期

**驗證重點** 確認回應依料號精準篩選

## 情境 #4：查無結果邊界測試

**輸入描述** 使用者查詢一個不存在的廠商「XX 線材」

**預期輸出** 回覆查無此廠商的採購訂單記錄

**驗證重點** 確認不捏造資料

## 可直接使用的測試題

```
東捷鍍鋅有哪些訂單還沒有交齊？
```

```
料號 SWCH18A-Φ8 目前的採購狀況如何？有沒有逾期？
```

```
目前有哪些訂單已經逾期未到？
```

```
訂單 PO-2026-0512 的交貨進度如何？
```
