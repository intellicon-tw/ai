# 製程規格查詢助手 - 驗測情境

## 情境 #1：料號精準查詢

**輸入描述** 使用者輸入「M8L25-Z 的鍍鋅厚度標準是多少？」

**預期輸出** 回傳 M8L25-Z 對應的鍍鋅厚度規範（含 µm 範圍與檢驗方法）

**驗證重點** 確認回應包含具體數值，且來源文件名稱正確

## 情境 #2：製程關鍵字查詢

**輸入描述** 使用者輸入「滾牙速度標準是多少？」

**預期輸出** 回傳滾牙製程參數（每分鐘轉數、進給速度、油液要求）

**驗證重點** 確認各項參數有明確數值，無捏造規格

## 情境 #3：表面處理規範查詢

**輸入描述** 使用者輸入「達克羅塗層厚度與鹽霧測試標準」

**預期輸出** 回傳達克羅厚度（µm）、鹽霧測試 720 小時等具體標準

**驗證重點** 確認數值來自知識庫，不混淆其他表面處理

## 情境 #4：查無資料邊界測試

**輸入描述** 使用者輸入不存在的料號 M99L99-X

**預期輸出** 回覆查無相關文件，建議聯繫研發部門

**驗證重點** 確認 Agent 不捏造資料，正確回覆查無結果

## 可直接使用的測試題

```
M8L25-Z 的鍍鋅厚度標準是多少？
```

```

```

```
達克羅塗層的厚度與鹽霧測試規範？
```

```

```
